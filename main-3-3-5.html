    Subsection 1.1.3 Variables

    In this section, we will discuss the two types of variables that appear in differential equations: dependent and independent variables. We will also explore how to identify these variables and understand their roles in the context of solving differential equations.
    As you can see, the differential equation,
    \begin{equation} \frac{dy}{dx} + 2y = 3x^2,\tag{1} \end{equation}
    has two variables: \(x\) and \(y\text{.}\) One of these variables is the â€œunknownâ€ that we are trying to solve for. But, which one?
    In general, the variable that you want to solve for will always have a derivative applied to it. The presence of the \(\frac{dy}{dx}\) in (1) implies
    1. the derivative (with respect to \(x\)) is being applied to \(y\text{,}\) and
    2. \(y\) must be a function of \(x\text{.}\)
      â€‰2â€‰
      Otherwise, it wouldnâ€™t make sense to take the derivative of \(y\) with respect to \(x\text{.}\)
    Since \(y\) depends on \(x\text{,}\) we call \(y\) the dependent variable. Consequently, \(x\) is the independent variable.

    Definition 1.3. Types of Variables.

    Dependent Variable
    The variable representing the unknown function we want to find. It will always have a derivative applied to it.
    Independent Variable
    The variable that the unknown function depends on.

    Example 1.4. Identify the Variables.

    Identify the dependent and independent variables in the differential equation
    \begin{equation*} \frac{dP}{ds} + \frac{P}{s^2} = 17s \text{,} \end{equation*}
    and state which variable represents the solution.
    Solution.
    Since \(\frac{dP}{ds}\) appears in this equation, it is implied that
    • \(P\) is a function that depends on \(s\) and must be the dependent variable,
    • \(s\) is the independent variable, and
    • \(P(s)\) represents the solution to this differential equation.

    Example 1.5. Identify the Variables.

    Identify the dependent and independent variables in the differential equation
    \begin{equation*} u'' + t^2 \ u = 0 \text{,} \end{equation*}
    and state which variable represents the solution.
    Solution.
    Since \(u''\) appears in this equation, it is implied that
    • \(u\) is a function that depends on \(t\) and must be the dependent variable,
    • \(t\) is the independent variable, and
    • \(u(t)\) represents the solution to this differential equation.

    Example 1.6. Identify the Variables.

    Identify the dependent and independent variables in the differential equation
    \begin{equation*} Q'' = 11Q \text{,} \end{equation*}
    and state which variable represents the solution.
    Solution.
    Since \(Q''\) appears in this equation, we know that \(Q\) is the dependent variable, but there is no sign of an independent variable. Typically the context of the problem will tell us the independent variable. Otherwise, we are free to select our own independent variable, say \(t\text{.}\) Now we can say that \(Q(t)\) would be the solution.
