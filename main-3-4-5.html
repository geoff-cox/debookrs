    Section 1.2.3 Types of Solutions

    At this point, you should comfortable with the idea that a solution to a differential equation is a function that satisfies it. You have also seen that a single differential equation can have more than one solution. However, this begs the question:
    “How many solutions does a differential equation have?”
    The answer depends on the context of the problem and is the focus of this section. To kick-off this discussion, lets start with an example.

    Example 1.11. Solution Verification with a Constant.

    Show that the functions
    \begin{equation*} y = e^{2x},\ \ y = 2e^{2x},\ \ y = 3e^{2x},\ \ y = -7e^{2x},\ \ y = 0.3e^{2x},\ \ y = \pi e^{2x} \end{equation*}
    are solutions to the differential equation \(\quad y' - 2y = 0\text{.}\)
    Solution.
    You could verify all of these functions one-by-one, but you will quickly find the process repetitive since each function has the pattern
    \begin{equation*} y = (\text{some number})\ e^{2x} \quad \text{or} \quad y = c e^{2x} \end{equation*}
    where \(c\) could be any of the numbers, \(1, 2, 3, -7, 0.3, \pi \text{,}\) above. So let’s show that \(y = c e^{2x}\) is a solution.
    Compute Derivatives. Find \(y'\) since it appears in the DE.
    \begin{align*} y \amp= c e^{2x} \\ y' \amp= c e^{2x}\cdot(2) = 2c e^{2x} \end{align*}
    Plug-In & Simplify. Plug \(y'\) and \(y'''\) into the DE and simplify.
    \begin{align*} y' - 2y \amp= 0 \\ 2c e^{2x} - 2(c e^{2x}) \amp= 0 \\ 2c e^{2x} - 2c e^{2x} \amp= 0 \\ 0 \amp= 0 \end{align*}
    Therefore, \(y = c e^{2x}\) is a solution to \(y' - 2y = 0\) where \(c\) is any constant. Notice that the specific value of \(c\) did not matter in the end.
    As you can see from the example above, the function \(y = c e^{2x}\) satisfies the differential equation \(y' - 2y = 0\) for ANY possible value of \(c\text{.}\)
    “How many choices for \(c\) are there?”
    Of course, the answer is infinitely many! This means that \(y' - 2y = 0\) has an infinite number of solutions. In most cases, as long as a differential equation has at least one solution, it also has infinitely many.
    This leads us to the following terms related to solutions of differential equations.

    Definition 1.12. Types of Solutions.

    Family of Solutions
    The collection of all possible solutions of a DE.
    General Solution
    The common form of the solutions in the family.
     5 
    By "form", we mean the common pattern (up to constants) that all solutions share.
    Particular Solution
    A single solution in the family.
    Applying these terms to the previous example, we can state the following:
    • \(y = c e^{2x}\) is the general solution of \(y' - 2y = 0\text{.}\)
    • \(y = 5e^{2x}\) is one particular solution in the family of solutions of \(y' - 2y = 0\text{.}\)
    • Any solution in the family of solutions of \(y' - 2y = 0\) looks like \(y = c e^{2x}\text{.}\)

    Remark 1.14.

    A family of solutions can be viewed as the collection of all particular solutions. Conversely, a particular solution can be viewed as a member of the family of solutions.

    Remark 1.15.

    When working on your own problems, remember that the following three statements are referring to the exact same thing.
    • Solve the following DE.
    • Find the general solution to the following DE.
    • Find the family of solutions for the following DE.
