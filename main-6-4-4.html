    Subsection 4.2.2 Matching Forms

    In the previous section, we explored how to directly compute backward Laplace transforms when the function of \(s\) matches a form in the common forward transform table. However, in practice, not all \(s\)-functions will align perfectly with these forms. In fact, the most challenging part of the Laplace Transform Method lies in preparing the \(s\)-function for the backward step. This process involves modifications of the function before matching the known forms in our table. In this discussion, we will cover some of the minor modifications that will need for the Laplace Transform Method.
    Many of the forms in TableÂ 4.16 can be matched just by observing the denominator of the \(s\)-function. The following table outlines the connection between the denominator and it associated transform.

    Missing Constants.

    The most common adjustment needed to match one of the \(s\)-functions in TableÂ 4.16 is to introduce a missing constant in the numerator. This is done by multiplying the numerator and denominator by this constant as shown in the following examples.

    Example 4.19.

    \(\quad\)Compute the inverse Laplace transforms for each.
    \(\ds \frac{12}{s}\)
    \(\hspace{27em}\)
    Solution.
    The \(s\) in the denominator tells us that we need \(L_1\text{.}\) Before we do, letâ€™s factor out \(12\text{:}\)
    \begin{equation*} \ds \ilap{\frac{12}{s}} = 12 \cdot \ilap{\frac{1}{s}} = 12 \cdot 1 = 12\text{.} \end{equation*}
    \(\ds \frac{5}{s^2 + 4}\)
    \(\hspace{27em}\)
    Solution.
    Since the denominator has the form, \(s^2 + \text{number}\text{,}\) and there is no \(s\) in the numerator, we should apply \(L_4\text{.}\) As before, it is helpful to first factor out the constant \(5\text{,}\)
    \begin{equation*} \ds \ilap{\frac{5}{s^2 + 4}} = 5\ \ilap{\frac{1}{s^2 + 4}} \quad \leftarrow \knowl{./knowl/xref/L4.html}{L_4} (b=2)\text{.} \end{equation*}
    According to \(L_4\text{,}\) we are missing \(2\) in the numerator. Letâ€™s put it there by multiplying by \(2/2\text{,}\) like so
    \begin{align*} = 5\ \ilap{\frac{2}{2}\frac{1}{s^2 + 4}} =\amp\ 5\ \ilap{\frac{1}{2}\frac{2}{s^2 + 4}}\\ =\amp\ \frac{5}{2} \ilap{\frac{2}{s^2 + 4}}\\ =\amp\ \frac{5}{2} \sin(2t) \end{align*}
    \(\ds \frac{17}{s^4}\)
    \(\hspace{27em}\)
    Solution.
    This denominator has the form \(s^{\text{power}}\text{,}\) which matches \(L_3\) with \(n = 3\text{.}\)
    \begin{equation*} \ds \ilap{\frac{17}{s^4}} = 17\ \ilap{\frac{1}{s^4}} \quad \leftarrow \knowl{./knowl/xref/L3.html}{L_3} (n=3)\text{.} \end{equation*}
    In this case, the numerator is missing a \(3!\text{.}\) We can introduce it by multiplying by \(3!/3!\text{,}\) like so
    \begin{gather*} = 17\ \ilap{\frac{3!}{3!}\frac{1}{s^4}} = \frac{17}{3!} \ilap{\frac{3!}{s^4}} = \frac{17}{6} t^3 \end{gather*}
    \(\ds \frac{7s}{s^2 + 25}\)
    \(\hspace{27em}\)
    Solution.
    The denominator here is \(s^2 + \text{number}\text{,}\) which matches \(L_5\) with \(b = 5\text{.}\) Therefore,
    \begin{align*} \ilap{\frac{7s}{s^2 + 25}} =\amp\ 7\ \ilap{\frac{s}{s^2 + 25}} \quad \leftarrow \knowl{./knowl/xref/L5.html}{\text{\(L_5\)}} (b=5)\\ =\amp\ 7\ \cos(5t)\text{.} \end{align*}
    \(\ds \frac{10}{(s - 3)^2 + 11}\)
    \(\hspace{27em}\)
    Solution.
    The form of this denominator is \(\ul{(s \pm \text{shift})^{2} + \text{number}}\) and has no \(s\) in the numerator. This matches \(L_7\) with \(a = 3\) and \(b = 5\text{.}\) Therefore,
    \begin{align*} \ilap{\frac{10}{(s - 3)^2 + 11}}\\ =\amp\ 10\ \ub{\ilap{\frac{\sqrt{11}}{\sqrt{11}}\frac{1}{(s - 3)^2 + 11}}}_{\large \knowl{./knowl/xref/L7.html}{\text{\(L_7\)}} (a=3,\ b=5)}\\ =\amp\ \frac{10}{\sqrt{11}} \ilap{\frac{\sqrt{11}}{(s - 3)^2 + 11}}\\ =\amp\ \frac{10}{\sqrt{11}} e^{3t} \sin(\sqrt{11}t) \end{align*}
    \(\ds \frac{2}{(s+7)^5}\)
    \(\hspace{27em}\)
    Solution.
    The denominator, \(\ul{(s \pm \text{shift})^{\text{power}}}\text{,}\) matches \(L_6\) with \(a = -7\) and \(n = 4\text{.}\) Therefore,
    \begin{align*} \ilap{\frac{2}{(s+7)^5}} =\amp\ 2\ \ub{\ilap{\frac{4!}{4!}\frac{1}{(s+7)^5}}}_{\large \knowl{./knowl/xref/L6.html}{\text{\(L_6\)}} (a=-7,\ n=4)}\\ =\amp\ \frac{2}{4!} \ilap{\frac{4!}{(s+7)^5}}\\ =\amp\ \frac{1}{12} t^4 e^{-7t} \end{align*}

    Splitting a Fraction.

    Another common modification is when the \(s\)-function contains a sum of different common forms. In these cases, you can usually split the fraction as the sum of these forms.

    Example 4.20.

    \(\quad\)Compute the inverse Laplace transforms for each.
    \(\ds \frac{2s - 3}{s^2 + 16}\)
    \(\hspace{27em}\)
    Solution.
    The key to this problem is to split the fraction into two separate terms:
    \begin{equation*} \ds \frac{2s - 3}{s^2 + 16} = \frac{2s}{s^2 + 16} - \frac{3}{s^2 + 16}. \end{equation*}
    Notice the first term matches with \(L_5\) and the second term matches with \(L_4\text{,}\) so
    \begin{align*} \amp\ \ilap{\frac{2s-3}{s^2 + 16}} \\ \amp \hspace{3em}= 2\ \ilap{\frac{s}{s^2 + 16}} - 3\ \ilap{\frac{1}{4}\frac{4}{s^2 + 16}} \\ \amp \hspace{3em}= 2\ \cos(4t) - \frac{3}{4} \sin(2t) \end{align*}
    \(\ds \frac{s^3 + 2s - 7}{s^7}\)
    \(\hspace{27em}\)
    Solution.
    We can split the fraction into the terms,
    \begin{equation*} \ds \frac{s^3 + 2s - 7}{s^7} = \frac{s^3}{s^7} + \frac{2s}{s^7} - \frac{7}{s^7} = \frac{1}{s^4} + \frac{2}{s^6} - \frac{7}{s^7}. \end{equation*}
    and apply the inverse Laplace transform to each
    \begin{gather*} \ilap{\frac{1}{s^4}} = \frac{1}{3!}\ilap{\frac{3!}{s^4}} = \frac{1}{3!}t^3\\ \ilap{\frac{2}{s^6}} = \frac{2}{5!}\ilap{\frac{5!}{s^6}} = \frac{2}{5!}t^5\\ \ilap{\frac{7}{s^7}} = \frac{7}{6!}\ilap{\frac{6!}{s^7}} = \frac{7}{6!}t^6 \end{gather*}
    Therefore, the inverse transform is
    \begin{equation*} \ds \ilap{\frac{s^3 + 2s - 7}{s^7}} = \frac{1}{3!}t^3 + \frac{2}{5!}t^5 - \frac{7}{6!}t^6. \end{equation*}
    This section explored two relatively minor modifications needed to match and find the backward Laplace transform. However, some \(s\)-functions require more complex techniques to reach the desired form. In the next section, we will discuss how completing the square can help us match the form where the \(s\)-variable is shifted.
