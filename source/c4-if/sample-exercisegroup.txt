		<exercise label="gen-sol-1">
			<statement><m>y' - 4y = x</m></statement>
			<solution>
			<p>
				Standard form: <m>y' + (-4)y = x</m>, so <m>P(x) = -4</m>.  
				Integrating factor: <m>\mu(x) = e^{\int -4 dx} = e^{-4x}</m>.
			</p>
			<md>
				<mrow>e^{-4x}y' - 4e^{-4x}y \amp = xe^{-4x}</mrow>
				<mrow>\frac{d}{dx}[e^{-4x}y] \amp = xe^{-4x}</mrow>
			</md>
			<p>
				Integrate: 
			</p>
			<md>
				<mrow>e^{-4x}y \amp = \int xe^{-4x} dx = -\frac{1}{4}xe^{-4x} - \frac{1}{16}e^{-4x} + C</mrow>
				<mrow>y \amp = -\frac{1}{4}x - \frac{1}{16} + Ce^{4x}</mrow>
			</md>
			</solution>
			<answer>
			<p><m>y = -\frac{1}{4}x - \frac{1}{16} + Ce^{4x}</m></p>
			</answer>
		</exercise>

		<exercise label="gen-sol-2">
			<statement><m>x^2 y' - y = 1, \quad x &gt; 0</m></statement>
			<solution>
			<p>
				Standard form: <m>y' - \frac{1}{x^2}y = \frac{1}{x^2}</m>.  
				Integrating factor: <m>\mu(x) = e^{\int -\frac{1}{x^2}dx} = e^{1/x}</m>.
			</p>
			<md>
				<mrow>\frac{d}{dx}[e^{1/x}y] \amp = \frac{e^{1/x}}{x^2}</mrow>
				<mrow>e^{1/x}y \amp = -e^{1/x} + C</mrow>
				<mrow>y \amp = -1 + Ce^{-1/x}</mrow>
			</md>
			</solution>
			<solution><title>(detailed)</title>
				<p>
					We put the equation in standard form to identify <m>P(x)</m> and compute the integrating factor.
				</p>
			
				<p>
					<me>
						y' + \us{P}{\boxed{-\frac{1}{x^2}}}\ y = \frac{1}{x^2} \quad \implies \quad
						\mu(x) = e^{\large \int -\sfrac{1}{x^2} dx} = e^{1/x}
					</me>
				</p>
			
				<p>
					Multiply by <m>e^{1/x}</m> and reverse the product rule on the left side.
					<md>
						<mrow> e^{1/x}y' - \frac{e^{1/x}}{x^2} y	\amp = \frac{e^{1/x}}{x^2}	</mrow>
						<mrow> \frac{d}{dx}\left[e^{1/x} y\right]	\amp = \frac{e^{1/x}}{x^2}	</mrow>
					</md>
				</p>
			
				<p>
					Next, integrate both sides and use <m>u</m>-substitution on the right side.
					<sidebyside widths="10% 60% 25% 5%" valigns="top top top top">

						<p/>
						<p>
							<md>
								<mrow>
									\int \frac{d}{dx}\left[e^{1/x} y\right] dx
													\amp = \int \frac{e^{1/x}}{x^2} dx \qquad \rightarrow
								</mrow>
								<mrow>	e^{1/x} y	\amp = \int -e^u du </mrow>
								<mrow>	e^{1/x} y	\amp = -e^u + c		</mrow>
								<mrow>	e^{1/x} y	\amp = -e^{1/x} + c	</mrow>
							</md>
						</p>
						<p>
							<md>
								<mrow>	u	\amp = \frac{1}{x}			</mrow>
								<mrow>	du	\amp = -\frac{1}{x^2} dx	</mrow>
							</md>
						</p>
						<p/>

					</sidebyside>
					
					Finally, solve for <m>y</m>:
					<md>
						<mrow> y = -1 + ce^{-1/x}	</mrow>
					</md>
				</p>
			</solution>
			<answer>
			<p><m>y = -1 + Ce^{-1/x}</m></p>
			</answer>
		</exercise>

		<exercise label="gen-sol-3">
			<statement><m>\dfrac{dy}{dx} - y = e^{3x}</m></statement>
			<solution>
			<p>
				Already in standard form: <m>y' + (-1)y = e^{3x}</m>.  
				Integrating factor: <m>\mu(x) = e^{\int -1 dx} = e^{-x}</m>.
			</p>
			<md>
				<mrow>\frac{d}{dx}[e^{-x}y] \amp = e^{2x}</mrow>
				<mrow>e^{-x}y \amp = \frac{1}{2}e^{2x} + C</mrow>
				<mrow>y \amp = \frac{1}{2}e^{3x} + Ce^{x}</mrow>
			</md>
			</solution>
			<answer>
			<p><m>y = \frac{1}{2}e^{3x} + Ce^{x}</m></p>
			</answer>
		</exercise>

		<exercise label="gen-sol-4">
			<statement><m>\dfrac{dy}{dx} = \dfrac{y}{x} + 2x + 1,\quad x &gt; 0</m></statement>
			<solution>
			<p>
				Standard form: <m>y' - \frac{1}{x}y = 2x + 1</m>.  
				Integrating factor: <m>\mu(x) = e^{\int -\frac{1}{x}dx} = \frac{1}{x}</m>.
			</p>
			<md>
				<mrow>\frac{d}{dx}\left[\frac{1}{x}y\right] \amp = 2 + \frac{1}{x}</mrow>
				<mrow>\frac{1}{x}y \amp = 2x + \ln x + C</mrow>
				<mrow>y \amp = 2x^2 + x \ln x + Cx</mrow>
			</md>
			</solution>
			<answer>
			<p><m>y = 2x^2 + x\ln x + Cx</m></p>
			</answer>
		</exercise>

		<exercise label="gen-sol-5">
			<statement><m>y\dfrac{dx}{dy} + 2x = 5y^3 </m></statement>
			<solution>
			<p>
				Treat <m>x</m> as dependent: <m>x' + \frac{2}{y}x = 5y^2</m> with <m>x' = \frac{dx}{dy}</m>.  
				Integrating factor: <m>\mu(y) = e^{\int \frac{2}{y}dy} = y^2</m>.
			</p>
			<md>
				<mrow>\frac{d}{dy}[y^2 x] \amp = 5y^4</mrow>
				<mrow>y^2 x \amp = y^5 + C</mrow>
				<mrow>x \amp = y^3 + \frac{C}{y^2}</mrow>
			</md>
			</solution>
			<answer>
			<p><m>x = y^3 + \frac{C}{y^2}</m></p>
			</answer>
		</exercise>

		<exercise label="gen-sol-6">
			<statement><m>-\dfrac{4M}{t^2} = t^5 - \dfrac{1}{t}\frac{dM}{dt} + 1</m></statement>
			<solution>
			<p>
				Rewrite: <m>M' - \frac{4}{t}M = t^6 + t</m> with <m>M' = \frac{dM}{dt}</m>.  
				Integrating factor: <m>\mu(t) = e^{\int -\frac{4}{t} dt} = t^{-4}</m>.
			</p>
			<md>
				<mrow>\frac{d}{dt}[t^{-4}M] \amp = t^2 + t^{-3}</mrow>
				<mrow>t^{-4}M \amp = \frac{1}{3}t^{3} - \frac{1}{2}t^{-2} + C</mrow>
				<mrow>M \amp = \frac{1}{3}t^{7} - \frac{1}{2}t^{2} + Ct^{4}</mrow>
			</md>
			</solution>
			<solution><title>(detailed)</title>
				<p>
					As always, start by putting the equation in standard form:
					<md>
						<mrow> \frac{1}{t} \frac{dM}{dt} - \frac{4}{t^2} \cdot M		\amp =  t^5 + 1	</mrow>
						<mrow> \frac{dM}{dt} + \us{P}{\boxed{-\frac{4}{t}}} \cdot M		\amp = t^6 + t	</mrow>
					</md>.
				</p>

				<sidebyside widths="35% 65%" valign="middle">
					<p>Find Integrating Factor, <m>\mu</m></p>
					<p>
						<me>
							\mu = e\vphantom{\large|}^{\int \sfrac{-4}{t}\ dt} = e^{-4\ln t} = e^{\ln t^{-4}} = t^{-4}
						</me>
					</p>
				</sidebyside>

				<sidebyside widths="35% 65%" valign="middle">
					<p>Multiply by <m>\mu</m></p>
					<p>
						<md>
							<mrow>
								t^{-4} \frac{dM}{dt} - t^{-4} \frac{4}{t} M
									\amp = t^{-4}(t^6 + t)
							</mrow>
						</md>
					</p>
				</sidebyside>

				<sidebyside widths="35% 65%" valign="middle">	
					<p>Complete the Product Rule</p>
					<p>
						<md>
							<mrow>
								\frac{d}{dt}\left[ \frac{1}{t^4} M \right]
									\amp = t^2 + t^{-3}
							</mrow>
						</md>
					</p>
				</sidebyside>

				<sidebyside widths="35% 65%" valign="middle">
					<p>Integrate Both Sides</p>
					<p>
						<md>
							<mrow>
								\int \frac{d}{dt}\left[ \frac{1}{t^4} M \right]\ dt
									= \int \left(t^2 + t^{-3}\right) dt
							</mrow>
						</md>
					</p>
				</sidebyside>

				<sidebyside widths="35% 65%" valign="middle">
					<p>Isolate <m>M</m></p>
					<p>
						<md>
							<mrow>
								\frac{1}{t^4} M = \frac{1}{3}t^{3} - \frac{1}{2}t^{-2} + c
							</mrow>
						</md>
					</p>
				</sidebyside>

				<sidebyside widths="35% 65%" valign="middle">		
					<p>General Solution</p>
					<p>
						<me>
							M = \frac{1}{3}t^{7} - \frac{1}{2}t^2 + ct^4
						</me>.
					</p>
				</sidebyside>
			</solution>
			<answer>
			<p><m>M = \frac{1}{3}t^{7} - \frac{1}{2}t^{2} + Ct^{4}</m></p>
			</answer>
		</exercise>

		<exercise label="gen-sol-7">

		<exercise label="gen-sol-8">
			<statement><m>\dfrac{dr}{d\theta} + r\tan \theta = \sec \theta,\ \ -\frac{\pi}{2} \le \theta \le \frac{\pi}{2}</m></statement>
			<solution>
			<p>
				Standard form: <m>r' + \tan\theta r = \sec\theta</m> with <m>r' = \frac{dr}{d\theta}</m>.  
				Integrating factor: <m>\mu(\theta) = e^{\int \tan\theta d\theta} = \sec\theta</m>.
			</p>
			<md>
				<mrow>\frac{d}{d\theta}[\sec\theta r] \amp = \sec^2\theta</mrow>
				<mrow>\sec\theta r \amp = \tan\theta + C</mrow>
				<mrow>r \amp = \sin\theta + C\cos\theta</mrow>
			</md>
			</solution>
			<answer>
			<p><m>r = \sin\theta + C\cos\theta</m></p>
			</answer>
		</exercise>

		<exercise label="ivp-1">
			<statement>
			<m>\dfrac{dy}{dx} - \dfrac{y}{x} = xe^x,\quad y(1) = e - 1 \quad (x &gt; 1)</m>
			</statement>
			<solution>
			<p>
				Standard form: <m>y' - \frac{1}{x} y = x e^x</m>.  
				Integrating factor: <m>\mu(x) = e^{\int -\frac{1}{x} dx} = x^{-1}</m>.
			</p>
			<md>
				<mrow>\frac{d}{dx}\left[\frac{1}{x}y\right] \amp = e^x</mrow>
				<mrow>\frac{1}{x}y \amp = \int e^x dx = e^x + C</mrow>
				<mrow>y \amp = x(e^x + C)</mrow>
			</md>
			<p>
				Apply <m>y(1) = e - 1</m>:
			</p>
			<me>
				e - 1 = 1(e + C) \Rightarrow C = -1.
			</me>
			<p>
				Final solution:
			</p>
			<me>
				y(x) = x(e^x - 1).
			</me>
			</solution>
			<answer>
			<p><m>y = x(e^x - 1)</m></p>
			</answer>
		</exercise>

		<exercise label="ivp-2">
			<statement>
			<m>e^t z' = 1 - 4e^t z,\quad z(0) = \dfrac{4}{3}</m>
			</statement>
			<solution>
			<p>
				Divide by <m>e^t</m>: <m>z' + 4z = e^{-t}</m>.  
				Integrating factor: <m>\mu(t) = e^{\int 4 dt} = e^{4t}</m>.
			</p>
			<md>
				<mrow>\frac{d}{dt}[e^{4t}z] \amp = e^{4t} e^{-t} = e^{3t}</mrow>
				<mrow>e^{4t}z \amp = \frac{1}{3} e^{3t} + C</mrow>
				<mrow>z \amp = \frac{1}{3}e^{-t} + Ce^{-4t}</mrow>
			</md>
			<p>
				Apply <m>z(0) = \frac{4}{3}</m>:
			</p>
			<me>
				\frac{4}{3} = \frac{1}{3} + C \Rightarrow C = 1.
			</me>
			<p>
				Final solution:
			</p>
			<me>
				z(t) = \frac{1}{3}e^{-t} + e^{-4t}.
			</me>
			</solution>
			<answer>
			<p><m>z = \frac{1}{3}e^{-t} + e^{-4t}</m></p>
			</answer>
		</exercise>

		<exercise label="ivp-3">
			<statement>
				<m>y' - 3y = 6,\quad y(0) = 5</m>
			</statement>
			<solution>
				<p>
					Standard form with <m>P(x) = -3</m>.  
					Integrating factor: <m>\mu(x) = e^{\int -3 dx} = e^{-3x}</m>.
				</p>
				<md>
					<mrow>\frac{d}{dx}[e^{-3x}y] \amp = 6e^{-3x}</mrow>
					<mrow>e^{-3x}y \amp = -2e^{-3x} + C</mrow>
					<mrow>y \amp = -2 + Ce^{3x}</mrow>
				</md>
				<p>
					Apply <m>y(0) = 5</m>:
				</p>
				<me>
					5 = -2 + C \Rightarrow C = 7.
				</me>
				<p>
					Final solution:
				</p>
				<me>
					y(x) = -2 + 7e^{3x}.
				</me>
			</solution>
			<answer>
			<p><m>y = -2 + 7e^{3x}</m></p>
			</answer>
		</exercise>
