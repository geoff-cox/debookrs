<section><title> Piecewise Forcing Functions </title> 

	<introduction>
		<p>
		  	Many physical systems experience sudden changes‚Äîlike a heater turning on, a traffic light switching, or a force applied at a specific moment. In mathematics, these sudden changes are modeled using <em>piecewise functions</em>: functions that behave differently over different intervals of time.
		</p>
	  
		<p>
		  	Instead of writing separate cases for each piece, we can use a clever tool called the <em>unit step function</em> to express everything in a single formula. This section introduces the unit step function and shows how it helps us construct, shift, and control piecewise behaviors in a clean, algebraic way.
		</p>
	  
		<p>
		  	By the end of this section, you'll be able to rewrite piecewise functions using step functions, model when things switch on or off, and even describe functions that turn on for only a limited time interval. These tools will be essential as we explore Laplace transforms of systems with piecewise or impulsive inputs.
		</p>
	</introduction>

	<subsection><title> Unit Step &amp; Piecewise Functions </title>

		<reading-questions>redacted</reading-questions>
		
		<subsubsection><title> The Standard Unit Step Function, <m>u(t)</m> </title>

			<p>
				In the last section, we hinted at a special function that could mimic the "on" and "off" behavior of piecewise functions. This function is called the <em>unit step function</em>, sometimes also known as the <em>Heaviside function</em>. It acts like a simple switch that flips from 0 to 1 at a specific time.
			</p>

			<p>
				Let's define it:
			</p>
			
			<definition xml:id="unit-step-function">
				<statement>
					<p>
						The <em>unit-step</em>, <m>u(t)</m>, is defined as
					</p>
					<sidebyside widths="2% 54% 5% 37% 2%" valign="top">
						<p/>
						<p>
							<me>
								u(t) = \left\{
								\begin{array}{ll}
									0,	&amp; t \lt 0\\
									1,	&amp; t \ge 0
								\end{array} \right.
							</me>
							It switches from 0 to 1 right at <m>t = 0</m>.
						</p>
						<p/>
						<p>
							<image>redacted</image>
						</p>
						<p/>
					</sidebyside>
				</statement>
			</definition>

			<exercise>
				<statement>
					<p>
						What are the only values <m>u(t)</m> can take for any value of <m>t</m>?
					</p>
				</statement>
				<choices>
					<choice correct="no">
						<statement>Only 0</statement>
					</choice>
					<choice correct="no">
						<statement>Only 1</statement>
					</choice>
					<choice correct="yes">
						<statement>Only 0 and 1</statement>
					</choice>
					<choice correct="no">
						<statement>Only positive numbers</statement>
					</choice>
					<choice correct="no">
						<statement>Any number</statement>
					</choice>
				</choices>
			</exercise>

			<p>
				The important thing to notice about <m>u(t)</m> is that it can only take two values, either 0 or 1.
			</p>

			<p>
				Because of this simple behavior, when you multiply a function by <m>u(t)</m>, you either shut the function off (multiply by 0) or turn it fully on (multiply by 1). For instance:
			</p>

			<me>
				\left(\frac{1}{5}t^2 - 1\right) u(t) =
				\left\{
				\begin{array}{ll}
					0, &amp; t \lt 0\\
					\frac{1}{5}t^2 - 1, &amp; t \ge 0
				\end{array}
				\right.
			</me>
			
			<p>
				In this example, for <m>t \lt 0</m> the entire function becomes zero. For <m>t \ge 0</m>, the function behaves just like <m>\frac{1}{5}t^2 - 1</m>. It's like flipping a switch at exactly <m>t = 0</m>.
			</p>

			<sidebyside widths="48% 4% 44%" valign="top">
				<p>
					We can think of it this way:
					<ul>
						<li>Before <m>t=0</m>: everything is OFF (multiplied by 0).</li>
						<li>After <m>t=0</m>: everything is ON (multiplied by 1).</li>
					</ul>
				</p>
				<p/>
				<p>
					<!-- <figure xml:id="unit-step-on-off-switch"><caption/> -->
						<image>redacted</image>
					<!-- </figure> -->
				</p>
			</sidebyside>

			<p>
				The unit step function gives us a smart way to model piecewise functions using a single formula instead of separate cases. In the next sections, we'll see how to shift this "switch" to turn functions on at times other than <m>t = 0</m>.
			</p>
			
		</subsubsection>

		<subsubsection><title> Unit Step Functions that Activate on <m>[c, \infty)</m> </title>

			<p>
				The standard unit step function <m>u(t)</m> flips a function on right at <m>t=0</m>. But what if we need to delay that switch‚Äîto turn a function on later, say at <m>t=2</m>, <m>t=5</m>, or even <m>t=-3</m>? We can make this happen by shifting the step function horizontally.
			</p>
			
			<p>
				We define a shifted unit step function by replacing <m>t</m> with <m>t-c</m>, where <m>c</m> is the shift:
			</p>

			<aside><title> üñãÔ∏è Notation: <m>u_c(t)</m> </title>
				<tabular top="minor">
					<col width="100%"/>
					<row>
						<cell>
							<p>
								Writing <m>u_c(t)</m> is just shorthand for <m>u(t - c)</m>. Both mean the same thing: a step function that switches from 0 to 1 at <m>t = c</m>.
							</p>
						</cell>
					</row>
				</tabular>
			</aside>

			<assemblage xml:id="shifted-unit-step-function"><title> Shifted Unit Step Function </title>
				<p>
					The <em>shifted unit step function</em>, <m>u_c(t)</m>, is defined as
				</p>

				<sidebyside widths="57% 1% 38%" margin="2%" valign="middle">
					<p>
						<md>
							<mrow>
								u_c(t) 
								\amp = u(t - c) = \left\{
									\begin{array}{ll}
										0,	&amp; t \lt c\\
										1,	&amp; t \ge c
									\end{array} 
									\right.
							</mrow>
						</md>
						This function switches from 0 to 1 at <m>t = c</m>.
					</p>
					<p/>
					<p>
						<image>redacted</image>
					</p>
				</sidebyside>
			</assemblage>
			
			<p>
				Multiplying a function by <m>u_c(t)</m> delays when it turns on. Instead of flipping on at <m>t=0</m>, it stays off until <m>t=c</m>. After that, the function behaves as normal.
			</p>
			
			<p>
				For example, the product
			</p>
			
			<me>
				\left(\frac{1}{5}t^2 - 1\right) u(t-2)
			</me>
			
			<p>
				would flatten the function to 0 for <m>t &lt; 2</m>, and then "turn it on" for <m>t \ge 2</m>.  
			</p>

		<subsubsection><title> Unit Step Functions that Activate on <m>(-\infty, c)</m> </title>

			<p>
				Until now, we've used the unit step function <m>u_{c}(t)</m> to switch a function on at a certain time <m>t = c</m>. But what if we want the opposite‚Äîto switch a function off at <m>t = c</m> instead?
			</p>

			<p>
				To do that, we reverse the role of the unit step function by subtracting it from 1. That is, the function
			</p>

			<me>
				1 - {\BO u_c(t)} = \us{\text{plug in } u_c}{\ul{1 - 
				\left\{\small
					\begin{array}{ll}
						\BO 0,	&amp; t \lt c\\
						\BO 1,	&amp; t \ge c
					\end{array} 
				\right.}}
				\quad =\ \us{1 \text{ minus each piece}}{\ul{
				\left\{\small
					\begin{array}{ll}
						1 - \BO 0,	&amp; t \lt c\\
						1 - \BO 1,	&amp; t \ge c
					\end{array} 
				\right.}}
				\quad =\ \us{\text{result}}{\ul{
				\left\{\small
					\begin{array}{ll}
						1,	&amp; t \lt c\\
						0,	&amp; t \ge c,
					\end{array} 
				\right.}}
			</me>

			<p>
				gives us a unit step function that starts at 1 and drops to 0 at <m>t = c</m>.
			</p>

			<assemblage xml:id="reversed-unit-step"><title> Reversed Unit Step Function </title>
				<p>
					The reversed unit step function that starts at 1 and drops to 0 at <m>t = c</m> is 
				</p>
				<sidebyside widths="5% 45% 5% 40% 5%" valign="middle">
					<p/>
					<p>
						<me>
							1 - u_{c}(t) =
							\left\{
							\begin{array}{ll}
								1, &amp; t \lt c\\
								0, &amp; t \ge c
							\end{array}
							\right.
						</me>
					</p>
					<p/>
					<p>
						<image>redacted</image>
					</p>
					<p/>
				</sidebyside>
			</assemblage>

			<p>
				When you multiply a function by <m>1 - u_{c}(t)</m>, you keep the function active until <m>t = c</m>, and then shut it off. 
			</p>

			<p>
				<sidebyside widths="45% 2% 47%" margin="3%" >
					<p>
						For example, the expression

						<me>
							\left(2e^{-\sfrac{t^2}{2}}\right)\left(1 - u_{1}(t)\right)
						</me>
						
						<p>
							represents a function that behaves like <m>2e^{-\sfrac{t^2}{2}}</m> before <m>t = 1</m>, but becomes zero at and after <m>t = 1</m>.
						</p>
					</p>
					<p/>
					<p>
						<image>redacted</image>
					</p>
				</sidebyside>
			</p>

			<example>
				<statement>
					<p>
						Write the piecewise definition of
						<me>
							f(t) = (3 - t^2)\left(1 - u_{2}(t)\right).
						</me>
					</p>
				</statement>
				<solution>redacted</solution>
			</example>

			<p> 
				While the unit step function, <m>u_{c}(t)</m>, activates functions to the right of <m>c</m>, this new reversed unit step function activates functions to the left of <m>c</m>. This is especially useful when we start building formulas for piecewise functions over different types of intervals.
			</p>

		</subsubsection>

		<subsubsection><title> Unit Step Functions that Activate on <m>[c, d)</m> </title>

			<p>
				We have already seen how <m>u_c(t)</m> switches a function on at <m>t=c</m> and how <m>1 - u_c(t)</m> switches a function off at <m>t=c</m>. Now let's combine these ideas to create functions that are active only over specific intervals.
			</p>
			
			<p>
				This situation can be achieved by subtracting two shifted unit step functions:
			</p>

			<p>
				<sidebyside widths="10% 30% 1% 40% 18%" valign="bottom">
					<p/>
					<p>
						<md>
							<mrow>\small
								u_{-1}(t) =
								\left\{
									\begin{array}{lr}
										0,	&amp; t \lt -1\\
										1,	&amp; t \ge -1
									\end{array} 
								\right.
							</mrow>
							<mrow>\small\phantom{-}</mrow>
						</md>

					</p>

								u_{3}(t) =
								\left\{
									\begin{array}{lr}
										0,	&amp; t \lt 3\\
										1,	&amp; t \ge 3
									\end{array} 
								\right.
							</mrow>
							<!-- <mrow>\small\phantom{-}</mrow> -->
						</md>

							<mrow>\quad u_{-1}(t) - u_{3}(t) \rightarrow</mrow>

			<p>
				In general, the difference of two unit step functions, <m>u_c(t) - u_d(t)</m>, is 1 on the interval <m>[c, d)</m> and 0 everywhere else. This can be expressed as the piecewise function:
				<md>
					<mrow>
						u_{c}(t) - u_{d}(t)
						= \left\{
							\begin{array}{lr}
								0,	&amp; 		t \lt c\\

						\right.
					</mrow>
				</md>
			</p>

				<p>
					The difference of shifted unit step functions given by
				</p>

				<p>
					<sidebyside widths="2% 47% 2% 47% 2%" valign="middle">
						<p/>
						<p>
							<me>
								u_c(t) - u_d(t) = \left\{
								\begin{array}{ll}
									0, &amp; t \lt c\\

				<p>
					is 1 on the interval <m>[c,d)</m> and 0 everywhere else.
				</p>
			</assemblage>
			
			<p>
				By multiplying a function <m>f(t)</m> by <m>u_c(t) - u_d(t)</m>, we can restrict <m>f(t)</m> so that it is active only on the interval <m>[c,d)</m> and zero everywhere else.

		<subsubsection><title> Writing Piecewise Functions in Unit Step Form </title>

			<p>
				The goal is to deconstruct the piecewise function into a sum of its pieces multiplied by a unit step function that is <m>1</m> on its region. That is, we would like to write <m>f(t)</m> as
			</p>

			<me>
				\small
				f(t) = 
				f_{1}(t) \left(
				\left\{
					\begin{array}{cl}
						1	\amp \text{region}\ 1\\
						0	\amp \text{otherwise}\\
					\end{array}
				\right.
				\right) +
				f_{2}(t) \left(
				\left\{

	<subsection><title> Unit Step &amp; Piecewise Laplace Transforms </title>

		<introduction>
			<p>
				Let's say we're asked to solve a second-order differential equation with a piecewise forcing function. For example:
			</p>
			
			<me>
				y'' + y = f(t), \quad \text{where } 
				f(t) = \left\{
					\begin{array}{ll}
						0, 		&amp; t \lt 3 \\
						2 - t,	&amp; t \ge 3
					\end{array}
				\right.
			</me>
			
			<p>
				Since <m>f(t)</m> is a piecewise function, we rewrite it in the unit step form
				<me>
					f(t) = (2 - t)\, u_3(t).
				</me>
			</p>
			
			<p>
				To solve the equation, we apply the Laplace transform to both sides:
				<me>
					\lap{y'' + y} = \ub{\lap{(2 - t)\, u_3(t)}}_{=\ ?}.
				</me>
				On the left-hand side, we can use our standard rules for <m>\lap{y''}</m> and <m>\lap{y}</m>. But, how do we take the Laplace transform of the right?
			</p>
			
			<p>
				These expression don't match anything in our Laplace tables yet. So before we can proceed, we need to establish the following two transforms:
				<ol>
					<li><m>u_c(t)</m>, the shifted step function by itself,</li>
					<li><m>f(t)\, u_c(t)</m>, a function that ‚Äústarts late‚Äù at <m>t = c</m>.</li>
				</ol> 
			</p>
			
			<p>
				In this subsection, we'll derive both <m>\lap{u_c(t)}</m> and <m>\lap{f(t)\,u_c(t)}</m>. Then we will generalize these results to compute the Laplace transform of piecewise functions. 
			</p>
			
		</introduction>

		<subsubsection><title> Laplace Transform of <m>u_c(t)</m> </title>

			<p>
				We will now jump right into the Laplace transform of a unit step function, <m>u_c(t)</m>. 
			</p>
		
			<assemblage xml:id="laplace-of-step"><title> Laplace Transform of a Shifted Step </title>
				<p>
					The Laplace transform of the shifted unit step function is
					<me>
						\lap{u_c(t)} = \frac{e^{-cs}}{s}.
					</me>
				</p>
				<p>
					<theorem><title> <m>\rightarrow \lap{u_c(t)}</m> </title>

						<p>
							To derive this, recall the definition of the Laplace transform:
							<me>
								\lap{u_c(t)} = \int_0^{\infty} u_c(t) \, e^{-st} \, dt.
							</me>
							But remember that
							<me>
								u_c(t) = \left\{
								\begin{array}{ll}
								0, &amp; t \lt c\\
								1, &amp; t \ge c
								\end{array}
								\right.
							</me>
							So the integrand is zero for <m>t \lt c</m>, and equal to <m>e^{-st}</m> for <m>t \ge c</m>.
						</p>

						<p>
							That simplifies our integral to just the portion from <m>t = c</m> to infinity:
							<me>
								\lap{u_c(t)} = 
								\os{\large t\ \lt\ c}{\overline{\int_0^{c} \ub{u_c(t)}_{=\ 0} \, e^{-st} \, dt}} 
								+ 
								\os{\large t\ \ge\ c}{\overline{\int_c^{\infty} \ub{u_c(t)}_{=\ 1} \, e^{-st} \, dt}}
								= \int_c^{\infty} e^{-st} \, dt
							</me>
						</p>
						<p>
							To compute the integral, we have to remember that this is an improper integral, so we re-express it as a limit
							<me>
								\lap{u_c(t)} = \int_c^{\infty} e^{-st} \, dt
								= \lim_{b \to \infty}\int_c^b e^{-st} \, dt
							</me>
							and first compute the integral
							<me>
								\int_c^b e^{-st} \, dt
								= \left[ \frac{-1}{s} e^{-st} \right]_c^b
								= \frac{-1}{s} \left( e^{-sb} - e^{-sc} \right)
								= \frac{e^{-sc}}{s} - \frac{e^{-sb}}{s}.
							</me>
						</p>
						<p>
							Now, we compute the limit of this expression as <m>b \to \infty</m>:
							<me>
								\lim_{b \to \infty} \left( \frac{e^{-sc}}{s} - \frac{e^{-sb}}{s} \right)
								= \frac{e^{-sc}}{s} - \lim_{b \to \infty} \frac{e^{-sb}}{s}
							</me>.
							As before, the <term>limit of the exponential function</term> converges if its power, <m>-sb</m>, goes to <m>-\infty</m>. This happens if <m>s \gt 0</m>.
						</p>
						<p>
							So, as long as <m>s \gt 0</m>, then <m>\lim_{b \to \infty} e^{-sb} = 0</m> and the transform becomes 
							<me>
								\lap{u_c(t)} = \frac{e^{-cs}}{s}, \quad s \gt 0
							</me>.
						</p>
					</theorem>
				</p>
			</assemblage>

			<p>
				This is a remarkably clean result. A delayed step in time becomes multiplication by an exponential decay factor in the Laplace domain. Now that we've established this, we can build up to more general transforms like <m>\lap{f(t)\, u_c(t)}</m>, which is the key to transforming most piecewise-defined functions.
			</p>

		<subsubsection><title> Laplace Transform of <m>f(t)\cdot u_c(t)</m> </title>

			<p>
				Now that we've seen how to compute <m>\lap{u_c(t)}</m>, let's tackle the next goal: finding the Laplace transform of a function that's ‚Äúturned on‚Äù at time <m>t = c</m>. That is, how do we handle <m>\lap{f(t)\, u_c(t)}</m>?
			</p>
			
			<p>
				This expression appears all the time when we rewrite piecewise functions using unit step notation. For example, from earlier:
				<me>
					f(t) = (2 - t)\, u_2(t).
				</me>
			</p>
		
			<assemblage xml:id="laplace-f-uc"><title> Laplace Transform of <m>f(t) \, u_c(t)</m> </title>
				<p>
					If a function is multiplied by a shifted step function, its Laplace transform becomes:
					<men xml:id="L14-eqn">
						\lap{f(t) \, u_c(t)} = e^{-cs} \lap{f(t+c)}.
					</men>
				</p>
				<p>
					<theorem><title> <m>\rightarrow \lap{f(t) \, u_c(t)}</m> </title>
						<p>
							To find the Laplace transform, we start by computing
							<me>
								\lap{f(t) \, u_c(t)} = \int_0^\infty f(t) \, u_c(t) \, e^{-st} \, dt.
							</me>
							Remember, <m>u_c</m> is either 0 or 1, so the integrand is the piecewise function
							<me> 
								f(t) \, {\BO u_c(t)} \, e^{-st} 
								=
								f(t) \, e^{-st} \cdot {\BO 
								\left\{
									\begin{array}{ll}
										0\\
										1
									\end{array}
								\right.}
								\quad = \quad
								\left\{
									\begin{array}{ll}
										0,				&amp; t \lt c\\
										f(t)e^{-st},	&amp; t \ge c,
									\end{array}
								\right.
							</me>
						</p>

						<p>
							Separating the integral over these regions leads to
							<me>
								\lap{f(t) \, u_c(t)} = \int_c^\infty f(t) e^{-st} \, dt.
							</me>
						</p>

						<p>
							This new integral looks a¬†lot like the Laplace transform of <m>f(t)</m>, but it starts at <m>t = c</m> instead of <m>t = 0</m>. Luckily, we can use a simple <m>u</m>-substitution to create a new variable that starts at <m>t = 0</m>. 
						</p>

						<aside><title> <m>\B \text{A}, \text{B}, \text{C}, \text{D}</m> Details </title>
							<tabular top="minor" bottom="minor">
								<col width="100%"/>
								<row>
									<cell>
										<p>
											<m>\B \text{A}\ </m>
											<m>u</m>-substitution
											<md>
												<mrow>t = u + c</mrow>
												<mrow>dt = du</mrow>
											</md>
										</p>
									</cell>
								</row>
								<row>
									<cell>
										<p>
											<m>\B \text{B}\ </m>
											Exponential Property
											<md>
												<mrow>e^{-s(u+c)} = e^{-su} e^{-sc}</mrow>
											</md>
										</p>
									</cell>
								</row>
								<row>
									<cell>
										<p>
											<m>\B \text{C}\ </m>
											Since the integration variable is <m>u</m>, we can factor out <m>e^{-sc}</m> as a constant.
										</p>
									</cell>
								</row>
								<row>
									<cell>
										<p>
											<m>\B \text{D}\ </m>
											Except for <m>c</m> and <m>s</m>, you can rename the integration variable as anything.
										</p>
									</cell>
								</row>
							</tabular>
						</aside>

						<p>
							Let <m>u = t - c</m>, so <m>du = dt</m>, and the <m>u</m>-integral is
							<md>
								<mrow>
									\int_{t\ =\ c}^{t\ =\ \infty} f(t)\ e^{-st} \, dt 
									\amp \os{\large\B \text{A}}{=} 
									\int_{u\ =\ 0}^{u\ =\ \infty} f(u+c)\ e^{-s(u+c)}\ du
								</mrow>
								<mrow>
									\amp \os{\large\B \text{B}}{=}
									\int_0^\infty f(u+c) \left(e^{-su} \cdot e^{-sc}\right) \, du
								</mrow>
								<mrow>
									\amp \os{\large\B \text{C}}{=}
									e^{-cs} \int_0^\infty f(u+c) e^{-su} \, du
								</mrow>
								<mrow>
									\amp \os{\large\B \text{D}}{=}
									e^{-cs} \ub{\int_0^\infty f(t+c) e^{-st} \, dt}_{\large \lap{f(t+c)}}
								</mrow>
							</md>
						</p>

						<p>
							Therefore,
							<me>
								\lap{f(t) \, u_c(t)} = e^{-cs} \lap{f(t+c)}.
							</me>
						</p>
					</theorem>
				</p>
			</assemblage>

				This rule lets us instantly find Laplace transforms for many common piecewise functions. Let's see how it works in practice with a few examples.

		<subsubsection><title> Laplace Transforms of Piecewise Functions </title>

  <p>
    We now have a complete set of tools to handle piecewise forcing functions. The key idea is that once we write the function in <em>unit step form</em>, everything else follows using the Laplace rules we've already established.
  </p>

  <p>
    So what‚Äôs the real challenge? It's not the Laplace transform anymore‚Äîit‚Äôs rewriting the function correctly using step functions.
  </p>

  <p>
    Any piecewise function can be expressed as a sum of terms like <m>f(t)\, u_c(t)</m> or <m>f(t)\, (u_c(t) - u_d(t))</m>. Once it's written in that form, we just apply:
    <ul>
      <li><m>\lap{u_c(t)} = \frac{e^{-cs}}{s}</m>, and</li>
      <li><m>\lap{f(t)\, u_c(t)} = e^{-cs} \lap{f(t + c)}</m>.</li>
    </ul>
  </p>

  <p>
    Let‚Äôs walk through an example to see how this process works from start to finish.
  </p>

  <assemblage xml:id="piecewise-laplace-summary"><title> Summary: Laplace Transforms of Piecewise Functions </title>
    <p>
      To compute <m>\lap{f(t)}</m> for a piecewise function:
      <ol>
        <li>Rewrite <m>f(t)</m> as a sum of terms using <m>u_c(t)</m> and <m>u_c(t) - u_d(t)</m>.</li>

        <li>Use the shift rule: <m>\lap{f(t)\, u_c(t)} = e^{-cs} \lap{f(t + c)}</m>.</li>
      </ol>
    </p>
