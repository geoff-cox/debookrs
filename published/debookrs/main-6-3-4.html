    Subsection 4.1.2 Properties

    Let’s build on what we did in the previous section.

    Example 4.7. Find a Laplace transform of the sum of functions.

    Find the Laplace transform of the function
    \begin{equation*} f(t) = 15 + 6e^{7t} - 11t. \end{equation*}
    Solution.
    Hopefully those functions look familiar from the previous section. In fact, we can write that
    \begin{equation*} f(t) = g(t) + 6x(t) - q(t). \end{equation*}
    Given that \(f\) can be written in terms of the functions \(g\text{,}\) \(x\) and \(q\) that we examined in the previous section, we might like to take advantage of the results we found. We will, but let’s use the definition of the Laplace transform to see exactly how we might incorporate those previous results.
    \begin{align*} F(s) =\amp\ \lap{ f(t) } \\ =\amp\ \int_0^{\infty} e^{-st}f(t)dt \\ =\amp\ \int_0^{\infty} e^{-st}\left(15 + 6e^{7t} - 11t\right)dt \\ =\amp\ \int_0^{\infty} \left( 15e^{-st} + 6e^{-st}e^{7t} - e^{-st}\cdot 11t \right) \end{align*}
    Now we can use properties of integration to rewrite this expression. Recall that we can split the integral up over addition and/or subtraction and we can pull out coefficients if we like.
    \begin{align*} F(s) =\amp\ \int_0^{\infty} \left(15e^{-st} + 6e^{-st}e^{7t} - e^{-st}\cdot 11t \right)dt \\ =\amp\ \int_0^{\infty}15e^{-st}dt + 6\int_0^{\infty}e^{-st}e^{7t}dt - \int_0^{\infty}e^{-st}\cdot 11tdt \end{align*}
    The first integral was the integral we evaluated in Example 1 of the previous section; similar with the other two integrals. We can actually write this as:
    \begin{align*} F(s) =\amp\ \int_0^{\infty}15e^{-st}dt + 6\int_0^{\infty}e^{-st}e^{7t}dt - \int_0^{\infty}e^{-st}\cdot 11tdt, \\ =\amp\ \lap{g(t)} + 6\lap{ x(t) } - \lap{ q(t) }, \\ =\amp\ \ub{\frac{15}{s}}_{s \gt 0} + 6\ub{\frac{1}{s - 7}}_{s \gt 7} - \ub{\frac{11}{s^2}}_{s \gt 0}. \end{align*}
    We need to deal with those conditions on \(s\text{.}\) If we need to make sure that \(s >0\) and that \(s \gt 7\text{,}\) it’s sufficient to say that \(s \gt 7\) (because this guarantees that \(s \gt 0\) also). So we actually have
    \begin{equation*} F(s) = \frac{15}{s} + 6\cdot \frac{1}{s-7} - \frac{11}{s^2}, \quad s \gt 7. \end{equation*}
    We found the desired answer, but now is a good time to step back and see that we just showed something very interesting and useful. We can actually rewrite that last result as the equation below.
    \begin{equation*} \lap{ g(t) + 6x(t) - q(t)} = \lap{g(t)} + 6\lap{ x(t) } - \lap{ q(t) } \end{equation*}
    It turns out (unsurprisingly) that the Laplace transform has the same properties that integrals do: we can separate over addition and subtraction, and we can pull out coefficients. This is summarized in a single property that is often referred to is the linearity of the Laplace transform.
    Property: The Laplace transform is a linear transform. This means that if \(a\) and \(b\) are constants, and \(f(t)\) and \(g(t)\) are functions that have Laplace transforms \(F(s)\) and \(G(s)\text{,}\) respectively, then
    \begin{equation*} \lap{ a f(t) \pm b g(t) } = a \lap{ f(t) } \pm b \lap{ g(t) }, \end{equation*}
    or, equivalently,
    \begin{equation*} \lap{ a f(t) \pm b g(t) } = a F(s) \pm b G(s). \end{equation*}
    We can use this property to actually pull more information out of the results we found in the previous section. In particular, we showed that
    \begin{equation*} \lap{15} = \frac{15}{s}. \end{equation*}
    Since 15 is a constant, we can use the linearity property as follows.
    \begin{align*} \lap{15} =\amp\ \frac{15}{s} \\ \lap{15 \cdot 1} =\amp\ 15 \cdot \frac{1}{s} \\ 15\lap{1} =\amp\ 15 \cdot \frac{1}{s} \end{align*}
    Now, if we divide both sides by 15 we have
    \begin{equation*} \lap{1} = \frac{1}{s}, \hspace{0.2cm}s \gt 0. \end{equation*}
    Similarly, we showed that \(\lap{ 11t } = \frac{11}{s^2}\text{,}\) and we can use the linearity property to show that
    \begin{equation*} \lap{t} = \frac{1}{s^2} \hspace{0.2cm}s \gt 0. \end{equation*}
    Assuming \(s \gt 0\text{,}\) and using both integration by parts (IBP) and L’Hospital’s Rule (LH), we can prove the following.
    • \(\ds \lap{t^2} = \frac{2}{s^3} \quad s \gt 0 \qquad \) Show the details.
    • \(\ds \lap{t^3} = \frac{6}{s^4} \quad s \gt 0 \qquad \) Show the details.
    • \(\ds \lap{t^4} = \frac{24}{s^5} \quad s \gt 0 \qquad \) Show the details.
    We organize these in a table.
    \begin{equation*} \begin{array}{|c|c|} \hline f(t) \amp \ds \lap{f(t)} = F(s) \\ \hline 1 \amp \ds \frac{1}{s}, \quad s \gt 0 \\ \hline t \amp \ds \frac{1}{s^2}, \quad s \gt 0 \\ \hline t^2 \amp \ds \frac{2}{s^3}, \quad s \gt 0 \\ \hline t^3 \amp \ds \frac{6}{s^4}, \quad s \gt 0 \\ \hline t^4 \amp \ds \frac{24}{s^5}, \quad s \gt 0 \\ \hline \end{array} \end{equation*}
    Based on this, can you identify a pattern? It seems that we may have identified another property.
    Property: The Laplace transform of \(t^n\text{,}\) for \(n = 0, 1, 2, \ldots\text{,}\) is given by
    \begin{equation*} \lap{ t^n } = \frac{n!}{s^{n+1}}. \end{equation*}
    While we are gleaning information from previous examples, let’s recall Example 4.5, from the previous section. There, we showed that
    \begin{equation*} \lap{ e^{7t} } = \frac{1}{s-7}, \quad s \gt 7. \end{equation*}
    What do you suppose is the Laplace transform of the function \(e^{23 t}\text{?}\) We can surmise another property that shows us how to take Laplace transforms of exponential functions.
    Property: The Laplace transform of \(e^{at}\) is given by
    \begin{equation*} \lap{ e^{at} } = \frac{1}{s - a}. \end{equation*}
