    Section 1.2.2 Verifying Solutions

    It is worth pointing out that the following differential equations are identical:
    \begin{equation*} y^\prime = 3y \quad \text{and} \quad y^\prime - 3y = 0\text{.} \end{equation*}
    This means that the solutions to both equations are the same and you are free to use either form when verifying solutions. For a beginner, it is often easier to verify solutions using the second form, as it is easier to see if the left-side of the equation simplifies to zero.

    TIP: Move Terms to the Left-Side before Verifying.

    Before verifying solutions, move all terms to the left-side of the equation using subtraction. This makes the right-side zero, simplifying the process.

    Solution Verification Steps.

    Verifying if a function is a solution to a differential equation involves checking if the function satisfies the equation, outlined as follows:
    Move Terms
    Move terms to the left-side of the equation.
    Compute Derivatives
    Compute & simplify derivatives appearing in the DE.
     4 
    This helps reduce errors by breaking the problem into more manageable steps.
    Plug-In & Simplify
    Substitute the function & derivatives into the equation and simplify each side of the equation.
    If you end up with a true statement \(\left(0 = 0\right)\text{,}\) then the function is a solution.
    Let’s look at a few warm-up examples.

    Example 1.7. Solution Verification 1.

    Is \(y= 2x^2\) a solution to \(\quad xy' - 2x^2 = y\) ?
    Solution.
    Let’s follow the steps outlined above.
    Move Terms: \(\quad xy' - 2x^2 - y = 0\)
    Compute Derivatives: Find \(y'\) since it appears in the DE.
    \begin{equation*} y = 2x^2, \quad y' = 4x \end{equation*}
    Plug-In & Simplify: Plug \(y\) and \(y'\) into the DE and simplify.
    \begin{align*} xy' - 2x^2 - y \amp= 0 \\ x\left(4x\right) - 2x^2 - \left( 2x^2 \right) \amp= 0 \\ 4x^2 - 2x^2 - 2x^2 \amp= 0 \\ 0 \amp= 0 \quad \leftarrow \text{true} \end{align*}
    Therefore, \(y= 2x^2\) is a solution to \(xy' - 2x^2 = y.\)

    Example 1.8. Solution Verification 2.

    Is \(P = \sin t\) a solution to \(\quad 2P'' + P = \sin t\) ?
    Solution.
    Move Terms: \(\quad 2P'' + P - \sin t = 0\)
    Compute Derivatives. Find \(P''\) since it appears in the DE.
    \begin{align*} P \amp= \sin t \\ P' \amp= \cos t \\ P'' \amp= -\sin t \end{align*}
    Plug-In & Simplify. Plug \(P\) and \(P''\) into the DE and simplify.
    \begin{align*} 2P'' + P - \sin t \amp= 0 \\ 2\left( -\sin t \right) + \sin t - \sin t \amp= 0 \\ -2\sin t + \cancel{\sin t} - \cancel{\sin t} \amp= 0 \\ -2\sin t \amp= 0\quad \leftarrow \text{false} \end{align*}
    Therefore, \(P = \sin t\) is not a solution to \(2P'' + P = \sin t.\)

    Example 1.9. Two Solutions for One DE.

    Verify that both \(y= 2e^{-3t}\) and \(y= e^{4t}\) are solutions to \(\quad y'' - y' - 12y = 0\) .
    Solution 1. \(\quad y= 2e^{-3t}\)
    Compute Derivatives. Find \(y'\) and \(y''\) since they appear in the DE.
    \begin{align*} y \amp= 2e^{-3t} \\ y' \amp= 2e^{-3t}\left(-3\right) = -6e^{-3t} \\ y'' \amp= -6e^{-3t}\left(-3\right) = 18e^{-3t} \end{align*}
    Plug-In & Simplify. Plug \(y, y',\) and \(y''\) into the DE and simplify.
    \begin{align*} y'' - y' - 12y \amp= 0 \\ 18e^{-3t} - \left( -6e^{-3t} \right) - 12\left( 2e^{-3t} \right) \amp= 0 \\ 18e^{-3t} + 6e^{-3t} - 24e^{-3t} \amp= 0 \\ 0 \amp= 0 \quad \leftarrow \text{true} \end{align*}
    Therefore, \(y= 2e^{-3t}\) is a solution to \(y'' - y' - 12y = 0.\)
    Solution 2. \(\quad y= e^{4t}\)
    Compute Derivatives. Find \(y'\) and \(y''\) since they appear in the DE.
    \begin{align*} y \amp= e^{4t} \\ y' \amp= e^{4t}\left(4\right) = 4e^{4t} \\ y'' \amp= 4e^{4t}\left(4\right) = 16e^{4t} \end{align*}
    Plug-In & Simplify. Plug \(y, y',\) and \(y''\) into the DE and simplify.
    \begin{align*} y'' - y' - 12y \amp= 0 \\ 16e^{4t} - 4e^{4t} - 12\left( e^{4t} \right) \amp= 0 \\ 16e^{4t} - 4e^{4t} - 12e^{4t} \amp= 0 \\ 0 \amp= 0 \quad \leftarrow \text{true} \end{align*}
    Therefore, \(y= e^{4t}\) is a solution to \(y'' - y' - 12y = 0.\)

    Example 1.10. Solution Verification.

    Is \(y= 3\sin(x^2)\) a solution to the DE \(\quad y' - xy'' = 12x^2\sin(x^2)\text{?}\)
    Solution.
    Move Terms: \(\quad y' - xy'' - 12x^2\sin(x^2) = 0\)
    Compute Derivatives. Find \(y'\) and \(y''\) since they appear in the DE.
    \begin{align*} y \amp= 3\sin(x^2) \\ y' \amp= 3\cos(x^2) \cdot 2x = 6x\cos(x^2) \\ y'' \amp= 6\cos(x^2) - 12x\sin(x^2) = 6\cos(x^2) - 12x\sin(x^2) \end{align*}
    Plug-In & Simplify. Plug \(y, y',\) and \(y''\) into the DE and simplify.
    \begin{align*} y' - xy'' - 12x^2\sin(x^2) \amp= 0 \\ 6x\cos(x^2) - x\left( 6\cos(x^2) - 12x\sin(x^2) \right) - 12x^2\sin(x^2) \amp= 0 \\ \cancel{6x\cos(x^2)} - \cancel{6x\cos(x^2)} - \cancel{12x^2\sin(x^2)} - \cancel{12x^2\sin(x^2)} \amp= 0 \\ 0 \amp= 0 \end{align*}
    Therefore, \(y= 3\sin(x^2)\) is a solution to \(y' - 12x^2\sin(x^2) = xy''.\)

    2. True/False. \(y = x^2 + 3\) is a solution to \(\frac{dy}{dx} - 3 = 2x\).

      \(y = x^2 + 3\) is a solution to the differential equation \(\displaystyle \frac{dy}{dx} - 3 = 2x\text{.}\)
    • True
    • Incorrect.
      \(y = x^2 + 3\) is not a solution since
      \begin{align*} \frac{dy}{dx} - 3 \amp= 2x \\ \frac{d}{dx}\left[x^2 + 3\right] - 3 \amp= 2x \\ 2x - 3 \amp= 2x \quad \leftarrow \text{false} \end{align*}
    • False
    • Correct!
      \(y = x^2 + 3\) is not a solution since
      \begin{align*} \frac{dy}{dx} - 3 \amp= 2x \\ \frac{d}{dx}\left[x^2 + 3\right] - 3 \amp= 2x \\ 2x - 3 \amp= 2x \quad \leftarrow \text{false} \end{align*}
