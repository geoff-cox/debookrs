    Subsection 4.2.3 Matching Forms, Part 1

    Let’s review the Laplace transform diagram to remember how we will leverage Laplace transforms to solve differential equations.
    The one last piece we need in order to solve DEs is to be able to take inverse Laplace transforms. We will focus on that piece now, and in the next section we will put all of our new skills together to solve DEs using Laplace transforms.
    When we take the inverse Laplace transform, we begin with a function of \(s\text{,}\) say \(Y(s)\text{,}\) and what we seek is a function of \(y(t)\) such that if we took the Laplace transform of \(y\text{,}\) we would get \(Y\text{.}\) That is, we seek to un-do the the Laplace transform. The notation looks similar:
    \begin{equation*} y(t) = \lap^{-1}\{ Y(s) \}, \end{equation*}
    and, as with the forward Laplace transform, the curly braces are not optional. Like the forward Laplace transform, the inverse Laplace transform is also a linear transform.
    In order to take inverse Laplace transforms, we will rely on the table we have been using.

    Example 4.15.

    Find the inverse Laplace transform of \(\ds F(s) = \frac{24}{s^5}.\)
    Solution.
    We look in the right column of the table and determine which of the forms the given function most closely matches. In this case, the form is like \(L3\), so we can find the inverse Laplace transform as follows:
    \begin{align*} f(t) \amp= \lap^{-1}\left\{ F(s) \right\}\\ \amp= \lap^{-1}\left\{ \frac{24}{s^5} \right\}\\ \amp= \lap^{-1}\left\{ \frac{4!}{s^{4+1}} \right\}\\ \amp= t^4 \quad (\knowl{./knowl/xref/LT-Table-L3.html}{\text{\(L3\)}}). \end{align*}
    Thus, we have shown that \(\ds \lap^{-1}\left\{ \frac{24}{s^5} \right\} = t^4.\)
    In the previous example, the function \(F(s)\) matched with one of the properties in the table exactly. This is rare, and we need to develop strategies for when the function is not an exact match.

    Example 4.16.

    Find the inverse Laplace transform of \(\ds G(s) = \frac{17}{s^4}.\)
    Solution.
    When we look in the right column of the table, this one seems to line up best with \(L3\)...but it’s not a perfect match. If we were going to use \(L3\), then \(n\) would have to be 3 (because this gives us the \(s^4\) that we have). We would have \(L3\) perfectly if we the numerator was \(3! = 6,\) but our numerator is 17. Luckily, we can use the linearity property to help. Since the 17 is not helpful to us, we can factor it out.
    \begin{align*} g(t) \amp= \lap^{-1}\{ G(s) \}\\ \amp= \lap^{-1}\left\{ \frac{17}{s^4} \right\}\\ \amp= 17 \cdot \lap^{-1}\left\{ \frac{1}{s^4} \right\} \end{align*}
    We have the 17 out of the way, so we are halfway done. Now we need to (carefully) get a 6 in that numerator. We will again lean on linearity, but we will also use a little trick mathematicians use from time to time: multiplying by 1. It’s always safe to multiply any expression by 1 because multiplying by 1 doesn’t change the value of an expression. The form of 1 that will be useful here is the form \(\ds \frac{6}{6},\) as below.
    \begin{align*} g(t) \amp= 17 \cdot \lap^{-1}\left\{ \frac{1}{s^4} \right\}\\ \amp= \frac{6}{6}\cdot 17 \cdot \lap^{-1}\left\{ \frac{1}{s^4} \right\}\\ \amp= \frac{1}{6}\cdot 17 \cdot \lap^{-1}\left\{ \frac{6}{s^4} \right\} \\ \amp= \frac{17}{6}\cdot \lap^{-1}\left\{ \frac{3!}{s^{3+1}} \right\}\\ \amp= \frac{17}{6}t^3. \end{align*}
    Thus, we have shown that \(\ds \lap^{-1}\left\{ \frac{17}{s^4} \right\} = \frac{17}{6}t^3.\)

    Example 4.17.

    Find \(\ds \lap^{-1} \left\{\frac{7}{s^2 + 25}\right\}.\)
    Solution.
    Our first goal is to identify which property in the table is going to be the closest match to the expression we are given. If we look in the right hand column of the Laplace transform table, the right choice seems to be either \(L4\) or \(L5\). The distinguishing factor between the two is whether or not we have an \(s\) in the numerator. In this case, we do not, so we aim to drive toward \(L4\).
    Note that we don’t have a perfect match with that property, so we will have to do a little manipulation, as we did in the previous example, to get our function to match up perfectly. In order for us to do that manipulation, we do need to have in mind what exactly we are aiming for. That is, do we want to make our expression like \(L4\) with \(b = 7\) or with \(b = 5\text{?}\) If we think about the kind of manipulation we did in the previous example, where we multiplied by a form of 1, we might guess that it’s easier to manipulate the constant in the numerator. (If we manipulate constants in the denominator, we need to worry about two terms).
    If we agree that we would like to match \(L4\) with \(b = 5,\) then we proceed using linearity in much the same way we did the previous example.
    \begin{align*} \lap^{-1} \left\{\frac{7}{s^2 + 25}\right\} \amp= 7\cdot \lap^{-1} \left\{\frac{1}{s^2 + 25}\right\}\\ \amp= 7\cdot \lap^{-1} \left\{\frac{1}{s^2 + 5^2}\right\}\\ \amp= \frac{5}{5}\cdot 7\cdot \lap^{-1} \left\{\frac{1}{s^2 + 5^2}\right\}\\ \amp= \frac{1}{5}\cdot 7\cdot \lap^{-1} \left\{\frac{5}{s^2 + 5^2}\right\}\\ \amp= \frac{7}{5}\sin(5t). \end{align*}

    Example 4.18.

    Find the inverse Laplace transform of \(\ds G(s) = \frac{2}{(s+7)^5}\text{.}\)
    Solution.
    We aim to make this function match \(L6\), as follows.
    \begin{align*} g(t) \amp= \lap^{-1}\left\{G(s) \right\}\\ \amp= \lap^{-1}\left\{ \frac{2}{(s+7)^5} \right\}\\ \amp= 2 \cdot \lap^{-1}\left\{ \frac{1}{(s+7)^5} \right\}\\ \amp= 2 \cdot \lap^{-1}\left\{ \frac{1}{(s+7)^{4+1}} \right\}\\ \amp= \frac{4!}{4!}\cdot 2 \cdot \lap^{-1}\left\{ \frac{1}{(s+7)^{4+1}} \right\}\\ \amp= \frac{1}{4!}\cdot 2 \cdot \lap^{-1}\left\{ \frac{4!}{\Big(s-(-7)\Big)^{4+1}} \right\}\\ \amp= \frac{1}{12} e^{-7t}t^4. \end{align*}

    We have shown how you can manipulate functions of \(s\) to match properties in the Laplace transform table by taking advantage of the linearity of the Laplace transform. However, there are other algebraic manipulations you can (and sometimes must!) do, so we look at more examples here.

    Example 4.19.

    Find the inverse Laplace transform of \(\ds H(s) = \frac{s-1}{s^2 - 2s + 5}\text{.}\)
    Solution.
    As a general guideline, you are advised to consider the denominator when determining trying to determine which property or properties in the Laplace transform table might be appropriate. In this case, we have a second-degree polynomial in the denominator. None of the properties in the table looks exactly the same, but \(L7\) and \(L8\) do have second-degree polynomials in the denominator. We just need to make our expression look like the form of those denominators, \((s - a)^2 + b^2.\) This entails completing the square.
    Margin note: if you need to review completing the square, look HERE.
    \begin{align*} H(s) \amp= \frac{s-1}{s^2 - 2s + 5}\\ \amp= \frac{s-1}{(s^2 - 2s) + 5}\\ \amp= \frac{s-1}{(s^2 - 2s + 1) + 5 - 1}\\ \amp= \frac{s-1}{(s - 1)^2 + 4}. \end{align*}
    Margin note: Notice that the sign on the 4 is positive. If the sign had been negative, this would not be a perfect match with the table and we’d have to use a different strategy. See Example (insert number here).
    Now that we have changed the way \(H(s)\) looks, we see that it is a perfect match with \(L8\), with \(a = 1\) and \(b = 2.\) Thus, we can take its inverse Laplace transform.
    \begin{align*} h(t) \amp= \lap^{-1} \left\{ H(s) \right\}\\ \amp= \lap^{-1} \left\{\frac{s-1}{(s - 1)^2 + 4} \right\}\\ \amp= \lap^{-1} \left\{\frac{s-1}{(s - 1)^2 + 2^2} \right\}\\ \amp= e^{t}\cos(2t). \end{align*}

    Example 4.20.

    Find the inverse Laplace transform of \(\ds P(s) = \frac{3s+2}{s^2 + 2s + 10}\text{.}\)
    Solution.
    As in the previous example, we have a second-degree polynomial in the denominator, so we might try to make this expression match \(L7\) and/or \(L8\) in the table. We begin by completing the square.
    \begin{align*} P(s) \amp= \frac{3s+2}{s^2 + 2s + 10}\\ \amp= \frac{3s+2}{(s^2 + 2s) + 10}\\ \amp= \frac{3s+2}{(s^2 + 2s + 1) + 10 - 1}\\ \amp= \frac{3s+2}{(s+1)^2 + 9}\\ \amp= \frac{3s+2}{(s+1)^2 + 3^2}. \end{align*}
    Now we have the denominator in a format so that it matches with \(L7\) or \(L8\), but unlike the previous example, the numerator is not an exact match with either \(L7\) or \(L8\). We note that the numerator does have an \(s\) in it, so we will start by aiming to see if we can make it a match with \(L8\). If we did have a perfect match with \(L8\), then we would have \(s+1\) in the numerator. Let’s focus on the numerator for a moment and consider the following.
    \begin{equation*} 3s+2 = 3(s+1) + \msquare \end{equation*}
    In order to make the expressions equal, what value would the \(\ds \msquare\) have? If we distribute the three, we would have
    \begin{equation*} 3s+2 = 3s + 3 + \msquare, \end{equation*}
    and thus it’s clear that \(\msquare\) should be equal to \(-1\text{.}\) Let’s use this to rewrite \(P(s)\) as follows.
    \begin{align*} P(s) \amp= \frac{3s+2}{(s+1)^2 + 3^2}\\ \amp= \frac{3(s+1) - 1}{(s+1)^2 + 3^2}\\ \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{(s+1)^2 + 3^2}. \end{align*}
    The advantage to writing \(P(s)\) this way is that the first fraction is now a perfect match for \(L8\). We need to find a perfect match for that second fraction. It looks close to \(L7\), but not perfect. In order to be a perfect match for \(L7\), we would need a 3 in the numerator. Luckily, we can multiply by one (in the form of \(\frac{3}{3})\) in order to get a 3 in the numerator, as follows.
    \begin{align*} P(s) \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{(s+1)^2 + 3^2}\\ \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{3}{3}\cdot \frac{1}{(s+1)^2 + 3^2}\\ \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{3}\cdot \frac{3}{(s+1)^2 + 3^2}. \end{align*}
    Now that we have done some algebraic manipulation, we are now ready to take the inverse Laplace transform.
    \begin{align*} p(t) \amp= \lap^{-1}\left\{ P(s) \right\}\\ \amp= \lap^{-1}\left\{3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{3}\cdot \frac{3}{(s+1)^2 + 3^2} \right\} \\ \amp= 3\lap^{-1}\left\{ \frac{s+1}{(s+1)^2 + 3^2} \right\} - \frac{1}{3}\lap^{-1}\left\{\frac{3}{(s+1)^2 + 3^2} \right\} (\knowl{./knowl/xref/LT-Table-L9.html}{\text{\(L9\)}}) \\ \amp= 3e^{-t}\cos(3t) - \frac{1}{3}e^{-t}\sin(3t). \end{align*}

    Example 4.21.

    Find \(\ds \lap^{-1}\left\{ \frac{s-11}{s^2 - 6s +14} \right\}.\)
    Solution.
    We complete the square in the denominator in the function
    \begin{align*} \frac{s-11}{s^2 - 6s - 3} \amp= \frac{s-11}{(s^2 - 6s) + 14}\\ \amp= \frac{s-11}{(s^2 - 6s + 9) + 14 - 9}\\ \amp= \frac{s-11}{(s - 3)^2 + 5}. \end{align*}
    This does not immediately appear to match \(L7\) or \(L8\), but we can actually write the denominator as \((s-11)^2 + (\sqrt{5})^2\) and proceed as in the previous examples, working toward matching \(L7\) and \(L8\).
    \begin{align*} \amp= \lap^{-1}\left\{ \frac{s-11}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{(s-3) -8}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= e^{3t}\cos(\sqrt{5}t)- \frac{8}{\sqrt{5}}e^{3t}\sin(\sqrt{5}t) \end{align*}

    Example 4.22.

    Find \(\ds \lap^{-1}\left\{ \frac{s+9}{s^2 - 2s - 3} \right\}.\)
    Solution.
    As in the previous examples, the denominator is a second-degree polynomial; therefore it is sensible for us to begin by completing the square in the denominator as we did in the previous two examples.
    \begin{align*} \frac{s+9}{s^2 - 2s - 3} \amp= \frac{s+9}{(s^2 - 2s) - 3}\\ \amp= \frac{s+9}{(s^2 - 2s + 1) - 3 - 1}\\ \amp= \frac{s+9}{(s-1)^2 - 4}\\ \amp= \frac{s+9}{(s-1)^2 - 2^2}. \end{align*}
    Take a careful look at the denominator here. It’s really close to matching \(L7\) or \(L8\), but it is not a match because of the negative sign in front of the \(2^2.\) We need to change course when this happens. Another algebraic manipulation that we might consider is a partial fraction decomposition.
    Margin note: If you need to review partial fraction decomposition, go HERE.
    We revert to the original expression, but this time, instead of completing the square, we factor the denominator.
    \begin{equation*} \frac{s+9}{s^2 - 2s - 3} = \frac{s+9}{(s-3)(s+1)}. \end{equation*}
    Since each of the factors in the denominator is a distinct linear factor, we know that the form of the partial fraction decomposition is
    \begin{equation*} \frac{s+9}{(s-3)(s+1)} = \frac{A}{s-3} + \frac{B}{s+1}. \end{equation*}
    Our next goal is to determine the coefficients \(A\) and \(B\) in this equations. There are multiple ways to achieve this and we demonstrate just one here. We multiply both sides of the equation by the least common denominator, \((s-3)(s+1)\text{,}\) and then expand and collect like terms, as shown.
    \begin{align*} \amp= \frac{A}{s-3}\cdot (s-3)(s+1) + \frac{B}{s+1} \cdot (s-3)(s+1)\\ s+9 \amp= A(s+1) + B(s-3)\\ s+9 \amp= As + A + Bs - 3B\\ s+9 \amp= (As + Bs) + (A - 3B)\\ 1s+9 \amp= (A+B)s + (A - 3B). \end{align*}
    At this point, we have a polynomial on the left hand side and a polynomial on the right hand side. The only way these can be equal to each other is if the corresponding coefficients are equal. That is, the coefficient on \(s\) on the left hand side is 1, while the coefficient on \(s\) on the right hand side of the equation is \(A+B\text{.}\) Since the polynomials are equal, we know that these are equal. That is, \(A + B = 1.\) Similarly, if we equate the constants, we have \(A - 3B = 9.\) Thus, we have the following system of two linear equations in terms of two unknown variables, \(A\) and \(B\text{.}\)
    \begin{align*} A + B \amp= 1\\ A - 3B \amp= 9. \end{align*}
    There are many ways to solve such an equation, and you are encouraged to choose the solution technique you like the most. Here we will solve the first equation for \(A\text{,}\) and then substitute into the second equation,
    \begin{align*} A + B \amp= 1 \amp A - 3B \amp= 9\\ A \amp= 1 - B \amp \amp\\ \amp \amp (1 - B) - 3B \amp= 9\\ \amp \amp 1 - 4B \amp= 9\\ \amp \amp -4B \amp= 8\\ \amp \amp B \amp= -2\\ A \amp= 1 - (-2) \amp \amp\\ \amp= 3 \amp \amp \end{align*}
    hence we have
    \begin{equation*} \frac{s+9}{(s-3)(s+1)} = \frac{3}{s-3} + \frac{-2}{s+1}. \end{equation*}
    Remember that our goal is to take the inverse Laplace transform. Our algebraic manipulation was helpful because we took a more complex expression and rewrote it as two simpler fractions. We can now use \(L2\) to find the inverse Laplace transform as follows.
    \begin{align*} \amp= \lap^{-1}\left\{ \frac{3}{s-3} + \frac{-2}{s+1} \right\}\\ \amp=3 \lap^{-1}\left\{ \frac{1}{s-3} \right\} - 2 \lap^{-1}\left\{ \frac{1}{s+1} \right\}\\ \amp= 3e^{3t} - 2e^{-t} \end{align*}
    In summary, when we want to take the inverse Laplace transform of a rational function with a second-degree polynomial in the denominator, we may complete the square or we may do a partial fraction decomposition. How will we know which is appropriate? Here are a few guidelines for you to consider.
    • Does the denominator factor in an obvious way? If so, factor the denominator and do a partial fraction decomposition if necessary.
    • If the denominator does not factor in an obvious way, try completing the square.
    • If you end up with addition outside of the parentheses, as in \((s - a)^2 + b^2,\) then you should aim to match \(L7\) and/or \(L8\).
    • If instead you end up with subtraction outside the parentheses, as in \((s - a)^2 - b^2,\) then you should factor and do a partial fraction decomposition. You may consider using the quadratic formula if the factorization is not obvious to you.
    • If you end up with no terms outside the parentheses, as in \((s - a)^2,\) then use \(L6\).
    If you have a rational function where the denominator is of higher degree, then partial fraction decomposition should be used to break the single fraction into several simpler fractions.

    Example 4.23.

    Find the inverse Laplace transform of
    \begin{equation*} R(s) = \frac{4s^3 - 13s^2 + 74s + 27}{s^4 - 4s^3 + 14s^2 + 44s + 25} = \frac{4s^3 - 13s^2 + 74s + 27}{(s^2 - 6s + 25)(s+1)^2}. \end{equation*}
    Solution.
    Note that as the quadratic term in the denominator does not factor, the denominator contains an irreducible quadratic factor and a repeated linear factor. We’ll proceed by simplifying this complicated fraction with a Partial Fraction Decomposition of the form
    \begin{equation*} \frac{4s^3 - 13s^2 + 74s + 27}{(s^2 - 6s + 25)(s+1)^2} = \frac{As + B}{s^2 - 6s + 25} + \frac{C}{s+1} + \frac{D}{(s+1)^2}. \end{equation*}
    You may also consider using technology to find a partial fraction decomposition. You should get
    \begin{equation*} \frac{4s^3 - 13s^2 + 74s + 27}{(s^2 - 6s + 25)(s+1)^2} = \frac{s + 2}{s^2 - 6s + 25} + \frac{3}{s+1} + \frac{-2}{(s+1)^2}. \end{equation*}
    Click here for the details.
    With the partial fraction decomposition in hand, we are prepared to take the inverse Laplace transform, using the same types of algebraic manipulations demonstrated in the previous examples.
    \begin{align*} r(t) \amp= \lap^{-1}\left\{ R(s) \right\}\\ \amp= \lap^{-1}\left\{ \frac{s + 2}{s^2 - 6s + 25} + \frac{3}{s+1} + \frac{-2}{(s+1)^2} \right\} \\ \amp= \lap^{-1}\left\{ \frac{s + 2}{s^2 - 6s + 25}\right\} + 3\cdot \lap^{-1}\left\{ \frac{1}{s+1} \right\} - 2\cdot \lap^{-1}\left\{\frac{1}{(s+1)^2} \right\} \\ \amp= \lap^{-1}\left\{ \frac{s + 2}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= \lap^{-1}\left\{ \frac{(s - 3) + 5}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= \lap^{-1}\left\{ \frac{s - 3}{(s-3)^2 + 4^2}\right\} + 5 \lap^{-1}\left\{ \frac{1}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= e^{3t}\cos(4t) + \frac{5}{4}\cdot \lap^{-1}\left\{ \frac{4}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= e^{3t}\cos(4t) + \frac{5}{4}e^{3t}\sin(4t) + 3e^{-t} - 2te^{-t}. \end{align*}
    Let’s review the Laplace transform diagram to remember how we will leverage Laplace transforms to solve differential equations.
    The one last piece we need in order to solve DEs is to be able to take inverse Laplace transforms. We will focus on that piece now, and in the next section we will put all of our new skills together to solve DEs using Laplace transforms.
    When we take the inverse Laplace transform, we begin with a function of \(s\text{,}\) say \(Y(s)\text{,}\) and what we seek is a function of \(y(t)\) such that if we took the Laplace transform of \(y\text{,}\) we would get \(Y\text{.}\) That is, we seek to un-do the the Laplace transform. The notation looks similar:
    \begin{equation*} y(t) = \lap^{-1}\{ Y(s) \}, \end{equation*}
    and, as with the forward Laplace transform, the curly braces are not optional. Like the forward Laplace transform, the inverse Laplace transform is also a linear transform.
    In order to take inverse Laplace transforms, we will rely on the table we have been using.

    Example 4.24.

    Find the inverse Laplace transform of \(\ds F(s) = \frac{24}{s^5}.\)
    Solution.
    We look in the right column of the table and determine which of the forms the given function most closely matches. In this case, the form is like \(L3\), so we can find the inverse Laplace transform as follows:
    \begin{align*} f(t) \amp= \lap^{-1}\left\{ F(s) \right\}\\ \amp= \lap^{-1}\left\{ \frac{24}{s^5} \right\}\\ \amp= \lap^{-1}\left\{ \frac{4!}{s^{4+1}} \right\}\\ \amp= t^4 \quad (\knowl{./knowl/xref/LT-Table-L3.html}{\text{\(L3\)}}). \end{align*}
    Thus, we have shown that \(\ds \lap^{-1}\left\{ \frac{24}{s^5} \right\} = t^4.\)
    In the previous example, the function \(F(s)\) matched with one of the properties in the table exactly. This is rare, and we need to develop strategies for when the function is not an exact match.

    Example 4.25.

    Find the inverse Laplace transform of \(\ds G(s) = \frac{17}{s^4}.\)
    Solution.
    When we look in the right column of the table, this one seems to line up best with \(L3\)...but it’s not a perfect match. If we were going to use \(L3\), then \(n\) would have to be 3 (because this gives us the \(s^4\) that we have). We would have \(L3\) perfectly if we the numerator was \(3! = 6,\) but our numerator is 17. Luckily, we can use the linearity property to help. Since the 17 is not helpful to us, we can factor it out.
    \begin{align*} g(t) \amp= \lap^{-1}\{ G(s) \}\\ \amp= \lap^{-1}\left\{ \frac{17}{s^4} \right\}\\ \amp= 17 \cdot \lap^{-1}\left\{ \frac{1}{s^4} \right\} \end{align*}
    We have the 17 out of the way, so we are halfway done. Now we need to (carefully) get a 6 in that numerator. We will again lean on linearity, but we will also use a little trick mathematicians use from time to time: multiplying by 1. It’s always safe to multiply any expression by 1 because multiplying by 1 doesn’t change the value of an expression. The form of 1 that will be useful here is the form \(\ds \frac{6}{6},\) as below.
    \begin{align*} g(t) \amp= 17 \cdot \lap^{-1}\left\{ \frac{1}{s^4} \right\}\\ \amp= \frac{6}{6}\cdot 17 \cdot \lap^{-1}\left\{ \frac{1}{s^4} \right\}\\ \amp= \frac{1}{6}\cdot 17 \cdot \lap^{-1}\left\{ \frac{6}{s^4} \right\} \\ \amp= \frac{17}{6}\cdot \lap^{-1}\left\{ \frac{3!}{s^{3+1}} \right\}\\ \amp= \frac{17}{6}t^3. \end{align*}
    Thus, we have shown that \(\ds \lap^{-1}\left\{ \frac{17}{s^4} \right\} = \frac{17}{6}t^3.\)

    Example 4.26.

    Find \(\ds \lap^{-1} \left\{\frac{7}{s^2 + 25}\right\}.\)
    Solution.
    Our first goal is to identify which property in the table is going to be the closest match to the expression we are given. If we look in the right hand column of the Laplace transform table, the right choice seems to be either \(L4\) or \(L5\). The distinguishing factor between the two is whether or not we have an \(s\) in the numerator. In this case, we do not, so we aim to drive toward \(L4\).
    Note that we don’t have a perfect match with that property, so we will have to do a little manipulation, as we did in the previous example, to get our function to match up perfectly. In order for us to do that manipulation, we do need to have in mind what exactly we are aiming for. That is, do we want to make our expression like \(L4\) with \(b = 7\) or with \(b = 5\text{?}\) If we think about the kind of manipulation we did in the previous example, where we multiplied by a form of 1, we might guess that it’s easier to manipulate the constant in the numerator. (If we manipulate constants in the denominator, we need to worry about two terms).
    If we agree that we would like to match \(L4\) with \(b = 5,\) then we proceed using linearity in much the same way we did the previous example.
    \begin{align*} \lap^{-1} \left\{\frac{7}{s^2 + 25}\right\} \amp= 7\cdot \lap^{-1} \left\{\frac{1}{s^2 + 25}\right\}\\ \amp= 7\cdot \lap^{-1} \left\{\frac{1}{s^2 + 5^2}\right\}\\ \amp= \frac{5}{5}\cdot 7\cdot \lap^{-1} \left\{\frac{1}{s^2 + 5^2}\right\}\\ \amp= \frac{1}{5}\cdot 7\cdot \lap^{-1} \left\{\frac{5}{s^2 + 5^2}\right\}\\ \amp= \frac{7}{5}\sin(5t). \end{align*}

    Example 4.27.

    Find the inverse Laplace transform of \(\ds G(s) = \frac{2}{(s+7)^5}\text{.}\)
    Solution.
    We aim to make this function match \(L6\), as follows.
    \begin{align*} g(t) \amp= \lap^{-1}\left\{G(s) \right\}\\ \amp= \lap^{-1}\left\{ \frac{2}{(s+7)^5} \right\}\\ \amp= 2 \cdot \lap^{-1}\left\{ \frac{1}{(s+7)^5} \right\}\\ \amp= 2 \cdot \lap^{-1}\left\{ \frac{1}{(s+7)^{4+1}} \right\}\\ \amp= \frac{4!}{4!}\cdot 2 \cdot \lap^{-1}\left\{ \frac{1}{(s+7)^{4+1}} \right\}\\ \amp= \frac{1}{4!}\cdot 2 \cdot \lap^{-1}\left\{ \frac{4!}{\Big(s-(-7)\Big)^{4+1}} \right\}\\ \amp= \frac{1}{12} e^{-7t}t^4. \end{align*}

    We have shown how you can manipulate functions of \(s\) to match properties in the Laplace transform table by taking advantage of the linearity of the Laplace transform. However, there are other algebraic manipulations you can (and sometimes must!) do, so we look at more examples here.

    Example 4.28.

    Find the inverse Laplace transform of \(\ds H(s) = \frac{s-1}{s^2 - 2s + 5}\text{.}\)
    Solution.
    As a general guideline, you are advised to consider the denominator when determining trying to determine which property or properties in the Laplace transform table might be appropriate. In this case, we have a second-degree polynomial in the denominator. None of the properties in the table looks exactly the same, but \(L7\) and \(L8\) do have second-degree polynomials in the denominator. We just need to make our expression look like the form of those denominators, \((s - a)^2 + b^2.\) This entails completing the square.
    Margin note: if you need to review completing the square, look HERE.
    \begin{align*} H(s) \amp= \frac{s-1}{s^2 - 2s + 5}\\ \amp= \frac{s-1}{(s^2 - 2s) + 5}\\ \amp= \frac{s-1}{(s^2 - 2s + 1) + 5 - 1}\\ \amp= \frac{s-1}{(s - 1)^2 + 4}. \end{align*}
    Margin note: Notice that the sign on the 4 is positive. If the sign had been negative, this would not be a perfect match with the table and we’d have to use a different strategy. See Example (insert number here).
    Now that we have changed the way \(H(s)\) looks, we see that it is a perfect match with \(L8\), with \(a = 1\) and \(b = 2.\) Thus, we can take its inverse Laplace transform.
    \begin{align*} h(t) \amp= \lap^{-1} \left\{ H(s) \right\}\\ \amp= \lap^{-1} \left\{\frac{s-1}{(s - 1)^2 + 4} \right\}\\ \amp= \lap^{-1} \left\{\frac{s-1}{(s - 1)^2 + 2^2} \right\}\\ \amp= e^{t}\cos(2t). \end{align*}

    Example 4.29.

    Find the inverse Laplace transform of \(\ds P(s) = \frac{3s+2}{s^2 + 2s + 10}\text{.}\)
    Solution.
    As in the previous example, we have a second-degree polynomial in the denominator, so we might try to make this expression match \(L7\) and/or \(L8\) in the table. We begin by completing the square.
    \begin{align*} P(s) \amp= \frac{3s+2}{s^2 + 2s + 10}\\ \amp= \frac{3s+2}{(s^2 + 2s) + 10}\\ \amp= \frac{3s+2}{(s^2 + 2s + 1) + 10 - 1}\\ \amp= \frac{3s+2}{(s+1)^2 + 9}\\ \amp= \frac{3s+2}{(s+1)^2 + 3^2}. \end{align*}
    Now we have the denominator in a format so that it matches with \(L7\) or \(L8\), but unlike the previous example, the numerator is not an exact match with either \(L7\) or \(L8\). We note that the numerator does have an \(s\) in it, so we will start by aiming to see if we can make it a match with \(L8\). If we did have a perfect match with \(L8\), then we would have \(s+1\) in the numerator. Let’s focus on the numerator for a moment and consider the following.
    \begin{equation*} 3s+2 = 3(s+1) + \msquare \end{equation*}
    In order to make the expressions equal, what value would the \(\ds \msquare\) have? If we distribute the three, we would have
    \begin{equation*} 3s+2 = 3s + 3 + \msquare, \end{equation*}
    and thus it’s clear that \(\msquare\) should be equal to \(-1\text{.}\) Let’s use this to rewrite \(P(s)\) as follows.
    \begin{align*} P(s) \amp= \frac{3s+2}{(s+1)^2 + 3^2}\\ \amp= \frac{3(s+1) - 1}{(s+1)^2 + 3^2}\\ \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{(s+1)^2 + 3^2}. \end{align*}
    The advantage to writing \(P(s)\) this way is that the first fraction is now a perfect match for \(L8\). We need to find a perfect match for that second fraction. It looks close to \(L7\), but not perfect. In order to be a perfect match for \(L7\), we would need a 3 in the numerator. Luckily, we can multiply by one (in the form of \(\frac{3}{3})\) in order to get a 3 in the numerator, as follows.
    \begin{align*} P(s) \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{(s+1)^2 + 3^2}\\ \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{3}{3}\cdot \frac{1}{(s+1)^2 + 3^2}\\ \amp= 3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{3}\cdot \frac{3}{(s+1)^2 + 3^2}. \end{align*}
    Now that we have done some algebraic manipulation, we are now ready to take the inverse Laplace transform.
    \begin{align*} p(t) \amp= \lap^{-1}\left\{ P(s) \right\}\\ \amp= \lap^{-1}\left\{3 \cdot \frac{s+1}{(s+1)^2 + 3^2} - \frac{1}{3}\cdot \frac{3}{(s+1)^2 + 3^2} \right\} \\ \amp= 3\lap^{-1}\left\{ \frac{s+1}{(s+1)^2 + 3^2} \right\} - \frac{1}{3}\lap^{-1}\left\{\frac{3}{(s+1)^2 + 3^2} \right\} (\knowl{./knowl/xref/LT-Table-L9.html}{\text{\(L9\)}}) \\ \amp= 3e^{-t}\cos(3t) - \frac{1}{3}e^{-t}\sin(3t). \end{align*}

    Example 4.30.

    Find \(\ds \lap^{-1}\left\{ \frac{s-11}{s^2 - 6s +14} \right\}.\)
    Solution.
    We complete the square in the denominator in the function
    \begin{align*} \frac{s-11}{s^2 - 6s - 3} \amp= \frac{s-11}{(s^2 - 6s) + 14}\\ \amp= \frac{s-11}{(s^2 - 6s + 9) + 14 - 9}\\ \amp= \frac{s-11}{(s - 3)^2 + 5}. \end{align*}
    This does not immediately appear to match \(L7\) or \(L8\), but we can actually write the denominator as \((s-11)^2 + (\sqrt{5})^2\) and proceed as in the previous examples, working toward matching \(L7\) and \(L8\).
    \begin{align*} \amp= \lap^{-1}\left\{ \frac{s-11}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{(s-3) -8}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= \lap^{-1}\left\{ \frac{s-3}{(s-3)^2 + (\sqrt{5})^2} \right\}\\ \amp= e^{3t}\cos(\sqrt{5}t)- \frac{8}{\sqrt{5}}e^{3t}\sin(\sqrt{5}t) \end{align*}

    Example 4.31.

    Find \(\ds \lap^{-1}\left\{ \frac{s+9}{s^2 - 2s - 3} \right\}.\)
    Solution.
    As in the previous examples, the denominator is a second-degree polynomial; therefore it is sensible for us to begin by completing the square in the denominator as we did in the previous two examples.
    \begin{align*} \frac{s+9}{s^2 - 2s - 3} \amp= \frac{s+9}{(s^2 - 2s) - 3}\\ \amp= \frac{s+9}{(s^2 - 2s + 1) - 3 - 1}\\ \amp= \frac{s+9}{(s-1)^2 - 4}\\ \amp= \frac{s+9}{(s-1)^2 - 2^2}. \end{align*}
    Take a careful look at the denominator here. It’s really close to matching \(L7\) or \(L8\), but it is not a match because of the negative sign in front of the \(2^2.\) We need to change course when this happens. Another algebraic manipulation that we might consider is a partial fraction decomposition.
    Margin note: If you need to review partial fraction decomposition, go HERE.
    We revert to the original expression, but this time, instead of completing the square, we factor the denominator.
    \begin{equation*} \frac{s+9}{s^2 - 2s - 3} = \frac{s+9}{(s-3)(s+1)}. \end{equation*}
    Since each of the factors in the denominator is a distinct linear factor, we know that the form of the partial fraction decomposition is
    \begin{equation*} \frac{s+9}{(s-3)(s+1)} = \frac{A}{s-3} + \frac{B}{s+1}. \end{equation*}
    Our next goal is to determine the coefficients \(A\) and \(B\) in this equations. There are multiple ways to achieve this and we demonstrate just one here. We multiply both sides of the equation by the least common denominator, \((s-3)(s+1)\text{,}\) and then expand and collect like terms, as shown.
    \begin{align*} \amp= \frac{A}{s-3}\cdot (s-3)(s+1) + \frac{B}{s+1} \cdot (s-3)(s+1)\\ s+9 \amp= A(s+1) + B(s-3)\\ s+9 \amp= As + A + Bs - 3B\\ s+9 \amp= (As + Bs) + (A - 3B)\\ 1s+9 \amp= (A+B)s + (A - 3B). \end{align*}
    At this point, we have a polynomial on the left hand side and a polynomial on the right hand side. The only way these can be equal to each other is if the corresponding coefficients are equal. That is, the coefficient on \(s\) on the left hand side is 1, while the coefficient on \(s\) on the right hand side of the equation is \(A+B\text{.}\) Since the polynomials are equal, we know that these are equal. That is, \(A + B = 1.\) Similarly, if we equate the constants, we have \(A - 3B = 9.\) Thus, we have the following system of two linear equations in terms of two unknown variables, \(A\) and \(B\text{.}\)
    \begin{align*} A + B \amp= 1\\ A - 3B \amp= 9. \end{align*}
    There are many ways to solve such an equation, and you are encouraged to choose the solution technique you like the most. Here we will solve the first equation for \(A\text{,}\) and then substitute into the second equation,
    \begin{align*} A + B \amp= 1 \amp A - 3B \amp= 9\\ A \amp= 1 - B \amp \amp\\ \amp \amp (1 - B) - 3B \amp= 9\\ \amp \amp 1 - 4B \amp= 9\\ \amp \amp -4B \amp= 8\\ \amp \amp B \amp= -2\\ A \amp= 1 - (-2) \amp \amp\\ \amp= 3 \amp \amp \end{align*}
    hence we have
    \begin{equation*} \frac{s+9}{(s-3)(s+1)} = \frac{3}{s-3} + \frac{-2}{s+1}. \end{equation*}
    Remember that our goal is to take the inverse Laplace transform. Our algebraic manipulation was helpful because we took a more complex expression and rewrote it as two simpler fractions. We can now use \(L2\) to find the inverse Laplace transform as follows.
    \begin{align*} \amp= \lap^{-1}\left\{ \frac{3}{s-3} + \frac{-2}{s+1} \right\}\\ \amp=3 \lap^{-1}\left\{ \frac{1}{s-3} \right\} - 2 \lap^{-1}\left\{ \frac{1}{s+1} \right\}\\ \amp= 3e^{3t} - 2e^{-t} \end{align*}
    In summary, when we want to take the inverse Laplace transform of a rational function with a second-degree polynomial in the denominator, we may complete the square or we may do a partial fraction decomposition. How will we know which is appropriate? Here are a few guidelines for you to consider.
    • Does the denominator factor in an obvious way? If so, factor the denominator and do a partial fraction decomposition if necessary.
    • If the denominator does not factor in an obvious way, try completing the square.
    • If you end up with addition outside of the parentheses, as in \((s - a)^2 + b^2,\) then you should aim to match \(L7\) and/or \(L8\).
    • If instead you end up with subtraction outside the parentheses, as in \((s - a)^2 - b^2,\) then you should factor and do a partial fraction decomposition. You may consider using the quadratic formula if the factorization is not obvious to you.
    • If you end up with no terms outside the parentheses, as in \((s - a)^2,\) then use \(L6\).
    If you have a rational function where the denominator is of higher degree, then partial fraction decomposition should be used to break the single fraction into several simpler fractions.

    Example 4.32.

    Find the inverse Laplace transform of
    \begin{equation*} R(s) = \frac{4s^3 - 13s^2 + 74s + 27}{s^4 - 4s^3 + 14s^2 + 44s + 25} = \frac{4s^3 - 13s^2 + 74s + 27}{(s^2 - 6s + 25)(s+1)^2}. \end{equation*}
    Solution.
    Note that as the quadratic term in the denominator does not factor, the denominator contains an irreducible quadratic factor and a repeated linear factor. We’ll proceed by simplifying this complicated fraction with a Partial Fraction Decomposition of the form
    \begin{equation*} \frac{4s^3 - 13s^2 + 74s + 27}{(s^2 - 6s + 25)(s+1)^2} = \frac{As + B}{s^2 - 6s + 25} + \frac{C}{s+1} + \frac{D}{(s+1)^2}. \end{equation*}
    You may also consider using technology to find a partial fraction decomposition. You should get
    \begin{equation*} \frac{4s^3 - 13s^2 + 74s + 27}{(s^2 - 6s + 25)(s+1)^2} = \frac{s + 2}{s^2 - 6s + 25} + \frac{3}{s+1} + \frac{-2}{(s+1)^2}. \end{equation*}

    With the partial fraction decomposition in hand, we are prepared to take the inverse Laplace transform, using the same types of algebraic manipulations demonstrated in the previous examples.
    \begin{align*} r(t) \amp= \lap^{-1}\left\{ R(s) \right\}\\ \amp= \lap^{-1}\left\{ \frac{s + 2}{s^2 - 6s + 25} + \frac{3}{s+1} + \frac{-2}{(s+1)^2} \right\} \\ \amp= \lap^{-1}\left\{ \frac{s + 2}{s^2 - 6s + 25}\right\} + 3\cdot \lap^{-1}\left\{ \frac{1}{s+1} \right\} - 2\cdot \lap^{-1}\left\{\frac{1}{(s+1)^2} \right\} \\ \amp= \lap^{-1}\left\{ \frac{s + 2}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= \lap^{-1}\left\{ \frac{(s - 3) + 5}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= \lap^{-1}\left\{ \frac{s - 3}{(s-3)^2 + 4^2}\right\} + 5 \lap^{-1}\left\{ \frac{1}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= e^{3t}\cos(4t) + \frac{5}{4}\cdot \lap^{-1}\left\{ \frac{4}{(s-3)^2 + 4^2}\right\} + 3e^{-t} - 2te^{-t}\\ \amp= e^{3t}\cos(4t) + \frac{5}{4}e^{3t}\sin(4t) + 3e^{-t} - 2te^{-t}. \end{align*}
    When we study Laplace Transforms, we will frequently want to manipulate an algebraic expression to make it match a given form.
    If you prefer to see a video of this example, click here
    Suppose we have the expression
    \begin{equation*} \frac{1}{s^2 + 25}, \end{equation*}
    and we think it looks much like the form
    \begin{equation*} \frac{b}{s^2 + b^2}. \end{equation*}
    It’s not a perfect match, though. Based on the denominator, we might guess \(b = 5.\)
    Notice that this would make the denominator a perfect match,
    \begin{equation*} \frac{1}{s^2 + 25} = \frac{1}{s^2 + 5^2}. \end{equation*}
    However, the numerator still isn’t quite right. It would be correct if we would multiply the numerator by 5. But if we multiply the numerator by 5, we would change the expression. In order to NOT change the expression, we will instead multiply by \(\frac{5}{5},\) as follows:
    \begin{align*} \frac{1}{s^2 + 25} \amp= \frac{1}{s^2 + 5^2} \\ \amp= \frac{5}{5}\cdot \frac{1}{s^2 + 5^2} \\ \amp= \frac{1}{5}\cdot \frac{5}{s^2 + 5^2} \\ \amp= \frac{1}{5}\cdot \frac{b}{s^2 + b^2}, \text{ where } b=5 \end{align*}
    Notice that when we multiply by \(\frac{5}{5}\text{,}\) we are just multiplying by one, so we don’t change the value of the expression.
    Maybe write now you’re wondering what the point is. Being able to use appropriate algebra to "match" forms is really important when we work with Laplace Transforms. Since it’s really just algebra, now is a great time to practice that skill--so when we are in the middle of studying Laplace Transforms, you can just focus on the "new" stuff.

        Answer.
