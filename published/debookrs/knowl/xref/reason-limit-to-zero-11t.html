We’ve seen the second limit, but let’s review it again. If \(s \gt 0\text{,}\) then \(sb \to \infty\) and
\begin{equation*} \lim_{b \to \infty} e^{-sb} = \lim_{b \to \infty} \frac{1}{\us{\infty}{\us{\downarrow}{\boxed{e^{sb}}}}} = 0 \end{equation*}
The first limit requires L’Hopital’s rule. Remember, when using L’Hopital’s rule here, we must take derivatives with respect to \(b\) (where \(s\) is constant). Here are the details:
\begin{align*} \lim_{b \to \infty}be^{-sb} \amp= \lim_{b \to \infty}\frac{\os{\infty}{\os{\uparrow}{\boxed{b}}}}{\us{\infty}{\us{\downarrow}{\boxed{e^{sb}}}}} \,\us{LH}{=}\, \lim_{b \to \infty}\frac{1}{se^{sb}} = \frac{1}{s}\lim_{b \to \infty}\frac{1}{\us{\infty}{\us{\downarrow}{\boxed{e^{sb}}}}} = 0, \quad s \gt 0 \end{align*}
