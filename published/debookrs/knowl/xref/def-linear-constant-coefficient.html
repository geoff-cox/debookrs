Definition 3.15.

A linear differential equation (18) is said to have constant coefficients if each \(a_{n}(x),\ \cdots , a_2(x),\ a_1(x),\ a_0(x)\) is constant and (18) becomes
\begin{equation} a_n\ y^{(n)} + \cdots + a_2\ y'' + a_1\ y' + a_0\ y = f(x),\tag{19} \end{equation}
When a function, \(a_0(x)\text{,}\) is constant, we usually drop the function notation and just write \(a_0(x) = a_0\text{.}\)
