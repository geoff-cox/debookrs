Example 3.10.

Find any \(r\) values that make \(y = e^{rx}\) a solution to the LHCC equation
\begin{equation} y'' - 9y = 0.\tag{15} \end{equation}
Solution.
Approach this as if you were verifying that \(y = e^{rx}\) is a solution to (15). So, plug it into the equation and check if left and right-hand sides are equal.
Since we need \(y^{\prime\prime}\text{,}\) let’s compute this first.
\(\displaystyle y' = e^{rx} \cdot (r) = re^{rx}\) (chain rule)
\(\displaystyle y'' = [re^{rx}]^{\prime} = r[e^{rx}]^{\prime} = r(e^{rx} \cdot (r)) = r^2e^{rx}\) (chain rule)
Plugging \(y^{\prime\prime}\) and \(y\) into the left side of (15) gives
\(\displaystyle r^2e^{rx} - 9e^{rx} = 0 \)
\(\displaystyle (r^2 - 9)\ e^{rx} = 0\) (factor out \(e^{rx}\))
Now, \(y = e^{rx}\) is only a solution if we can show LHS = RHS (\(0 = 0\) in this case). It should be clear that this depends on the values of \(r\text{.}\) This is just an algebra problem where we must solve for \(r\text{.}\)
\(\displaystyle r^2 - 9 = 0,\) \(\displaystyle e^{rx} = 0\) (set each factor to 0)
\(r = \pm 3,\) no solution
 8 
Attempting to solve \(\displaystyle e^{rx} = 0\) for \(r\) by applying \(\ln\) would give
\begin{equation*} \displaystyle \ln(e^{rx}) = \ln(0) \, \leftarrow \text{not defined} \end{equation*}
So, the only \(r\) values that make the LHS = RHS is \(r = -3, 3\text{.}\)
