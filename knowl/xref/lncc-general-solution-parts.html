LNCC General Solution Parts.

The linear nonhomogeneous constant coefficient equation
\begin{equation} a_n y^{(n)} + \cdots + a_1 y' + a_0 y = f(x)\tag{52} \end{equation}
has a two-part general solution that can be written as
\begin{equation} y = y_h + y_p\text{.}\tag{53} \end{equation}
Plugging this into (52), we can separate \(y_h\) and \(y_p\) like so
\begin{equation*} \us{\large \text{terms of } \knowl{./knowl/xref/lncc-general-solution.html}{(53)} \text{ that give } 0}{\ub{a_n y_h^{(n)} + \cdots + a_1 y_h' + a_0 y_h}} + \us{\large \text{terms of } \knowl{./knowl/xref/lncc-general-solution.html}{(53)} \text{ that give } f(x)}{\ub{a_n y_p^{(n)} + \cdots + a_1 y_p' + a_0 y_p}} = f(x)\text{.} \end{equation*}
We call \(y_h\) the homogeneous part of (53) and find it by solving
\begin{equation*} a_n y^{(n)} + \cdots + a_2 y'' + a_1 y' + a_0 y = 0\text{.} \end{equation*}
The part, \(y_p\text{,}\) is the particular solution of (53), which accounts for \(f(x)\text{.}\)
