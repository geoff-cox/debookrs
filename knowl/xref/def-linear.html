Definition 15.

A differential equation is linear if it contains only linear terms. That is, it can be expressed as
\begin{equation} \underset{\text{linear term}}{\underbrace{a_n(t)y^{(n)}}} + \cdots + \underset{\text{linear term}}{\underbrace{a_2(t)y''}} + \underset{\text{linear term}}{\underbrace{a_1(t)y'}} + \underset{\text{linear term}}{\underbrace{a_0(t)y}} = \underset{\text{linear term}}{\underbrace{f(t).}}\tag{3} \end{equation}
If just one of the terms in the equation is nonlinear, then the entire differential equation is nonlinear.
