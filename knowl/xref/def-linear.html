Definition 1.1.

A differential equation is linear if it contains only linear terms. That is, it can be written as
\begin{equation} \underset{\text{linear term}}{\underbrace{a_n(t)y^{(n)}}} + \cdots + \underset{\text{linear term}}{\underbrace{a_2(t)y''}} + \underset{\text{linear term}}{\underbrace{a_1(t)y'}} + \underset{\text{linear term}}{\underbrace{a_0(t)y}} = \underset{\text{linear term}}{\underbrace{f(t).}}\tag{1} \end{equation}
If one of the terms is not linear, then the differential equation is nonlinear.
