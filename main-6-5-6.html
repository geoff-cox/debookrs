    Section 5.4 IF: Step-By-Step

    Now that we are familiar with the strategy behind the integrating factor method, letâ€™s streamline the process into a sequence of steps, summarized below.

    Method 3. Integrating Factor.

    The general solution to the first-order linear DE in the standard form
    \begin{equation} \frac{dy}{dx} + P(x)\ y = Q(x),\tag{22} \end{equation}
    can be determined through the following steps:
    Step 1: Find the Integrating Factor
    \begin{equation*} \mu = e^{\int P(x) dx}\text{.} \end{equation*}
    Step 2: Rewrite the DE
    Multiply the equation by \(\mu\text{,}\) and rewrite it in the form
    \begin{equation} \frac{d}{dx}\left[\mu\ y\right] = \mu\ Q(x). \tag{23} \end{equation}
    Step 3: Apply Direct Integration
    Integrate both sides of (23) and solve for \(y\text{.}\)
    Now letâ€™s apply this to some examples.

    Example 18.

    Find the general solution to the differential equation
    \begin{equation*} y' + 6y = 1\text{.} \end{equation*}
    Solution.
    Step 1: Find the Integrating Factor. \(\ds \quad \mu = e^{\int 6\ dx} = e^{6x}\)
    Step 2: Rewrite the DE.
    Multiply by \(\mu\text{:}\)
    \(\ds e^{6x}y' + 6e^{6x}y = e^{6x} \)
    Rewrite:
    \(\ds \frac{d}{dx}\left[e^{6x}y\right] = e^{6x} \)
    Step 3: Apply Direct Integration
    \begin{align*} e^{6x}y \amp = \int e^{6x}\ dx \\ y \amp = \frac{1}{6}e^{-6x} + C. \end{align*}

    Example 19.

    Solve
    \begin{equation*} -\frac{4}{t^2} \cdot M = t^5 - \frac{1}{t} \cdot\frac{dM}{dt} + 1. \end{equation*}
    Solution.
    We always begin by classifying the DE. You should confirm that this DE is first order and linear, and therefore it is appropriate to use our integrating factor technique. Keep in mind that our goal is to find an unknown function \(M(t)\) that satisfies this equation.
    We work first to get the DE in an appropriate form so that we can un-do the product rule. We begin by getting the dependent variable, \(M\) and its derivative on the same side of the equation, and then we multiply both sides of the equation by \(t\) so that the coefficient on the derivative is 1.
    \begin{align} \frac{1}{t}\cdot \frac{dM}{dt} - \frac{4}{t^2} \cdot M \amp = t^5 + 1 \tag{24}\\ \frac{dM}{dt} - \frac{4}{t} \cdot M \amp = t^6 + t \tag{25} \end{align}
    If follow the process we took in the previous example, we would first multiply by the unknown function \(z\) that will help us un-do the product rule, then we would determine \(z\text{.}\) Instead, letâ€™s first determine \(z\text{,}\) and then weâ€™ll multiply by it. As in ((31)), we need to identify the function \(P\) that is attached to the independent variable by multiplication. In equation ((24)), we have \(P(t) = -\frac{4}{t}\text{,}\) (note the inclusion of the negative sign) so we know that the function \(z\) that will help us un-do the product rule is given by
    \begin{equation*} z = e^{\int P(t) dt} = e^{\int -\frac{4}{t} dt} = e^{-4\ln t}. \end{equation*}
    (Note, again, that we seek only one function \(z\text{,}\) so we omit the absolute value and the \(+C\) that we normally see when we integrate the function \(P(t) = -\frac{4}{t}.\))
    While our integrating factor \(z\) is technically correct, it can (and should) be simplified using rules of logarithms as follows.
    \begin{equation*} z = e^{-4\ln t} = e^{\ln (t^{-4})} = t^{-4}. \end{equation*}
    Weâ€™ll end up using lots of rules of logarithms throughout this chapter; review those rules (HERE) if you need a refresher.
    Now that we have identified the integrating factor \(z\text{,}\) we can multiply both sides of the DE (which we put in standard form in ((24))) by it.
    \begin{align*} t^{-4} \cdot \frac{dM}{dt} - t^{-4}\cdot \frac{4}{t} \cdot M \amp = t^{-4}(t^6 + t) \\ t^{-4} \cdot \frac{dM}{dt} - \cdot \frac{4}{t^5} \cdot M \amp = t^2 + t^{-3} \end{align*}
    If we have correctly determined the integrating factor, then we should be able to un-do the product rule on the left hand side. Notice that we can identify the functions \(f\) and \(g\) as follows.
    \begin{equation*} \underbrace{t^{-4}}_{\scriptsize f}\cdot \underbrace{\frac{dM}{dt}}_{\scriptsize g'} + \underbrace{- \frac{4}{t^5}}_{\scriptsize f'}\cdot \underbrace{M}_{\scriptsize g} = t^2 + t^{-3}. \end{equation*}
    It is critical that you verify that the function \(z\) does, in fact, create a situation where we see a function \(f\) and its derivative \(f'\) as shown above. In this case,
    \begin{align*} f'(t) \amp = \frac{d}{dt} \Big(f(t)\Big) \\ \amp = \frac{d}{dt}\Big( t^{-4} \Big) \\ \amp = -4t^{-5} \\ \amp = -\frac{4}{t^5}, \end{align*}
    so we can confirm that we have correctly identified the functions \(f\) and \(g\) to un-do the product rule. This, we can re-write the left hand side of the equation as follows.
    \begin{equation*} \frac{d}{dt}\Big( t^{-4}\cdot M \Big) = t^2 + t^{-3}. \end{equation*}
    Weâ€™ve made a big step forward, here! Remember that our goal is to find the unknown function \(M\text{,}\) so we are much closer since we \(M\) appears in only one place in this equation. In order to un-do the differentiation, we can integrate both sides of the equation.
    \begin{align*} \int \frac{d}{dt}\Big( t^{-4}\cdot M \Big)dt \amp = \int t^2 + t^{-3} dt \\ t^{-4}\cdot M \amp = \frac{1}{3}t^{3} - \frac{1}{2}t^{-2} + C \\ \frac{M}{t^4} \amp = \frac{1}{3}t^{3} - \frac{1}{2}t^{-2} + C \end{align*}
    Finally, we can solve for \(M\) by multiplying both sides of the equation by \(t^4.\)
    \begin{equation*} M = \frac{1}{3}t^{7} - \frac{1}{2}t^2 + Ct^4 \end{equation*}
    We have made a point of simplifying whenever possible, and you should look to do that as you work. You may be tempted to replace the \(Ct^3\) with a single constant; remember, however, that a constant times a function is not a constant, so we cannot simplify that last term any further.
    Hence, we have found the solution to the DE. We can always verify our solution by substituting back in to the original DE. This is optional, but the details are included here.
    \begin{align*} LHS \amp = -\frac{4}{t^2} \cdot M \\ \amp = -\frac{4}{t^2} \cdot \left( \frac{1}{3}t^{7} - \frac{1}{2}t^2 + Ct^4 \right) \\ \amp = -\frac{4}{3}t^5 + 2 - 4Ct^2 \end{align*}
    \begin{align*} RHS \amp = t^5 - \frac{1}{t} \cdot\frac{dM}{dt} + 1 \\ \amp = t^5 - \frac{1}{t} \cdot\frac{d}{dt}\left( \frac{1}{3}t^{7} - \frac{1}{2}t^2 + Ct^4 \right) + 1 \\ \amp = t^5 - \frac{1}{t} \cdot \left( \frac{7}{3}t^{6} - t + 4Ct^3 \right) + 1 \\ \amp = t^5 -\frac{7}{3}t^{5} +1 - 4Ct^2 + 1 \\ \amp = -\frac{4}{3}t^5 + 2 - 4Ct^2 \\ \amp = LHS \end{align*}

    Example 20.

    Solve the following initial value problem.
    \begin{equation*} A = \sec\theta A' - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta}, \hspace{1cm} A(0) = -\frac{1}{2} \end{equation*}
    Solution.
    As always, we begin by identifying the independent and dependent variables and classifying the DE.
    • independent variable: \(\theta\)
    • dependent variable: \(A\)
    • order: first order
    • linearity: linear
    • goal: solve for the unknown function \(A(\theta)\)
    Since this DE is first order and linear, we can use the integrating factor technique. We begin to put the DE in standard form for this technique by getting the dependent variable and its derivative on the left hand side of the equals sign.
    \begin{align*} A \amp = \sec\theta A' - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ -\sec\theta A' + A \amp = - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \end{align*}
    Next we want to make the coefficient on the \(A'\) term equal to 1. We can divide by \(-\sec \theta\text{,}\) which is equivalent to multiplying by \(-\cos \theta.\)
    \begin{equation} A' - \cos\theta A = \theta e^{\theta^2 + \sin\theta}\tag{26} \end{equation}
    Now we can identify the function \(P(\theta)\text{,}\) which consists of whatever is attached to the dependent variable \(A\) by multiplication. In this case, \(P(\theta) = - \cos \theta\text{.}\) Then we find \(z\) as follows.
    \begin{equation*} z = e^{\int P(\theta) d\theta} = e^{\int -\cos \theta d\theta} = e^{-\sin \theta} \end{equation*}
    (As before, we need only one integrating factor \(z\text{,}\) so we omit the \(+C\) we normally see when we evaluate an indefinite integral).
    Once we have identified our integrating factor, we can multiply both sides of the DE in its standard form (as in equation ((26))) by the function \(z\text{.}\)
    \begin{align*} e^{-\sin \theta}A' - \cos\theta e^{-\sin \theta}A \amp = \theta e^{\theta^2 + \sin\theta}e^{-\sin \theta} \\ \amp = \theta e^{\theta^2 + \sin\theta-\sin \theta} \\ \amp = \theta e^{\theta^2} \end{align*}
    If we have correctly determined the integrating factor \(z\text{,}\) we should be able to un-do the product rule on the left hand side. We can identify the functions \(f\) and \(g\) below:
    \begin{equation*} \underbrace{e^{-\sin \theta}}_{\scriptsize f}\cdot \underbrace{A'}_{\scriptsize g'} + \underbrace{- \cos\theta e^{-\sin \theta}}_{\scriptsize f'}\cdot \underbrace{A}_{\scriptsize g} = \theta e^{\theta^2}. \end{equation*}
    It is critical that you verify that the function \(z\) does, in fact, create a situation where we see a function \(f\) and its derivative \(f'\) as shown above. In this case,
    \begin{align*} f'(t) \amp = \frac{d}{dt} \Big(f(t)\Big) \\ \amp = \frac{d}{dt}\Big( e^{-\sin \theta} \Big) \\ \amp = e^{-\sin \theta} \cdot \frac{d}{dt}\Big( -\sin\theta \Big) \\ \amp = e^{-\sin \theta} \cdot \Big( -\cos\theta \Big) \\ \amp = -\cos\theta e^{-\sin \theta}, \end{align*}
    so we can confirm that we have correctly identified the functions \(f\) and \(g\) to un-do the product rule. This, we can re-write the left hand side of the equation as follows.
    \begin{equation*} \frac{d}{dt}\Big( e^{-\sin \theta}\cdot A \Big) = \theta e^{\theta^2}. \end{equation*}
    Our goal is to solve for \(A\text{.}\) Since it is currently inside the derivative, we will integrate both sides of the equation.
    \begin{align*} \int \frac{d}{dt}\Big( e^{-\sin \theta}\cdot A \Big) d\theta \amp = \int \theta e^{\theta^2} d\theta \\ e^{-\sin \theta}\cdot A \amp = \int \theta e^{\theta^2} d\theta \end{align*}
    In order to evaluate the integral on the right hand side, we can make the substitution \(u = \theta^2\text{,}\) which means that \(du = 2\theta d\theta\text{,}\) or equivalently \(\frac{1}{2}du = \theta d\theta\text{.}\)
    \begin{align*} e^{-\sin \theta}\cdot A \amp = \int \underbrace{e^{\theta^2}}_{\scriptsize e^u} \underbrace{\theta d\theta}_{\scriptsize \frac{1}{2}du} \\ \amp = \int \frac{1}{2}e^u du \\ \amp = \frac{1}{2}e^u + C \\ \amp = \frac{1}{2}e^{\theta^2} + C \end{align*}
    Now we can solve for \(A\text{.}\)
    \begin{align*} A \amp = \frac{1}{2}\frac{e^{\theta^2}}{e^{-\sin \theta}} + \frac{C}{e^{- \sin \theta}} \\ \amp = \frac{1}{2}e^{\theta^2 + \sin \theta} + Ce^{\sin \theta} \end{align*}
    Finally, we need to apply the initial condition we are provided so that we can determine the constant \(C\text{.}\)
    \begin{align*} -\frac{1}{2} \amp = A(0) \\ \amp = \frac{1}{2}e^{0^2 + \sin(0)} + Ce^{\sin(0)} \\ \amp = \frac{1}{2}e^0 + Ce^0 \\ \amp = \frac{1}{2} + C \\ -1 \amp = C \end{align*}
    We can then write the particular solution to the initial value problem.
    \begin{equation*} A = \frac{1}{2}e^{\theta^2 + \sin \theta} - e^{\sin \theta} \end{equation*}
    As usual, we can check our answer by substituting back into the original DE. While this step is not required, it is always a good idea to check and we include the details here.
    \begin{align*} LHS \amp = A \\ \amp = \frac{1}{2}e^{\theta^2 + \sin \theta} - e^{\sin \theta} \end{align*}
    \begin{align*} RHS \amp = \sec\theta A' - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ \amp = \sec\theta\cdot \frac{d}{d\theta}\left( \frac{1}{2}e^{\theta^2 + \sin \theta} - e^{\sin \theta} \right) - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ \amp = \sec\theta\cdot \left[ \frac{1}{2}e^{\theta^2 + \sin \theta}\cdot\frac{d}{d\theta}\left( \theta^2 + \sin \theta \right) \\ - e^{\sin\theta} \cdot \frac{d}{d\theta}\left( \sin \theta \right) \right] - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ \amp = \sec\theta\cdot \left[ \theta e^{\theta^2 + \sin \theta} + \frac{1}{2}\cos \theta e^{\theta^2 + \sin \theta} - \cos \theta e^{\sin\theta} \right] - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ \amp = \frac{1}{\cos\theta}\cdot \left[ \theta e^{\theta^2 + \sin \theta} + \frac{1}{2}\cos \theta e^{\theta^2 + \sin \theta} - \cos \theta e^{\sin\theta} \right] - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ \amp = \frac{\theta e^{\theta^2 + \sin \theta}}{\cos\theta} + \frac{1}{2}e^{\theta^2 + \sin \theta} - e^{\sin\theta} - \frac{\theta e^{\theta^2 + \sin\theta}}{\cos \theta} \\ \amp = \frac{1}{2}e^{\theta^2 + \sin \theta} - e^{\sin\theta} \\ \amp = LHS \end{align*}

    Example 21.

    Consider the following DE.
    \begin{equation*} x^2y' = 5x^3 + 2x^3y \end{equation*}
    Solution.
    First we note that this equation is first order and linear.
    Our first step would be to move closer to seeing the form shown in (31). As such, we will put the terms containing the derivative, \(y'\text{,}\) and the function, \(y\text{,}\) on the same of the equation.
    \begin{equation*} x^2 y' - 2x^3 y = 5x^3 \end{equation*}
    In an ideal world, we would immediately be able to un-do the product rule on the left hand side of the equation. Unfortunately, this doesnâ€™t work right now, so we need to do a little work to force it to work. %Obviously this sentence is awful. We would still like to push this into the form shown in ((31)), so we can first divide the entire equation by the coefficient on \(y'\text{,}\)
    \begin{equation*} y' - 2x y = 5x, \end{equation*}
    and then multiply by the function \(z\) that will help us un-do the product rule.
    \begin{equation} z\cdot y' - 2x z\cdot y = z\cdot 5x \tag{27} \end{equation}
    In Example 6, we showed that the function \(z\) we need is given by \(z = e^{\int P(x) dx}\text{.}\) We need only identify the function \(P(x)\) as in ((31)). For this DE, we have \(P(x) = -2x\text{.}\) Take special note of the inclusion of the negative sign. Hence, we have
    \begin{equation*} z = e^{\int P(x) dx} = e^{\int -2x dx} = e^{-x^2}. \end{equation*}
    Note that we have assumed that the constant of integration is zero because we know we need just one function \(z\text{,}\) and we choose the simplest one. Now we can replace the replace the \(z\)â€™s in Equation ((27)) with \(z = e^{-x^2}\) to yield
    \begin{equation*} e^{-x^2}\cdot y' - 2x e^{-x^2}\cdot y = 5xe^{-x^2}. \end{equation*}
    If weâ€™ve done everything correctly so far, we should be able to undo the product rule on the left hand side. We can identify functions \(f\) and \(g\) as follows:
    \begin{equation*} \underbrace{e^{-x^2}}_{\scriptsize f}\cdot \underbrace{y'}_{\scriptsize g'} - \underbrace{2x e^{-x^2}}_{\scriptsize f'}\cdot \underbrace{y}_{\scriptsize g} = 5xe^{-x^2}. \end{equation*}
    We need to verify at this step that if we take the derivative of \(f(x) = e^{-x^2}\text{,}\) we do actually get the expression weâ€™ve labeled as \(f'\text{.}\) In this case, we do get \(f'(x) = -2xe^{-x^2}\text{,}\) as expected, so we have correctly determined the function \(z\) that will help us un-do the product rule. This function, that plays a central role in this solution technique, is called the integrating factor.
    We have identified how we can un-do the product rule so now we need to do just that.
    \begin{equation*} \frac{d}{dx}\Big( \underbrace{e^{-x^2}}_{\scriptsize f}\cdot \underbrace{y}_{\scriptsize g} \Big) = 5xe^{-x^2}. \end{equation*}
    We have just made a huge step forward in solving this DE! We have made it so that the unknown function \(y\) now appears in the equation only once, so we can now work to solve for \(y\text{.}\) As it stands, the function \(y\) is inside a derivative, so we can integrate both sides (with respect to the independent variable).
    \begin{equation*} \int \frac{d}{dx}\Big( e^{-x^2} \cdot y \Big)dx = \int 5xe^{-x^2}dx \end{equation*}
    The integration and differentiation will un-do each other on the left hand side. On the right hand side, we need a substitution, \(u = -x^2\text{,}\) in order to achieve the integration. This means \(du = -2x dx\text{,}\) or equivalently \(-\frac{1}{2}du =x dx.\)
    \begin{align*} e^{-x^2} \cdot y \amp = \int 5xe^{-x^2}dx \\ \amp = 5\int e^{-x^2}\cdot x dx \\ \amp = 5 \int e^{u}\cdot -\frac{1}{2} du \\ \amp = -\frac{5}{2}\int e^u du \\ \amp = -\frac{5}{2}\int e^u + C \\ \amp = -\frac{5}{2}e^{-x^2} + C \end{align*}
    We are almost done; we need only divide both sides of the equation by \(e^{-x^2}\) to solve for \(y\text{.}\)
    \begin{equation*} y = -\frac{5}{2} + \frac{C}{e^{-x^2}} \end{equation*}
    While the equation above is a valid solution, we may prefer to rewrite the second term using rules of exponents to yield
    \begin{equation*} y = -\frac{5}{2} + Ce^{x^2}. \end{equation*}
    We have now found the general solution to the DE. As always, we can verify that we have found the correct solution by substituting back into the original DE. %Put this in a collapsible?
    \begin{align*} \mbox{LHS} \amp = x^2y' \\ \amp = x^2 \cdot \frac{d}{dx}\left( -\frac{5}{2} + Ce^{x^2} \right) \\ \amp = x^2 \cdot \left( 0 + 2Cxe^{x^2} \right) \\ \amp = 2C x^3 e^{x^2} \end{align*}
    \begin{align*} \mbox{RHS} \amp = 5x^3 + 2x^3y \\ \amp = 5x^3 + 2x^3\left( -\frac{5}{2} + Ce^{x^2} \right) \\ \amp = 5x^3 - 5x^3+ 2C x^3 e^{x^2} \\ \amp = 2C x^3 e^{x^2} \end{align*}
    Since we get the same result when we substitute our solution in on the right and left hand sides of the DE, we confirm that we did find the solution.
