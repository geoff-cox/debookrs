    Additional Examples.

    Example 2.21.

    Determine if the following DE is separable.
    \begin{equation*} \frac{dz}{dt} = \cos^2 z \end{equation*}
    Solution.
    This differential equation is first order. Since the derivative is already isolated, we need to determine if we can express the right-hand side as a product of functions where one function does not contain the dependent variable \(z\text{,}\) and the other does not contain the independent variable \(t\text{.}\) Consider the equivalent equation:
    \begin{equation*} \frac{dz}{dt} = \big( 1 \big) \cdot \big( \cos^2 z \big) \end{equation*}
    It may seem trivial, but this satisfies the definition of separable. The first factor is a function that does not contain \(z\text{,}\) and the second factor is a function that does not contain \(t\text{.}\) The first factor is a constant function, which is perfectly acceptable. Therefore, this differential equation is separable.

    Reading Questions Additional Practice

    1. True/False.

      The differential equation, \(\displaystyle \frac{dy}{dx} = y^2\text{,}\) is separable.

    • True.

    • Even though no \(x\) terms are present, we still can group the terms like so:
      \begin{equation*} \frac{dy}{dx} = y^2 = (1)(y^2) = \underset{f(x)}{\left(\fbox{\(1\)}\right)}\underset{g(y)}{\left(\fbox{\(y^2\)}\right)}. \end{equation*}

    • False.

    • Even though no \(x\) terms are present, we still can group the terms like so:
      \begin{equation*} \frac{dy}{dx} = y^2 = (1)(y^2) = \underset{f(x)}{\left(\fbox{\(1\)}\right)}\underset{g(y)}{\left(\fbox{\(y^2\)}\right)}. \end{equation*}

    2. True/False.

      The differential equation, \(\displaystyle \frac{dy}{dx} = x - 1\text{,}\) is separable.

    • True.

    • As in the previous problem, we can group the terms like so:
      \begin{equation*} \frac{dy}{dx} = x - 1 = (x - 1)(1) = \underset{f(x)}{\left(\fbox{\(x - 1\)}\right)}\underset{g(y)}{\left(\fbox{\(1\)}\right)}. \end{equation*}

    • False.

    • As in the previous problem, we can group the terms like so:
      \begin{equation*} \frac{dy}{dx} = x - 1 = (x - 1)(1) = \underset{f(x)}{\left(\fbox{\(x - 1\)}\right)}\underset{g(y)}{\left(\fbox{\(1\)}\right)}. \end{equation*}

    3. True/False.

      The differential equation, \(\displaystyle \frac{dy}{dx} = 15\text{,}\) is separable.

    • True.

    • Despite not having any \(x\) nor \(y\) terms, it is still technically separable since
      \begin{equation*} \frac{dy}{dx} = 15 = (15)(1) = \underset{f(x)}{\left(\fbox{\(15\)}\right)}\underset{g(y)}{\left(\fbox{\(1\)}\right)}. \end{equation*}

    • False.

    • Despite not having any \(x\) nor \(y\) terms, it is still technically separable since
      \begin{equation*} \frac{dy}{dx} = 15 = (15)(1) = \underset{f(x)}{\left(\fbox{\(15\)}\right)}\underset{g(y)}{\left(\fbox{\(1\)}\right)}. \end{equation*}

    4. True/False.

      The differential equation \(x+\frac{dy}{dx}=y\) can be solved using separation.

    • True.

    • The differential equation is first order and can be written as \(\frac{dy}{dx}=y-x\text{,}\) which is not separable.

    • False.

    • The differential equation is first order and can be written as \(\frac{dy}{dx}=y-x\text{,}\) which is not separable.
    Hint.
    Check to see that the differential equation is first order and separable.

    5. True/False.

      The differential equation \(\frac{dz^2}{d^2t}=\cos^2(z)\) can be solved using separation.

    • True.

    • The differential equation is separable, but not first order.

    • False.

    • The differential equation is separable, but not first order.
    Hint.
    Check to see that the differential equation is first order and separable.

    6. Multiple Choice.

      Write the differential equation \(x\frac{dy}{dx}+10x^2y=6x^2\) in separable form.
    • \(\frac{dy}{dx}=6x-10xy\)
    • This is not written in separable form.
    • \(\frac{dy}{dx}=x(6-10y)\)
    • This is written in separable form.
    • \(\frac{dy}{dx}=x(6+10y)\)
    • This is written in separable form, but is not equivalent to the original differential equation.
    Hint.
    Try writing \(\frac{dy}{dx}\) as the product of the independent and dependent variables.
