    Section 1.3.2 Linear Terms

    The most informative label you can place on a differential equation is whether it is linear or not. When you first learn about linearity in mathematics, it often involves functions whose graphs are straight lines. However, when we talk about the linearity of a differential equation, the concept is different and can be tricky to grasp for a beginner.
    To prepare for the definition of linearity of a DE, we need the following:
    • Recall the difference between a dependent and independent variable.
       7 
      For example, \(y\) is the dependent and \(t\) is the independent variable in the equation
      \begin{equation*} t y'' + \sin(t)y = 0\text{.} \end{equation*}
    • “Terms” are parts in the equation that are separated by \(+\text{,}\) \(-\text{,}\) or \(=\) signs.
    • A term is linear if it looks like
    \begin{equation*} \left(\text{function of } t\right), \ \ \left(\text{function of } t\right) {\color{blue} y }, \ \ \left(\text{function of } t\right) {\color{blue} y' }, \ \ \left(\text{function of } t\right) {\color{blue} y'' }, \ \ ... \end{equation*}
    where \({\color{blue} y }\) is the dependent variable and \(t\) is the independent variable. Here are a few examples of linear terms:
    \begin{equation*} 3, \quad 3t, \quad 2t^2{\color{blue} y }, \quad \sin(t){\color{blue} y' }, \quad \left(t^2 - \frac{1}{t}\right){\color{blue} y''' }, \quad e^{t}{\color{blue} y^{(7)} } \text{.} \end{equation*}
    Note, the (function of \(t\)) in each term does not need to be linear, however there can be at most one dependent variable, \(\big( {\color{blue} y}, {\color{blue} y'}, {\color{blue} y'''}, {\color{blue} y^{(7)}} \big)\) in each term.
    You have seen examples of a linear terms, but what about a nonlinear terms? The following list gives you some rules of thumb for what nonlinear terms tend to look like.

    Identifying Nonlinear Terms.

    Assume \(y\) is the dependent variable for some differential equation.
     8 
    Remember, the linearity of a term only depends on the dependent variable.
    • raised to a power other than 1.
    • inside another function.
    • multiplied or divided by one another.

    Example 1.32.

    Determine the linearity of each term in the following differential equations.
    \begin{equation*} \frac{1}{t}y'' + y^2 + \ln(y') = e^t \end{equation*}
    Solution.
    \begin{align*} \underset{\text{linear}}{\underline{\frac{1}{t}{\color{blue} y'' }}} + \underset{\text{nonlinear}}{\underline{{\color{blue} y}^2}} + \underset{\text{nonlinear}}{\underline{\ln({\color{blue} y'})}} =\ \amp \underset{\text{linear}}{\underline{e^t}} \end{align*}
    \begin{equation*} P^{(6)} + \frac{m P'}{P} = (m - 1)^2 \end{equation*}
    Solution.
    \begin{align*} \underset{\text{linear}}{\underline{P^{(6)}}} + \underset{\text{nonlinear}}{\underline{\frac{m {\color{blue} P'}}{{\color{blue} P}}}} =\ \amp \underset{\text{linear}}{\underline{(m - 1)^2}} \end{align*}

    Reading Questions Check your Understanding

    Assume that \(y\) and \(t\) are the dependent and independent variables, respectively.

    True-False.

    1. \(3t\) is a linear term.
    • True
    • Correct! The term \(3t\) is linear because it is a function of the independent variable only.
    • False
    • Incorrect, review the examples of linear terms in the section on Linear Terms.
    2. \(y \sin(t)\) is a linear term.
    • True
    • Correct!
    • False
    • Incorrect! The term \(\sin(t)y\) is linear.
    3. \(t\sin(y')\) is a linear term.
    • True
    • Incorrect, review Identifying Nonlinear Terms.
    • False
    • Correct! A term that includes the derivative of the dependent variable inside another function is nonlinear.
    4. \(e^{ty}\) is a linear term.
    • True
    • Incorrect, even though there are no parentheses, \(y\) is inside the exponential function \(e^{(ty)}\text{.}\)
    • False
    • Correct! A term that includes the dependent variable inside another function is nonlinear.
    5. A linear term can contain the dependent variable multiplied by the independent variable.
    • True
    • Correct! \(t y\) is a linear term.
    • False
    • Incorrect, if you write down this term you would get \(t y\text{.}\)

    Multiple-Choice.

    6. Which of the following terms is linear?
    • \(y^2\)
    • Incorrect. \(y^2\) is nonlinear because the dependent variable is raised to a power other than one.
    • \(\sin(t)y'\)
    • Correct! \(\sin(t)y'\) is linear because it is a function of the independent variable multiplied by the first derivative of the dependent variable.
    • \(\ln(y)\)
    • Incorrect. \(\ln(y)\) is nonlinear because it includes the dependent variable inside another function.
    • \(y \cdot y'\)
    • Incorrect. \(y \cdot y'\) is nonlinear because it involves the product of the dependent variable and its derivative.
    7. Identify the linear term from the following options.
    • \(\frac{1}{t}y''\)
    • Correct! \(\frac{1}{t}y''\) is linear because it is a function of the independent variable multiplied by the second derivative of the dependent variable.
    • \(y^3\)
    • Incorrect. \(y^3\) is nonlinear because the dependent variable is raised to a power other than one.
    • \(e^t y^2\)
    • Incorrect. \(e^t y^2\) is nonlinear because the dependent variable is squared.
    • \(y \cdot \cos(y)\)
    • Incorrect. \(y \cdot \cos(y)\) is nonlinear because it involves the product of the dependent variable and a function of the dependent variable.
    8. Which term is NOT considered linear?
    • \(\ln(y')\)
    • Correct! \(\ln(y')\) is nonlinear because it includes the logarithm of the derivative of the dependent variable.
    • \(3t\)
    • Incorrect. \(3t\) is linear because it is a function of the independent variable only.
    • \(2t^2 y\)
    • Incorrect. \(2t^2 y\) is linear because it is a function of the independent variable multiplied by the dependent variable.
    • \(e^t y^{(7)}\)
    • Incorrect. \(e^t y^{(7)}\) is linear because it is a function of the independent variable multiplied by the seventh derivative of the dependent variable.
    9. Which of the following is a characteristic of a nonlinear term?
    • It involves the dependent variable raised to the first power.
    • Incorrect. This is a characteristic of a linear term.
    • It involves the dependent variable only as a constant.
    • Incorrect. This is a characteristic of a linear term.
    • It includes the dependent variable inside another function.
    • Correct! A nonlinear term includes the dependent variable inside another function.
    • It involves the independent variable only.
    • Incorrect. This is a characteristic of a linear term.
    10. Which term is an example of a nonlinear term?
    • \(3\)
    • Incorrect. \(3\) is linear because it is a constant.
    • \(3t\)
    • Incorrect. \(3t\) is linear because it is a function of the independent variable only.
    • \(y^2\)
    • Correct! \(y^2\) is nonlinear because the dependent variable is squared.
    • \(2t^2 y\)
    • Incorrect. \(2t^2 y\) is linear because it is a function of the independent variable multiplied by the dependent variable.
    11. Which of the following terms is linear?
    • \(t y''\)
    • Correct! \(t y''\) is linear because it is a function of the independent variable multiplied by the second derivative of the dependent variable.
    • \(y y'\)
    • Incorrect. \(y y'\) is nonlinear because it involves the product of the dependent variable and its derivative.
    • \(e^y\)
    • Incorrect. \(e^y\) is nonlinear because it involves the exponential function of the dependent variable.
    • \(\ln(y)\)
    • Incorrect. \(\ln(y)\) is nonlinear because it includes the dependent variable inside another function.
