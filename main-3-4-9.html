    Section 1.2.5 Additional Examples

    This is just a place to store extra examples which may or may not be used.

    Example 19. Solution Verification with Two Constants.

    Verify that \(y = \frac{1}{2}x^2 + c_1 x + c_2\) is a solution to \(\quad y'' = 1\text{.}\)
    Solution.
    Move Terms: \(\quad y'' - 1 = 0\)
    Compute Derivatives. Find \(y''\) since it appears in the DE.
    \begin{align*} y \amp= \frac{1}{2}x^2 + c_1 x + c_2 \\ y' \amp= x + c_1 \\ y'' \amp= 1 \end{align*}
    Plug-In & Simplify. Plug \(y''\) into the DE and simplify.
    \begin{align*} y'' - 1 \amp= 0 \\ 1 - 1 \amp= 0 \\ 0 \amp= 0 \end{align*}
    Therefore, \(y = \frac{1}{2}x^2 + c_1 x + c_2\) is a solution to \(y'' = 1\text{.}\)
     6 
    In fact, it is the general solution, but we don’t show that here.

    Example 20.

    Solve
    \begin{equation*} y'' - y' - 12y = 0, \quad y(0) = 4, \quad y'(0) = -2. \end{equation*}
    Solution.
    The general solution to this DE was already found to be
    \begin{equation*} y = c_1e^{-3t} + c_2e^{4t}\text{.} \end{equation*}
    Notice that two initial conditions are provided in this example. This extra piece of information is needed since our general solution contains two constants.
    Since one initial condition involves the derivative, we will need to compute this before applying this condition.
    \begin{equation*} y' = -3c_1e^{-3t} + 4c_2e^{4t}. \end{equation*}
    Now let’s apply the initial conditions:
    \begin{align*} y(0) \amp= 4 \amp\Rightarrow \hspace{0.2cm} 4 \amp= c_1e^{-3\cdot 0} + c_2e^{4\cdot 0}\\ \amp \amp 4 \amp= c_1 + c_2 \\ \amp \amp \amp \\ y'(0) \amp= -5 \amp\Rightarrow \hspace{0.2cm} -5 \amp=-3c_1e^{-3\cdot} + 4c_2e^{4\cdot} \\ \amp \amp -5 \amp= -3c_1 + 4c_2 \end{align*}
    We now have two equations and two unknowns,
    \begin{align*} c_1 + c_2 \amp= 4 \\ -3c_1 + 4c_2 \amp= -5 \end{align*}
    There are multiple ways to solve this system of equations, and the reader is encouraged to verify that \(c_1 = 3\) and \(c_2 = 1\text{.}\) Here is one
    Solution.
    We will solve for \(c_2\) in the first equation and then substitute into the second.
    \begin{align*} 4 \amp= c_1 + c_2 \\ \Rightarrow c_2 \amp= 4 - c_1 \\ \amp \\ -5 \amp= -3c_1 + 4c_2 \\ \Rightarrow -5 \amp= -3c_1 + 4(4 - c_1) \\ -5 \amp= -3c_1 + 16 - 4c_1 \\ -21 \amp= -7c_1 \\ 3 \amp= c_1 \\ \amp \\ c_2 \amp=4 - c_1 \\ \amp= 4 - 3 \\ \amp= 1 \end{align*}
    Thus, the solution we that satisfies the DE and both of the provided initial conditions is
    \begin{equation*} y = 3e^{-3t} + e^{4t}. \end{equation*}

    Note 21. Don’t assume a function that satisfies a DE is a general solution!

    A function that contains constants and satisfies a differential equation does not mean it is the general solution since it could be missing a term. For example, you could easily show that \(y = \frac{1}{2}x^2 + c_1 x\) is also a solution to
    \begin{equation*} \quad y'' = 1\text{,} \end{equation*}
    but you already know from Example 19 that \(y = \frac{1}{2}x^2 + c_1 x + c_2\) is a solution, which is more general than \(y = \frac{1}{2}x^2 + c_1 x\text{.}\)

    Example 22.

    Verify that \(\DS y = Ce^{x^2} + 3\text{,}\) where \(C\) is any constant, is a solution to the DE
    \begin{equation*} \DS \frac{dy}{dx} = 2xy - 6x. \end{equation*}
    Solution.
    We’ll approach this the same way we approached the algebraic equation in the example above: we will to substitute the function \(\DS y = Ce^{x^2} + 3\) in on the left and right hand sides of the equation
    \begin{align*} \overset{LHS\vphantom{\Big|}}{\frac{dy}{dx}} \amp=\frac{d}{dx}\left( Ce^{x^2} + 3 \right)\\ \amp= C\frac{d}{dx}\left( e^{x^2} \right) + \frac{d}{dx}\left( 3 \right)\\ \amp= C \cdot e^{x^2} \cdot \frac{d}{dx}\left( x^2 \right) + 0\\ \amp= C \cdot e^{x^2} \cdot 2x\\ \amp= 2Cxe^{x^2} \end{align*}
    \begin{align*} \overset{RHS\vphantom{\Big|}}{2xy - 6x} \amp= 2x \left( Ce^{x^2} + 3 \right) - 6x\\ \amp= 2Cxe^{x^2} + 6x - 6x\\ \amp= 2Cxe^{x^2} \end{align*}
    Since both the \(LHS\) and \(RHS\) turn out to be \(2Cxe^{x^2}\text{,}\) we have shown that \(\DS y = Ce^{x^2} + 3\) satisfies and is, thus, a solution to \(\DS \frac{dy}{dx} = 2xy - 6x.\)

    Note 23. Example Notes.

    Since we used a generic constant \(C\text{,}\) we have shown that \(\DS y = Ce^{x^2} + 3\) is a solution for any value we like for the constant \(C\text{.}\) In particular,
    • for \(C=-41, \quad y = -41e^{x^2} + 3 \quad\) is a solution.
    • for \(C=199, \quad y = 199e^{x^2} + 3 \quad\) is a solution.
    • for \(C=0, \quad y = 3 \quad\) is a solution.
    Since there is no limit to what \(C\) we can choose, this DE must have an infinite number of solutions. In the next section, we will describe this idea further.

    Example 24.

    Determine if \(y = c_1e^{-3t} + c_2e^{4t}\) is a solution to the DE \(y'' - y' - 12y = 0\) where \(c_1\) and \(c_2\) are constants.
    Solution.
    Find \(y, y', \text{ and } y''\text{.}\)
    \begin{align*} y \amp= c_1e^{-3t} + c_2e^{4t} \hspace{-0.4cm} \amp\Rightarrow\hspace{0.1cm} y' \amp= -3c_1e^{-3t} + 4c_2e^{4t} \hspace{-0.4cm} \amp\Rightarrow\hspace{0.1cm} y'' \amp= 9c_1e^{-3t} + 16c_2e^{4t} \end{align*}
    and substitute
    \begin{align*} LHS \amp= y'' - y' - 12y \\ \amp= \left(9c_1e^{-3t} + 16c_2e^{4t}\right) - \left( -3c_1e^{-3t} + 4c_2e^{4t} \right) - 12\left( c_1e^{-3t} + c_2e^{4t} \right) \\ \amp= 9c_1e^{-3t} + 16c_2e^{4t} + 3c_1e^{-3t} - 4c_2e^{4t} - 12c_1e^{-3t} - 12c_2e^{4t} \\ \amp= \left( 9c_1e^{-3t} + 3c_1e^{-3t} - 12c_1e^{-3t} \right) + \left( 16c_2e^{4t} - 4c_2e^{4t} - 12c_2e^{4t} \right) \\ \amp = 0 \\ \amp= RHS \end{align*}
    Which shows that \(y = c_1e^{-3t} + c_2e^{4t}\) is a solution to \(y'' - y' - 12y = 0.\)

    Note 25. Example Notes.

    In this example, we showed that the function \(y = c_1e^{-3t} + c_2e^{4t}\) was a solution \(y'' - y' - 12y = 0\text{.}\)
    Notice that we were able to verify the solution, even though the constants, \(c_1\) and \(c_2\text{,}\) were not known. This suggests that you can substitute any values of \(c_1\) and \(c_2\) into the solution and it will still be a solution to the DE. Try it!
    Now, since there are infinitely many choices for \(c_1\) and \(c_2\text{,}\) then, logically, there must be infinitely many solutions to the DE, \(y'' - y' - 12y = 0\text{.}\)

    Example 26.

    Verify that \(y = Ce^{-5x^2} + \frac{3}{5}\) is a solution to the DE
    \begin{equation*} \frac{dy}{dx} = \big( 2x \big) \big( 3 - 5y \big) \end{equation*}
    Solution.
    Working out the left and right hand sides of this DE, we have:
    \begin{align*} LHS: \\ \frac{dy}{dx} \amp=\frac{d}{dx}\left( Ce^{-5x^2} + \frac{3}{5} \right)\\ \amp= Ce^{-5x^2} \cdot \frac{d}{dx}(-5x^2) + 0\\ \amp= Ce^{-5x^2} \cdot -10x\\ \amp= -10Cxe^{-5x^2} \end{align*}
    \begin{align*} RHS: \\ \big(2x\big)\amp\big(3 - 5y\big)\\ \amp= 2x \left[3 - 5\left(Ce^{-5x^2} + \frac{3}{5}\right) \right]\\ \amp= 2x \left[ 3 - 5Ce^{-5x^2} - 3 \right]\\ \amp= 2x \left[-5Ce^{-5x^2}\right]\\ \amp= -10Cxe^{-5x^2} \end{align*}
    Since \(LHS = RHS\text{,}\) we have verified that \(y = Ce^{-5x^2} + \frac{3}{5}\) is a solution to
    \begin{equation*} \frac{dy}{dx} = \big( 2x \big) \big( 3 - 5y \big) \end{equation*}
