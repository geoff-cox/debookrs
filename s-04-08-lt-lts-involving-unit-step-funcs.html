    Section 12.2 Laplace Transforms Involving Unit Step Functions

    Swap sections 4.8 and 4.9
    We are now able to rewrite piecewise defined functions in terms of unit step functions. We now need to take Laplace transforms of piecewise defined functions, and then we will be ready to solve DEs involving piecewise defined functions.

    Example 39.

    Consider the function
    \begin{equation*} g(t) = \left\{ \begin{array}{ll} 0, \amp t \lt 2, \\ 6, \amp 2 \le t \lt 5,\\ -4, \amp t \ge 5 \end{array} \right. \end{equation*}
    1. Write \(g(t)\) in terms of unit step functions.
      \begin{equation*} g(t) = 6\ U(t-2) - 10\ U(t-5). \end{equation*}
    2. Find the Laplace transform of \(g(t)\) using the definition of the Laplace transform.
    3. Based on your answers to parts (a) and (b), hypothesize about a rule for taking the Laplace transform of a unit step function.
    Solution.
    1. The function changes behavior at \(t = 2\) and \(t = 5.\) We can then assemble the function as follows, thinking about turning the pieces â€™onâ€™ and â€™offâ€™ as described in the previous section.
      \begin{align*} g(t) =\amp\ 0 + (-0 + 6)\cdot U(t-2) + (-6 + -4) \cdot U(t-5)\\ =\amp\ 6\ U(t-2) - 10\ U(t-5) \end{align*}
    2. We will use the definition of the Laplace transform and properties of integration, as follows.
      \begin{align*} G(s) =\amp\ \lap{ g(t) }\\ =\amp\ \int_0^{\infty} e^{-st}g(t)dt\\ =\amp\ \int_0^2 e^{-st}g(t)dt + \int_2^5 e^{-st}g(t)dt + \int_5^{\infty} e^{-st}g(t)dt\\ =\amp\ \int_0^2 e^{-st}\cdot 0dt + \int_2^5 e^{-st}\cdot 6dt \\ =\amp\ \int_0^2 e^{-st}\cdot 0dt + \int_2^5 e^{-st}\cdot 6dt \\ =\amp\ 0 + 6\int_2^5 e^{-st}dt \\ =\amp\ -\frac{6}{s}e^{-st}\Big|_2^5 \\ =\amp\ -\frac{6}{s}\left[e^{-s\cdot 5} - e^{-s\cdot 2}\right]\\ =\amp\ -\frac{6e^{-5s}}{s} + \frac{6e^{-2s}}{s}\\ =\amp\ -\frac{6e^{-5s}}{s} + \frac{6e^{-2s}}{s}\\ =\amp\ -\frac{6e^{-5s}}{s} + \frac{6e^{-2s}}{s}\\ =\amp\ -\frac{6e^{-5s}}{s} + \frac{6e^{-2s}}{s} - \frac{4e^{-5s}}{s},\\ =\amp\ \frac{6e^{-2s}}{s} - \frac{10e^{-5s}}{s}, s \gt 0 \end{align*}
    3. If we put together the results from the last two questions, we have
      \begin{equation*} \lap{ 6\ U(t-2) - 10\ U(t-5) } = \frac{6e^{-2s}}{s} - \frac{10e^{-5s}}{s}. \end{equation*}
      Based on this, we can surmise the following property for Laplace transforms:
      \begin{equation*} \lap{ U(t-a) } = \frac{e^{-as}}{s}. \end{equation*}
    We will add this property and a couple more to our table of Laplace transforms. These properties (\(L14\) through \(L16\)) will be added to the bottom of the table we used previously.
    Table of Laplace Transforms
    \begin{equation*} \begin{array}{|c|c|c|} \hline \amp f(t) \amp \lap{ f(t) } \amp \text{Condition/Notes}\\ \hline \hline \text{L1} \amp 1 \amp \ds \frac{1}{s} \amp s>0\\ \hline \text{L2} \amp e^{at} \amp \ds \frac{1}{s-a} \amp s>a\\ \hline \text{L3} \amp t^n, \hspace{0.25cm}n = 1, 2, \ldots \amp \ds \frac{n!}{s^{n+1}} \amp s>0\\ \hline \text{L4} \amp \sin (bt) \amp \ds \frac{b}{s^2 + b^2} \amp s>0\\ \hline \text{L5} \amp \cos(bt) \amp \ds \frac{s}{s^2 + b^2} \amp s>0\\ \hline \text{L6} \amp t^n e^{at}, \hspace{0.25cm}n = 1, 2, \ldots \amp \ds \frac{n!}{(s-a)^{n+1}} \amp s>a\\ \hline \text{L7} \amp e^{at} \sin(bt) \amp \ds \frac{b}{(s-a)^2 + b^2} \amp s>a\\ \hline \text{L8} \amp e^{at} \cos(bt) \amp \ds \frac{s-a}{(s-a)^2 + b^2} \amp s>a\\ \hline \text{L9} \amp \alpha f(t) + \beta g(t) \amp \ds \alpha \lap{ f(t) } + \beta \lap{ g(t) } \amp \alpha, \beta \text{ constant}\\ \hline \text{L10} \amp f'(t) \amp s\lap{ f(t) } - f(0) \amp \\ \hline \text{L11} \amp f''(t) \amp s\lap{ f'(t) } - f'(0) \amp \\ \hline \text{L12} \amp f^{(n)}(t) \amp s\lap{ f^{(n-1)}(t) } - f^{(n-1)}(0) \amp \\ \hline \text{L13} \amp t^n f(t) \amp \ds (-1)^n\frac{d^n}{ds^n}\big( \lap{ f(t) } \big) \amp \\ \hline \text{L14.} \amp U(t-a) \amp \ds \frac{e^{-as}}{s}\amp \\ \hline \text{L15.} \amp \ds f(t) U(t-a) \amp \ds e^{-as} \lap{ f(t+a) }\amp \\ \hline \text{L16.} \amp \ds f(t-a)U(t-a) \amp \ds e^{-as}F(s)\amp \\ \hline \hline \end{array} \end{equation*}
    Letâ€™s practice using the new properties that have been added to the table for Laplace transforms involving unit step functions.

    Example 40.

    Find the Laplace transform of the function
    \begin{equation*} q(t) = 3\ U(t-7) + t^2 U(t-4) \end{equation*}
    Solution.
    We will need to use linearity and then properties \(L14\) and \(L15\). We begin as follows.
    \begin{align*} Q(s) =\amp\ \lap{ q(t) }\\ =\amp\ \lap{ 3\ U(t-7) + t^2 U(t-4) }\\ =\amp\ 3\lap{ U(t-7) } + \lap{ t^2 U(t-4) }\\ =\amp\ 3\cdot \frac{e^{-7s}}{s} + \lap{ t^2 U(t-4) } \end{align*}
    (Point out common mistake here?? In the margin??)
    For the second term, we need to use \(L15\), with \(f(t) = t^2\) and \(a = 4\text{.}\) Then we have
    \begin{align*} Q(s) =\amp\ 3\cdot \frac{e^{-7s}}{s} + \lap{ t^2 U(t-4) },\\ =\amp\ 3\cdot \frac{e^{-7s}}{s} + e^{-4s} \lap{ f(t+4) }. \end{align*}
    We arenâ€™t done yet, because we still need to evaluate \(\lap{ f(t+4) }.\) First, since \(f(t) = t^2,\) then we know that
    \begin{align*} f(t+4) =\amp\ (t+4)^2,\\ =\amp\ t^2 + 8t + 16. \end{align*}
    We need to take the Laplace transform of this function; we will use linearity and properties \(L1\) and \(L3\). We continue below.
    \begin{align*} Q(s) =\amp\ 3\cdot \frac{e^{-7s}}{s} + e^{-4s} \lap{ f(t+4) }\\ =\amp\ 3\cdot \frac{e^{-7s}}{s} + e^{-4s} \lap{ t^2 + 8t + 16 } \\ =\amp\ 3\cdot \frac{e^{-7s}}{s} + e^{-4s} \Big[ \lap{ t^2 } + 8\lap{ t } + 16\lap{ 1 } \Big]\\ =\amp\ 3\cdot \frac{e^{-7s}}{s} + e^{-4s} \left[ \frac{2}{s^3} + \frac{8}{s^2} + \frac{16}{s} \right] \end{align*}

    Example 41.

    Evaluate \(\lap{ e^{5t} U(t-6)}.\)
    Solution.
    Here we will use \(L15\), with \(f(t) = e^{5t}\) and \(a = 6.\)
    \begin{align*} \lap{ e^{5t} U(t-6) } =\amp\ e^{-6s}\lap{ f(t+6) }\\ =\amp\ e^{-6s}\lap{ e^{5(t+6)} }\\ =\amp\ e^{-6s}\lap{ e^{5t+30} }\\ =\amp\ e^{-6s}\lap{ e^{5t}\cdot e^{30} }\\ =\amp\ e^{-6s}e^{30}\lap{ e^{5t} }\\ =\amp\ e^{-6s}e^{30}\cdot \frac{1}{s-5}\\ =\amp\ \frac{e^{-6s + 30}}{s-5} \end{align*}
    So far we have been taking Laplace transforms of functions involving unit step functions, and we have used properties \(L14\) and \(L15\). We will use \(L16\) for inverse Laplace transforms involving unit step functions.

    Example 42.

    Find the inverse Laplace transform of
    \begin{equation*} X(s) = \frac{e^{-2s}}{s^3}. \end{equation*}
    Solution.
    We have a function of \(s\) that contains an exponential, and we want to find the inverse Laplace transform. The presence of the exponential is a cue that we will be dealing with unit step functions. In particular, we will use \(L16\). First it may be easier for us to rewrite \(X(s)\) as
    \begin{align*} X(s) =\amp\ e^{-2s}\cdot \frac{1}{s^3}\\ =\amp\ e^{-2s}\cdot F(s) \end{align*}
    where \(\ds F(s) = \frac{1}{s^3}.\) Now we can apply \(L16\) with \(a = 2\text{.}\)
    \begin{align*} x(t) =\amp\ \lap^{-1}\left\{ X(s) \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{e^{-2s}}{s^3} \right\}\\ =\amp\ \lap^{-1}\left\{ e^{-2s}\cdot \frac{1}{s^3} \right\}\\ =\amp\ f(t-2)\cdot U(t-2) \end{align*}
    We know exactly what the function \(U(t-2)\) is; but we donâ€™t yet know what \(f(t-2)\) is. We do know the function \(F(s),\) and therefore we need to take the inverse Laplace transform of \(F(s)\) to find the function \(f(t).\)
    \begin{align*} f(t) =\amp\ \lap^{-1}\left\{ F(s) \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{1}{s^3} \right\}\\ =\amp\ \frac{1}{2} \lap^{-1}\left\{ \frac{2}{s^3} \right\}\\ =\amp\ \frac{1}{2}t^2 \end{align*}
    Now that we know the function \(f(t),\) we are prepared to finish assembling our answer.
    \begin{align*} x(t) =\amp\ f(t-2)\cdot U(t-2)\\ =\amp\ \frac{1}{2}(t-2)^2 \cdot U(t-2) \end{align*}

    Example 43.

    Find the inverse Laplace transform of
    \begin{equation*} A(s) = \frac{e^{-3s}}{s^2 + 16}. \end{equation*}
    Solution.
    The cue that we need to use \(L16\) when taking an inverse Laplace transform is the presence of an exponential. We can rewrite \(A(s)\) as
    \begin{align*} A(s) =\amp\ e^{-3s}\cdot \frac{1}{s^2 + 16},\\ =\amp\ e^{-3s}\cdot F(s), \end{align*}
    where \(\ds F(s) = \frac{1}{s^2 + 16}.\) This form aligns with \(L16\), using \(a = 3\text{,}\) and we can proceed to take the inverse Laplace transform as follows.
    \begin{align*} a(t) =\amp\ \lap^{-1}\left\{ A(s) \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{e^{-3s}}{s^2 + 16} \right\}\\ =\amp\ \lap^{-1}\left\{ e^{-3s}\cdot \frac{1}{s^2 + 16} \right\}\\ =\amp\ f(t - 3)\cdot U(t-3) \end{align*}
    We are not done yet; the function \(U(t-3)\) is well-defined, but we donâ€™t yet know what \(f(t-3)\) is. Recall that \(\ds F(s) = \frac{1}{s^2 + 16}.\) First we can find \(f(t)\) by taking the inverse Laplace transform of \(F(s).\)
    \begin{align*} f(t) =\amp\ \lap^{-1}\{ F(s) \}\\ =\amp\ \lap^{-1}\{ \frac{1}{s^2 + 16} \}\\ =\amp\ \lap^{-1}\{ \frac{1}{s^2 + 4^2} \}\\ =\amp\ \frac{1}{4}\cdot \lap^{-1}\{ \frac{4}{s^2 + 4^2} \}\\ =\amp\ \frac{1}{4}\sin(4t) \end{align*}
    Now that we know \(f(t),\) we are prepared to evaluate \(f(t-3).\)
    \begin{align*} a(t) =\amp\ f(t - 3)\cdot U(t-3)\\ =\amp\ \frac{1}{4}\sin\big(4(t-3)\big) \cdot U(t-3) \end{align*}

    Example 44.

    Find the inverse LT of
    \begin{equation*} Y(s) = \frac{se^{-7s}}{s^2 + 4s + 5} \end{equation*}
    Solution.
    It may be easiest for us to rewrite \(Y(s)\) before we begin, as
    \begin{align*} Y(s) =\amp\ e^{-7s}\cdot \frac{s}{s^2 + 4s + 5},\\ =\amp\ e^{-7s}F(s), \end{align*}
    where \(F(s) = \ds \frac{s}{s^2 + 4s + 5}.\) Since we have done several examples like this one, we know that we will need to find the inverse Laplace transform of \(F(s),\) so letâ€™s find that now. This will require some algebraic manipulation as we saw in (name the section here).
    \begin{align*} f(t) =\amp\ \lap^{-1}\left\{ F(s) \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{s}{s^2 + 4s + 5} \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{s}{(s^2 + 4s + 4) + 5 - 4} \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{s}{(s+2)^2 + 1} \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{(s+2) - 2}{(s+2)^2 + 1^2} \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{s+2}{(s+2)^2 + 1^2} - 2\cdot \frac{1}{(s+2)^2 + 1^2} \right\}\\ =\amp\ \lap^{-1}\left\{ \frac{s+2}{(s+2)^2 + 1^2} \right\} - 2\cdot \lap^{-1}\left\{\frac{1}{(s+2)^2 + 1^2} \right\}\\ =\amp\ e^{-2t}\cos(t) - 2e^{-2t}\sin(t) \end{align*}
    Now we return to the question at hand: finding the inverse Laplace transform of \(Y(s).\)
    \begin{align*} y(t) =\amp\ \lap^{-1}\left\{ Y(s) \right\}\\ =\amp\ \lap^{-1}\left\{ e^{-7s}F(s) \right\}\\ =\amp\ f(t-7)\cdot U(t-7)\\ =\amp\ \Big[e^{-2(t-7)}\cos(t-7) - 2e^{-2(t-7)}\sin(t-7)\Big]\cdot U(t-7) \end{align*}
