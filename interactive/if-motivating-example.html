    Subsection A Motivating Example

    In the previous section, we saw that certain first-order linear differential equations can be rewritten so that the left-hand side is a single derivative. However, most equations do not initially have this structure. Instead, they are missing a crucial function that allows us to apply the product rule in reverse.
    The goal now is to illustrate how to determine this missing function—known as the integrating factor—so that we can rewrite one of the sides as a single derivative.
    Consider the equation:
    \begin{equation} \frac{dy}{dx} + 2y = 5.\tag{13} \end{equation}
    The claim is that the left side of this equation is currently an incomplete product rule that can be completed and reversed by multiplying a function \(\mu\) onto both sides of the equation. In short, this allows us to rewrite (13) as
    \begin{equation*} \frac{d}{dx} \left[\mu y \right] = 5\mu. \end{equation*}
    However, the question still remains: Which function, \(\mu\text{,}\) does this? Let’s find out.

    Subsubsection Finding \(\mu(x)\text{:}\) A Step-by-Step Breakdown

    The strategy for determining \(\mu(x)\) can be seen in the slide show

    Details 11. Finding \(\mu\) Slide Show for Equation (13).

    Alternatively, a shorter version is displayed here.
    1️⃣ Multiply by \(\ds\mu\ \left\lbrace\begin{array}{c}\\\\\\\end{array}\right.\)
    \begin{align} \frac{dy}{dx} + 2 y \amp = 5\tag{14}\\ \mu\frac{dy}{dx} + {\color{blue} \us{\downarrow}{\boxed{ 2\mu }}} y\amp = 5\mu\tag{15} \end{align}
    3️⃣ Condition on \(\mu\)
    \begin{equation*} {\color{blue} \boxed{\frac{d\mu}{dx} = 2\mu}} \end{equation*}
    2️⃣ Product Rule \(\ds\ \left\lbrace\vphantom{\begin{array}{c}\\\\\\\end{array}}\right.\)
    \begin{align} \mu\frac{dy}{dx} + {\color{blue} \os{\uparrow}{\boxed{\frac{d\mu}{dx}}}} y \amp = 5 \mu \tag{16}\\ \frac{d}{dx} \left[\mu y \right] \amp = 5\mu \tag{17} \end{align}
    Each step should be followed in the labeled order, detailed as follows:
    1️⃣
    Multiply (14) by \(\mu\) to get (15).
    2️⃣
    Use the product rule on the left side of (17) to get (16).
    3️⃣
    In order for (14) and (17) to be equal, (15) and (16) must also be equal. Therefore, we arrive at the following condition on \(\mu\text{:}\)
    \begin{equation} \frac{d\mu}{dx} = 2\mu\text{.}\tag{18} \end{equation}

    Details 12. Clarifying “3️⃣ Condition on \(\mu\)”.

    At the end of the day, all we are saying here is that in order for
    \begin{equation*} \mu\frac{dy}{dx} + {\color{blue} \boxed{ A }}\ y = 5\mu \end{equation*}
    to be equivalent to
    \begin{equation*} \mu\frac{dy}{dx} + {\color{blue} \boxed{ B }}\ y = 5 \mu \end{equation*}
    then \({\color{blue} A = B}\text{.}\)
    This is how we get the equation for \(\mu\text{.}\) In particular, in order for equation (15) to be equivalent to (16), then
    \begin{equation*} {\color{blue} \frac{d\mu}{dx} = 2\mu}\text{.} \end{equation*}
    This is a separable differential equation, which has the general solution
    \begin{equation*} \mu(x) = ce^{2x}\text{.} \end{equation*}

    Details 13. Solving Equation (18) with separation of variables.

    \begin{align*} \frac{d\mu}{dx} = 2\mu \quad \implies \quad \frac{1}{\mu}\ d\mu \amp = 2 dx \\ \int \frac{1}{\mu}\ d\mu \amp = \int 2 dx \\ \ln|\mu| \amp = 2x + c \\ \mu(x) \amp = ce^{2x} \end{align*}
    Although, any non-zero choice for \(c\) will yield a function that converts (14) into (17), the convention is to chose \(c=1\text{.}\) Therefore, the integrating factor for equation (13) is \(\mu(x) = e^{2x}\text{.}\)
    By multiplying (13) by \(e^{2x}\text{,}\) we are able to transform it into:
    \begin{equation*} \frac{d}{dx} \left[ e^{2x} y \right] = 5e^{2x}. \end{equation*}
    Once in this form, we can solve for \(y\) using direct integration. This example demonstrates how to find the integrating factor for a specific equation. Next, we will generalize this idea by finding a formula that gives \(\mu\) for any first-order linear differential equation.
