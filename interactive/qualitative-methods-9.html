    1. Multiple-Choice.

    (a) Does time matter here?
    For the autonomous equation
    \begin{equation*} y' = y^2 - 1\text{,} \end{equation*}
    the slope field shows a slope of \(-2\) at \((t,y) = (5,-1)\text{.}\) What is the slope at \((t,y) = (-10,-1)\text{?}\)
    üîó
    • \(0\)
    • The slope doesn‚Äôt magically flatten; it‚Äôs determined by \(y\text{.}\)
    • \(2\)
    • The sign doesn‚Äôt flip when \(t\) changes‚Äîtime isn‚Äôt part of the slope function.
    • \(-2\)
    • Correct! Autonomous equations ignore \(t\) entirely. If \(y=-1\text{,}\) the slope is always \(f(-1) = -2\text{,}\) no matter the time.
    • Impossible to know
    • It‚Äôs absolutely possible‚Äîyou just evaluate \(f(y)\) at \(y=-1\text{.}\)
    üîó
    (b) Horizontal Shift Symmetry.
    Fill in the blank: If \(y(t)\) is a solution to an autonomous equation, then _______ is also guaranteed to be a solution.
    üîó
    • \(y(t + c)\)
    • Correct‚Äîautonomous equations allow time-shifted versions of any solution.
    • \(c \, y(t)\)
    • No‚Äîscaling the solution doesn‚Äôt necessarily work unless the equation is linear.
    • \(y(t) + c\)
    • No‚Äîadding a constant changes \(y\) in a way that won‚Äôt satisfy the DE.
    • \(y(t)^c\)
    • Raising the solution to a power doesn‚Äôt preserve the equation‚Äôs behavior.
    üîó
    (c) Time Shifts in Practice.
    A biologist models population growth with the autonomous equation
    \begin{equation*} \frac{dy}{dt} = y(1 - y). \end{equation*}
    They find a solution curve \(y(t)\) that fits data collected in spring. Which of the following will always produce another valid solution?
    üîó
    • \(y(t + 5)\)
    • Yes! Sliding the solution along the t-axis still solves the equation.
    • \(2y(t)\)
    • Scaling \(y\) won‚Äôt necessarily satisfy the same DE.
    • \(y(t) + 5\)
    • Adding to \(y\) changes the values in a way that usually breaks the equation.
    • \(y(-t)\)
    • Reversing time is not the same as sliding it‚Äîit usually won‚Äôt satisfy the DE.
    üîó
    (d) Equilibrium Identification.
    For the equation
    \begin{equation*} \dfrac{dy}{dt} = y^2 - 4y + 3, \end{equation*}

    5. Find the Equilibrium Solution.

    Find the equilibrium solutions for the autonomous differential equation
    \begin{equation*} \dfrac{dy}{dx} = y^2\text{.} \end{equation*}
    üîó
    Solution.
    To find equilibrium solutions, we set \(f(y) = y^2 = 0\text{.}\) This gives us the solution \(y = 0\text{.}\) So, \(y = 0\) is the only equilibrium solution for this equation.
    üîó
    This means that if the solution starts at \(y(0) = 0\text{,}\) it will remain at zero for all time. If it starts anywhere else, it will either increase or decrease away from zero.
    üîó
    üîó

    6. Finding Equilibrium Points.

    Find all equilibrium solutions of the autonomous differential equation
    \begin{equation*} \dfrac{dy}{dx} = y^2 - y^4 = 0\text{.} \end{equation*}
    üîó
    Solution.
    To find the equilibrium solutions, we set the right-hand side equal to zero:
    üîó
    \begin{equation*} y^2 - y^4 = 0. \end{equation*}
    Factoring gives us:
    üîó
    \begin{equation*} y^2(1 - y^2) = 0. \end{equation*}
    This has solutions:
    üîó
    1. \(\displaystyle y = 0\)
      üîó
    2. \(\displaystyle y = 1\)
      üîó
    3. \(\displaystyle y = -1\)
      üîó
    These are the equilibrium solutions where the slope is zero. At these points, the solution does not change over time.
