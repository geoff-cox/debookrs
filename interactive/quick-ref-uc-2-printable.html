    ğŸ”—

    ğŸ”—
    A particular solution, \(y_h\text{,}\) is a function that can be plugged into (0) and all the terms combine to give the forcing function, \(f(x)\text{.}\)

    Homogeneous Solution
    ğŸ”—
    A homogeneous solution, \(y_h\text{,}\) is a function such that, when it is plugged into (0) all the terms cancel to zero. That is, \(y_h\) is the solution to the homogeneous version of (0) (\(f(x) = 0 \)).
