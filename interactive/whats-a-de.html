    Section What is a Differential Equation?

    üìù: üéß Listen.

    üîó
    In this section, we‚Äôll explore what a differential equation is, starting with an analogy, and then moving to a formal definition.
    üîó

    Subsection Connection to Algebra and Calculus

    To learn something new, it helps to connect it to something you already know. So let‚Äôs begin by linking differential equations to algebra.
    üîó
    Suppose you‚Äôre asked to solve each of the following equations for \(y\text{:}\)
    üîó
    \begin{equation*} y + 2 = 11, \qquad y + 2t = 11, \qquad y^3 + 2t = 11 \end{equation*}
    The goal of algebra is to figure out what number or function goes in place of \(y\) to make both sides equal. Instead of guessing, you learned rules for isolating \(y\text{.}\) If we apply those rules, we get:
    üîó
    \begin{align*} y = 9, \qquad y = 11 - 2t, \qquad y^3 \amp = 11 - 2t \\ y \amp = \left(11 - 2t\right)^{1/3} \end{align*}
    Now suppose you‚Äôre asked to solve a new equation:
    üîó
    \begin{equation*} y' + 2t = 11 \end{equation*}
    As before, we isolate \(y'\text{,}\) giving:
    üîó
    \begin{equation*} y' = 11 - 2t \end{equation*}
    So what function has a derivative of \(11 - 2t\text{?}\) In other words, what goes in place of \(y\) to make this equation true?
    üîó
    From calculus, you might recognize this as an anti-derivative problem. So the solution is:
    üîó
    \begin{equation*} y = \int (11 - 2t)\, dt = 11t - t^2 + C. \end{equation*}
    Believe it or not, you just solved a differential equation using algebra and calculus! Most problems won‚Äôt be quite that straightforward, but the goal is the same: find a function that fits in place of \(y\) and makes the equation true.
    üîó
    üîó

    Subsection Definition

    üìù: üéß Listen.

    üîó
    Essentially, a differential equation is any equation where the unknown has a derivative applied to it. Here‚Äôs the formal definition.
    üîó

    üìô Definition 1. Differential Equation.

    A differential equation (DE) is an equation that involves one or more derivatives of an unknown function. If the function depends on a single variable, it‚Äôs called an ordinary differential equation (ODE). If the function depends on more than one variable, it‚Äôs called a partial differential equation (PDE).
    üîó
    üîó

    üìù: DE \(\Rightarrow\) ODE.

    üìå Since this course only covers ordinary differential equations, any time we say differential equation or DE, we mean ODE.
    üîó
    So, for an equation to qualify as a differential equation, it must include two things: First, it needs a derivative‚Äîsomething like \(f'\) or \(\frac{dy}{dx}\text{.}\) Second, it must have an equals sign.
    üîó
    That means all of the following are differential equations:
    üîó
    \begin{equation*} \frac{dy}{dx} + 1 = y, \qquad f^{\prime\prime} + x^2 + 3x = 19, \qquad e^t = \tan(y^\prime) \end{equation*}
    On the other hand, these are not differential equations‚Äîeither because they don‚Äôt contain a derivative, or because they don‚Äôt include an equals sign:
    üîó
    \begin{equation*} \frac{d^2 y}{dx^2} + 2\frac{dy}{dx}, \qquad \sin y + e^x = 0 \end{equation*}

    Checkpoint 2. üìñ‚ùìDifferential Equation Definition.

    (a) üìñ‚ùì Not Required for a DE.
    Which of the following is NOT required for an equation to be classified as a differential equation?
    • A \(y\)-variable
    • Correct! A \(y\)-variable is not a requirement for a differential equation.
    • An unknown function
    • Incorrect. A differential equation does include an unknown function, which we are solving for.
    • One or more derivatives of an unknown function
    • Incorrect. The presence of at least one derivative is essential to define a differential equation.
    • An equal sign, "="
    • Incorrect. An equality sign is required for an equation to be classified as a differential equation.
    üîó
    (b) üìñ‚ùì Identifying the Differential Equation.
    Which of the following equations is a differential equation.
    • \(\quad y'' + 1 = y\)
    • Correct! This equation involves a derivative, making it a differential equation.
    • \(\quad x^2 + 3x = 19\)
    • Incorrect. This equation does not contain any derivatives, so it is not a differential equation.
    • \(\quad \sin y + e^x = 0\)
    • Incorrect. This equation does not contain any derivatives, so it is not a differential equation.
    • \(\quad y^2 + 5 = 0\)
    • Incorrect. This equation does not contain any derivatives, so it is not a differential equation.
    üîó
    üîó

    Subsection üì§ Wrap-Up

    üóùÔ∏è \(\textbf{Key Takeaways...}\)
    • A differential equation involves an unknown function and at least one of its derivatives.
      üîó
    • It must contain an equals sign to be considered an equation.
      üîó
    • Differential equations are a natural extension of algebra and calculus.
      üîó
    • Notation varies by context but generally falls into prime, Leibniz, or dot notation.
