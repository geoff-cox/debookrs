    2. Multiple-Choice.

    (a) Integration Technique Used.
    What integration technique is used to evaluate
    \begin{equation*} \int t^3 \ln t \, dt\text{?} \end{equation*}
    üîó
    • Substitution.
    • Substitution isn‚Äôt the best choice for this integral.
    • Integration by parts.
    • Correct! Integration by parts handles products like \(t^3 \ln t\text{.}\)
    • Partial fraction decomposition.
    • Partial fractions don‚Äôt apply here.
    • The product rule.
    • The product rule is for differentiation, not integration.
    üîó
    (b) When to Use Direct Integration.
    What method is used to solve a differential equation of the form
    \begin{equation*} \frac{d}{dx}[g(x,y)] = f(x)\text{?} \end{equation*}
    üîó
    • Direct integration.
    • Correct! Integrate both sides with respect to \(x\text{.}\)
    • Separation of variables.
    • Separation isn‚Äôt needed‚Äîthe equation integrates directly.
    • Substitution.
    • Substitution isn‚Äôt the primary method here.
    • Partial fractions.
    • Partial fractions are sometimes used inside integration, but the method here is simply direct integration.
    üîó
    (c) Conditions for Using the Integrating Factor Method.
    What properties must a differential equation have for the integrating factor method to apply?
    üîó
    • First order.
    • Correct! IF method is for first-order equations.
    • Separable.
    • Separable equations don‚Äôt require the integrating factor method.
    • Linear.
    • Correct! The IF method works only for linear equations.
    • Quadratic.
    • ‚ÄúQuadratic‚Äù refers to algebraic equations, not differential equations.
    üîó
    (d) Choosing the Integrating Factor.
    For the linear equation \(y' + 3y = x\text{,}\) what is the integrating factor?
    üîó
    • \(\ds e^{\int 3 \, dx} = e^{3x}\)
    • Correct! The integrating factor is built from \(P(x) = 3\text{.}\)
    • \(\ds e^{\int x \, dx} = e^{x^2/2}\)
    • This would come from \(Q(x)\text{,}\) not \(P(x)\)‚Äîwrong choice.
    • \(\ds e^{-3x}\)
    • Watch the sign‚Äîthe integrating factor has \(e^{+\int P(x) dx}\text{.}\)
    • \(\ds 3e^x\)
    • The integrating factor isn‚Äôt just an arbitrary exponential‚Äîit comes from integrating \(P(x)\text{.}\)
    üîó
    (e) Purpose of Multiplying by the Integrating Factor.
    Why do we multiply a first-order linear equation by its integrating factor?
    üîó
    • To rewrite the equation so it can be integrated directly.
    • Correct! The integrating factor allows the left-hand side to become \(\frac{d}{dx}[\mu(x)y]\text{,}\) which is integrable.
    • To convert the equation into a separable equation.
    • Not quite‚Äîalthough integration becomes possible, the equation doesn‚Äôt become separable.
    • To set up the product rule for solving the equation.
    • The product rule is reversed here, not applied to expand further.
    • To make the equation factorable.
    • The integrating factor rewrites the equation into an integrable form, not just a ‚Äúfactorable‚Äù one.
    üîó
    (f) Next Step After Using the Integrating Factor.
    You rewrote \(\dfrac{dy}{dx} + 2y = 5\) as
    \begin{equation*} \frac{d}{dx} \left[e^{2x} y\right] = 5e^{2x}\text{.} \end{equation*}
    What should you do next?
    üîó
    • Apply the product rule to expand the left-hand side.
    • The product rule was already reversed to reach this form; expanding again would undo progress.
    • Integrate both sides to solve for \(y\text{.}\)
    • Correct! Once the left-hand side is a single derivative, integration gives the solution for \(y\text{.}\)
    • Multiply by another integrating factor.
    • Only one integrating factor is needed. Multiplying by another one is unnecessary.
    • Differentiate both sides to simplify the expression.
    • Further differentiation is not needed; integration is the next step.
    üîó
    (g) Choosing the IF for a Given Equation.
    Find the integrating factor for
    \begin{equation*} x^2 y' - y = 1\text{.} \end{equation*}
    üîó
    • \(e^{1/x^2}\)
    • Not quite‚Äîcheck the exponent carefully.
    • \(x^2\)
    • This is not the exponential integrating factor needed.
    • \(e^{1/x}\)
    • Correct! The integrating factor comes from \(P(x) = -\frac{1}{x^2}\text{.}\)
    • \(\frac{1}{x}\)
    • This is not the correct integrating factor.
    üîó
    (h) Sequence of IF Method Steps.
    Which of the following best represents a succinct version of the integrating factor method? Let \(\mu\) be the integrating factor.
    üîó
    • standard form‚Äîcompute \(\mu\)‚Äîmultiply \(\mu\)‚Äîintegrate‚Äîisolate
    • Correct! That‚Äôs the sequence every time.
    • compute \(\mu\)‚Äîmultiply \(\mu\)‚Äîdifferentiate‚Äîisolate
    • Differentiation is not used.
    • integrate‚Äîcompute \(\mu\)‚Äîmultiply \(\mu\)‚Äîstandard form‚Äîisolate.
    • Correct steps, wrong order.
    • isolate‚Äîstandard form‚Äîcompute \(\mu\)‚Äîmultiply \(\mu\)‚Äîintegrate.
    • Correct steps, wrong order.

    Exercises üèóÔ∏è Warm-ups & Drills

    Which Method Applies.

    For each differential equation below, identify which statement applies and explain how you know. You do not need to solve any of the equations.
    üîó
    1. Can be solved by separating variables, but not using an integrating factor.
      üîó
    2. Can be solved using an integrating factor, but not by separating variables.
      üîó
    3. Can be solved both by using an integrating factor and by separating variables.
      üîó
    4. Cannot be solved using either technique.
      üîó
    1.
    \(x^2 \dfrac{dy}{dx} + \cos x = y\)
    Answer.
    b. Can be solved using an integrating factor, but not by separating variables.
    üîó
    üîó
    Solution.
    This DE is first order and linear, so it can be solved using an integrating factor.
    üîó
    It is not separable. After isolating the derivative:
    \begin{equation*} \dfrac{dy}{dx} = \dfrac{y - \cos x}{x^2}, \end{equation*}
    we see that \(y\) and \(x\) cannot be separated by multiplication or division.
    üîó
    Therefore, this DE can be solved using an integrating factor, but not by separating variables.
    üîó
    üîó
    üîó
    2.
    \(\dfrac{dx}{dt} + xt = e^x\)
    Answer.
    d. Cannot be solved using either technique.
    üîó
    üîó
    Solution.
    This DE is first order but not linear because the dependent variable \(x\) appears inside the nonlinear term \(e^x\text{.}\)
    üîó
    It is also not separable. After isolating the derivative:
    \begin{equation*} \frac{dx}{dt} = e^x - xt, \end{equation*}
    there‚Äôs no way to write the right-hand side as a product of a function of \(x\) and a function of \(t\text{.}\)
    üîó
    Therefore, this DE cannot be solved using either technique.
    üîó
    üîó
    üîó
    3.
    \((t^2 + 1) \dfrac{dy}{dt} = yt - y\)
    Answer.
    c. Can be solved both by using integrating factor and by separating variables.
    üîó
    üîó
    Solution.
    This DE is first order and linear, so it can be solved using an integrating factor.
    üîó
    It is also separable. Solving for the derivative:
    \begin{align*} (t^2 + 1) \frac{dy}{dt} \amp = yt - y\\ \frac{dy}{dt} \amp = \frac{yt - y}{t^2+1}\\ \frac{dy}{dt} \amp = \frac{y(t - 1)}{t^2+1}\\ \frac{dy}{dt} \amp = y \cdot \frac{t - 1}{t^2+1} \end{align*}
    Here, we can separate the variables: \(\dfrac{dy}{y} = \dfrac{t - 1}{t^2 + 1} dt\text{.}\)
    üîó
    Therefore, this DE can be solved both by using an integrating factor and by separating variables.
    üîó
    üîó
    üîó
    4.
    \(\dfrac{dy}{dt} - y^2 t = t\)
    Answer.
    a. Can be solved by separating variables, but not using an integrating factor.
    üîó
    üîó
    Solution.
    This DE is first order but not linear because the dependent variable \(y\) is raised to a power (\(y^2\)).
    üîó
    It is separable. Solving for the derivative:
    \begin{align*} \frac{dy}{dt} - y^2 t \amp = t\\ \frac{dy}{dt} \amp = t + y^2 t\\ \frac{dy}{dt} \amp = t \cdot (1 + y^2) \end{align*}
    allows us to separate variables: \(\dfrac{dy}{1+y^2} = t \, dt\text{.}\)
    üîó
    Therefore, this DE can be solved by separating variables, but not using an integrating factor.
    üîó
    üîó
    üîó
    5.
    \(3r = \dfrac{dr}{d\theta} - \theta^3\)
    Answer.
    b. Can be solved using an integrating factor, but not by separating variables.
    üîó
    üîó
    Solution.
    This DE is first order and linear, so it can be solved using an integrating factor.
    üîó
    It is not separable. Solving for the derivative:
    \begin{equation*} \frac{dr}{d\theta} = 3r + \theta^3, \end{equation*}
    shows no way to separate \(r\) and \(\theta\) into separate multiplicative terms.
    üîó
    Therefore, this DE can be solved using an integrating factor, but not by separating variables.

    Exercises ‚úçüèª Solve the Differential Equations

    General Solution.

    Use an integrating factor to find the general solution to each equation.
    üîó
    1.
    \(y' - 4y = x\)
    Answer.
    \(y = -\frac{1}{4}x - \frac{1}{16} + Ce^{4x}\)
    üîó
    üîó
    Solution.
    Standard form: \(y' + (-4)y = x\text{,}\) so \(P(x) = -4\text{.}\) Integrating factor: \(\mu(x) = e^{\int -4 dx} = e^{-4x}\text{.}\)
    üîó
    \begin{align*} e^{-4x}y' - 4e^{-4x}y \amp = xe^{-4x}\\ \frac{d}{dx}[e^{-4x}y] \amp = xe^{-4x} \end{align*}
    Integrate:
    üîó
    \begin{align*} e^{-4x}y \amp = \int xe^{-4x} dx = -\frac{1}{4}xe^{-4x} - \frac{1}{16}e^{-4x} + C\\ y \amp = -\frac{1}{4}x - \frac{1}{16} + Ce^{4x} \end{align*}
    üîó
    üîó
    2.
    \(x^2 y' - y = 1, \quad x > 0\)
    Answer.
    \(y = -1 + Ce^{-1/x}\)
    üîó
    üîó
    Solution 1.
    Standard form: \(y' - \frac{1}{x^2}y = \frac{1}{x^2}\text{.}\) Integrating factor: \(\mu(x) = e^{\int -\frac{1}{x^2}dx} = e^{1/x}\text{.}\)
    üîó
    \begin{align*} \frac{d}{dx}[e^{1/x}y] \amp = \frac{e^{1/x}}{x^2}\\ e^{1/x}y \amp = -e^{1/x} + C\\ y \amp = -1 + Ce^{-1/x} \end{align*}
    üîó
    Solution 2. (detailed)
    We put the equation in standard form to identify \(P(x)\) and compute the integrating factor.
    üîó
    \begin{equation*} y' + \us{P}{\boxed{-\frac{1}{x^2}}}\ y = \frac{1}{x^2} \quad \implies \quad \mu(x) = e^{\large \int -\sfrac{1}{x^2} dx} = e^{1/x} \end{equation*}
    üîó
    Multiply by \(e^{1/x}\) and reverse the product rule on the left side.
    \begin{align*} e^{1/x}y' - \frac{e^{1/x}}{x^2} y \amp = \frac{e^{1/x}}{x^2} \\ \frac{d}{dx}\left[e^{1/x} y\right] \amp = \frac{e^{1/x}}{x^2} \end{align*}
    üîó
    Next, integrate both sides and use \(u\)-substitution on the right side.
    üîó
    \begin{align*} \int \frac{d}{dx}\left[e^{1/x} y\right] dx \amp = \int \frac{e^{1/x}}{x^2} dx \qquad \rightarrow\\ e^{1/x} y \amp = \int -e^u du \\ e^{1/x} y \amp = -e^u + c \\ e^{1/x} y \amp = -e^{1/x} + c \end{align*}
