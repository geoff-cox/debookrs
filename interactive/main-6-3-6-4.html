    Subsubsection Integrating Factor

    Suppose \(f\) and \(g\) are differentiable functions. The product rule states that
    \begin{equation} \underset{\xrightarrow[\LARGE\text{forward}]{} }{\ds [f \cdot g ]'} = \underset{\xleftarrow[\LARGE\text{backward}]{} }{\ds f \cdot g' + f' \cdot g}.\tag{31} \end{equation}
    The “forward” version goes from left to right, while the “backward” version, which we’ll focus on, goes from right to left. Let’s practice applying both versions with a few examples.

    Details 23. Solving (29).

    The integrating factor for the first-order linear differential equation
    \begin{equation*} y' + P(x)y = Q(x) \end{equation*}
    is \(\mu = e^{\int P(x) dx}\text{.}\) Multiplying both sides by \(\mu\) transforms the equation into:
    \begin{equation*} \frac{d}{dx}\left[\mu(x) \cdot y\right] = \mu(x)Q(x)\text{.} \end{equation*}

    Derivation of the Integrating Factor.

    Note, all first-order linear differential equations can be written in the form
    \begin{equation} \frac{dy}{dx} + P(x) y = Q(x)\text{,}\tag{32} \end{equation}
    where \(P(x)\) and \(Q(x)\) are functions of \(x\text{.}\) The goal is to write the left-side as a single derivative. However, backward product rule is missing a piece, \(\mu(x)\text{,}\) that we still need to find. We find \(\mu(x)\) by multiplying it onto (32), like so
    \begin{align*} \frac{dy}{dx} + P(x)\amp\ y = Q(x) \\ \mu \frac{dy}{dx} + {\color{blue} \mu P(} \amp\ {\color{blue}\! x)} y = \mu Q(x) \\ {\color{blue} \updownarrow}\ \ \amp \ {\color{blue} \text{match & solve}}\\ \mu \frac{dy}{dx} + {\color{blue} \frac{d\mu}{dx}}\amp\ y = \mu Q(x) \\ \frac{d}{dx}\big[\mu \cdot \amp\ y\big] = \mu Q(x) \end{align*}
    \begin{align*} {\color{blue} \frac{d\mu}{dx} =}\amp\ {\color{blue} \mu P(x)} \\ {\color{blue} \frac{1}{\mu}d\mu =}\amp\ {\color{blue} P(x) dx} \\ {\color{blue} \int \frac{1}{\mu}d\mu =}\amp\ {\color{blue} \int P(x) dx} \\ {\color{blue} \ln|\mu| =}\amp\ {\color{blue} \int P(x) dx} \\ {\color{blue} \mu =}\amp\ {\color{blue} e^{\int P(x) dx}} \end{align*}

    Another Approach to the IF Method...

    The integrating factor method uses a three-step strategy to solve first-order linear differential equations in the standard form:
     2 
    Every linear differential equation looks like
    \begin{equation*} a_n(x)y^{(n)} + \cdots + a_2(x)y'' + a_1(x)y' + a_0(x)y = f(x)\text{.} \end{equation*}
    The first-order version of this is
    \begin{equation*} a_1(x)y' + a_0(x)y = f(x)\text{,} \end{equation*}
    but we can divide both sides of this equation by \(a_1(x)\text{:}\)
    \begin{equation*} y' + \us{P(x)}{\ub{\frac{a_0(x)}{a_1(x)}}}y = \us{Q(x)}{\ub{\frac{f(x)}{a_1(x)}}}\text{.} \end{equation*}
    Since every first-order linear differential equation can always be written as
    \begin{equation*} y' + P(x)y = Q(x)\text{,} \end{equation*}
    we call this the first-order linear standard form.
    \begin{equation*} y' + P(x)y = Q(x)\text{.} \end{equation*}
    The heart of this method is the first step: finding the correct integrating factor. The second step uses this factor to write the left side as a single derivative. Finally, the last step finds the solution by direct integration.

    Step 1: Finding the Integrating Factor.

    Understanding how to find the integrating factor is crucial. Let’s carefully walk through this important process.
    Consider the differential equations (specific on the left, general on the right):
    \begin{equation*} \us{\text{product rule?}}{\ub{y' + x^2 y}} = 5x^2\text{,} \end{equation*}
    \begin{equation*} \us{\text{product rule?}}{\ub{y' + P(x)y}} = Q(x)\text{.} \end{equation*}
    We would like to rewrite the left side as a single derivative using the backward product rule. However, the \(y'\) alone suggests there is a missing second function that the product rule needs. Maybe there is a function that we can multiply both sides of the equation by to get the desired product rule. Let’s call it \(\mu\) for now and see if we can find it. We start by multiplying both sides by \(\mu\text{:}\)
    \begin{equation*} \mu y' + 3x^2 \mu y = 5x^2\mu\text{.} \end{equation*}
    \begin{equation*} \mu y' + P(x)\mu y = Q(x)\mu\text{.} \end{equation*}
    Notice that the left side almost looks like the product rule \(\left[ \mu y \right]' = \mu y' + \mu' y\text{,}\) except for the boxed value where \(\mu'\) should be (see below).
    \begin{equation*} \overset{\ds \mu} {\overset{\downarrow}{\vphantom{\Big[}\ds \mu}} \cdot \overset{\ds y'} {\overset{\downarrow}{\vphantom{\Big[}\ds y'}} + \overset{\ds \color{blue}{\mu'}} {\overset{\downarrow}{\vphantom{\Big[}\ds \color{blue}{\boxed{3x^2 \mu}}}} \cdot \overset{\ds y} {\overset{\downarrow}{\vphantom{\Big[}\ds y}} = 5x^2\mu\text{,} \end{equation*}
    \begin{equation*} \overset{\ds \mu} {\overset{\downarrow}{\vphantom{\Big[}\ds \mu}} \cdot \overset{\ds y'} {\overset{\downarrow}{\vphantom{\Big[}\ds y'}} + \overset{\ds \color{blue}{\mu'}} {\overset{\downarrow}{\vphantom{\Big[}\ds \color{blue}{\boxed{P(x) \mu}}}} \cdot \overset{\ds y} {\overset{\downarrow}{\vphantom{\Big[}\ds y}} = Q(x)\mu\text{.} \end{equation*}
    This shows that the \(\mu\) that works is the one that satisfies
    \begin{equation*} \mu' = 3x^2 \mu \end{equation*}
    \begin{equation*} \mu' = P(x) \mu\text{.} \end{equation*}
    Careful inspection shows that these are separable differential equations where the unknown \(\mu\) can be found using separation of variables!
    \begin{align*} \int \frac{1}{\mu}d\mu \amp = \int 3x^2\ dx \\ \ln|\mu| \amp = x^3 + C \\ \mu \amp = \pm e^{x^3 + C} \\ \mu \amp = {\color{green} \us{C}{\boxed{\pm e^C}}} e^{x^3} \\ \mu \amp = Ce^{x^3} \end{align*}
    \begin{align*} \int \frac{1}{\mu}d\mu \amp = \int P(x)\ dx \\ \ln|\mu| \amp = \int P(x)\ dx \\ \mu \amp = e^{\int P(x)\ dx} \end{align*}
    Although the solution conatains a constant, we only need one function that works, so we set \(C = 1\text{.}\) The function \(\mu\) is called the integrating factor and finding it is the first step toward solving a first order differential equation.

    Step 2: Rewriting the Equation.

    backward product rule
    Integrating Factor: \(\mu = e^{x^3}\)
    \begin{align*} y' + x^2 y =\ \amp 5x^2 \\ e^{x^3}y' + x^2 e^{x^3}y =\ \amp 5x^2e^{x^3} \\ \left[ e^{x^3}y \right]' =\ \amp 5x^2e^{x^3} \end{align*}
    Integrating Factor: \(\mu = e^{\int P(x)\ dx}\)
    \begin{align*} y' + P(x)y =\ \amp Q(x) \\ \mu y' + P(x) \mu y =\ \amp Q(x) \mu \\ \left[ \mu y \right]' =\ \amp Q(x) \mu \text{.} \end{align*}

    Step 3: Solving the Equation.

    The final step is to apply direct integration. This was covered in a previous section, so we will skip some of the details here.
    \begin{align*} \int \left[ e^{x^3}y \right]' \ dx =\ \amp \int 5x^2e^{x^3} \ dx \\ e^{x^3}y =\ \amp \frac{5}{3} e^{x^3} + C \\ y =\ \amp \frac{5}{3} + Ce^{-x^3} \end{align*}
    \begin{align*} \int \left[ \mu y \right]' \ dx =\ \amp \int Q(x) \mu \ dx \\ \mu y =\ \amp \int Q(x) \mu \ dx \\ y =\ \amp \frac{1}{\mu} \int Q(x) \mu \ dx \end{align*}
    where \(\mu = e^{\int P(x)\ dx}\text{.}\)
    General Solution.
    The general solution for the first-order linear differential equation
    \begin{equation*} y' + P(x)y = Q(x) \end{equation*}
    is given by
    \begin{equation*} y = \frac{1}{\mu} \int Q(x) \mu \ dx \end{equation*}
    where \(\mu = e^{\int P(x)\ dx}\) is the integrating factor.
    In summary, the integrating factor method is a three-step process that utilizes the integrating factor to rewrite the left side of the equation, enabling direct integration and leading to the general solution.

    Another Approach to the IF Method...

    \begin{align*} \frac{d}{dx}\left[ y \cdot e^{-2x} \right] \amp = 5 e^{-2x} \quad \leftarrow \text{Easy to solve!} \end{align*}
    and then we integrate both sides with respect to \(x\) to free the unknown \(y\) from inside the \(\ds\frac{d}{dx}\text{,}\) so that we can solve for it. However, we have not talked about how the integrating factor gets the differential equation into this form. Luckily, the product rule for deriviatives holds the answer. Therefore, let’s start this discussing by reminding ourselves that the product rule is given by
    \begin{equation*} \frac{d}{dx}[\ f \cdot g\ ] = \frac{d}{dx}[\ f\ ] \cdot g + f \cdot \frac{d}{dx}[\ g\ ]. \end{equation*}
    Ok, the claim is that after a some math the differential equation
    \begin{equation} \frac{dy}{dx} - 2y = 5 \tag{33} \end{equation}
    can be rewritten as
    \begin{equation} \frac{d}{dx}\left[ y \cdot e^{-2x} \right] = 5 e^{-2x}. \tag{34} \end{equation}
    To explain this, it may be easier to work backwards. That is, let’s start with (34) and see how it may connect back to (33). We start by applying the product rule on the left side,
    \begin{align} \frac{d}{dx}[\ y \cdot e^{-2x}\ ] \amp = 5 e^{-2x} \tag{35}\\ \frac{d}{dx}[\ y\ ] \cdot e^{-2x} + y \cdot \frac{d}{dx}[\ e^{-2x}\ ] \amp = 5 e^{-2x} \tag{36}\\ \frac{dy}{dx} \cdot e^{-2x} + y \cdot \left(-2 e^{-2x}\right) \amp = 5 e^{-2x} \tag{37}\\ \frac{dy}{dx} \cdot e^{-2x} - 2 y \cdot e^{-2x} \amp = 5 e^{-2x}. \tag{38} \end{align}
    At this point stop and compare equation (33) with the last equation (38). Do you see how the two equations are connected?
    You should see that multiplying both sides of (33) by \(e^{-2x}\) gives you (38).
    In a nutshell, the integrating factor technique attempts to solve
    \begin{equation*} y' + P(x)y = Q(x) \end{equation*}
    by multiplying both sides by an unknown function, \(\mu\)
    \begin{equation*} \mu y' + \mu P(x)y = \mu Q(x) \end{equation*}
    then find a \(\mu\) so that the left-hand side can be written as the product rule of \(\mu\) and \(y\text{.}\)
    \begin{equation*} \us{\large \text{product rule}}{\ub{\mu y' + \mu P(x)y}} = \mu Q(x) \end{equation*}
    To clarify, let’s look at an example where we know \(\mu\text{.}\)
    The previous example shows how to solve the equation when you know what \(\mu\) is. However, in a real problem, you will have to find \(\mu\) first!
    Let’s build on the previous example to try and understand why \(\mu = e^{6x}\) worked. It is important to try and think of \(\mu(x)\) as the missing part needed to to be consistent with the product rule. That is, the left-side of the equation
    \begin{equation*} \us{\large \text{product rule}}{\us{\large \text{incomplete}}{\ub{y' + 6y}}} = 1 \end{equation*}
    is missing a second function needed to be consistent with the product rule. So let’s fill-in this missing function by multiplying both sides by \(\mu\text{,}\)
    \begin{equation} \mu y' + 6 \mu y = \mu.\tag{39} \end{equation}
    Is the left-side, \(\mu y' + 6 \mu y\text{,}\) equal to \(\left[\mu y\right]'\text{?}\) Well, not exactly, since taking this derivative shows the difference between these two, namely
    \(\ds\left[\mu y\right]'\) \(=\) \(\mu y'\) \(+\) \(\mu'\) \(y\)
    \(\uparrow\)
    difference
    \(\downarrow\)
    left-side \(\rightarrow\) \(\mu y'\) \(+\) \(6\mu\) \(y\)
    However, what if \(\mu\) was chosen so that \(\mu' = 6\mu\text{?}\) It turns out that this is exactly how we chose \(\mu\text{.}\) Finding \(\mu\) such that \(\mu' = 6\mu\) is itself a differential equation which can be solved using separation of variables. (You may try it yourself first for practice, then look below to see how to solve this DE.)
    Using separation of variables, solve
    \begin{align*} \mu' =\amp\ 6\mu. \end{align*}
    to get the solution
     3 
    \begin{align*} \frac{d\mu}{dx} =\amp\ 6\mu\\ \frac{1}{\mu}d\mu =\amp\ 6\ dx\\ \int\frac{1}{\mu}d\mu =\amp\ \int 6\ dx\\ \ln|\mu| =\amp\ 6x + C\\ |\mu| =\amp\ e^{6x + C}\\ \mu =\amp\ \pm e^C e^{6x}\\ \mu =\amp\ A e^{6x} \end{align*}
    \begin{equation*} \mu = A e^{6x}. \end{equation*}
    Since, we only need one \(\mu\) to work, we choose \(A=1\) for convenience, hence
    \begin{equation*} \mu = e^{6x}. \end{equation*}
    Now you see where the \(\mu\) came from in part 1 of ??. In fact, this example outlines how \(\mu\) is found for any first order linear differential equation of the form
    \begin{equation} y' + P(x)y = Q(x).\tag{40} \end{equation}
