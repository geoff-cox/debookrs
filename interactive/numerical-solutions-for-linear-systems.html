    Section 14.6 Euler‚Äôs Method

    We‚Äôve explored what linear systems are, how to visualize their behavior, and even how to write them neatly in vector form. Now it‚Äôs time to start solving them.
    üîó
    We‚Äôll begin with the classic trick that worked for single equations ‚Äî guessing exponential solutions ‚Äî then finish by showing how to approximate solutions step by step with a numerical method.
    üîó

    Subsection Numerical Solutions for Linear Systems

    Not every system is easy to solve exactly. Sometimes, you just want to see what the solution looks like ‚Äî or approximate it for a while. That‚Äôs where numerical methods shine.
    üîó
    One of the simplest is Euler‚Äôs Method, which you‚Äôve seen for single equations. We‚Äôll now apply it to systems.
    üîó
    Suppose we have:
    \begin{align*} \frac{dx}{dt} \amp = x + y\\ \frac{dy}{dt} \amp = -x + y \end{align*}
    with initial conditions \(x(0) = 1\text{,}\) \(y(0) = 0\text{,}\) and a step size of \(h = 0.1\text{.}\)
    üîó

    üåå Example 285. Euler‚Äôs Method for a System.

    Use Euler‚Äôs method to take two steps for the system above.
    üîó
    Solution.
    Step 1: At \(t = 0\text{,}\) we have \(x_0 = 1\text{,}\) \(y_0 = 0\text{.}\) The derivatives are \(x' = 1 + 0 = 1\text{,}\) \(y' = -1 + 0 = -1\text{.}\)
    üîó
    Update:
    üîó
    \begin{equation*} x_1 = 1 + 0.1(1) = 1.1, \qquad y_1 = 0 + 0.1(-1) = -0.1. \end{equation*}
    Step 2: At \(t = 0.1\text{,}\) we have \(x_1 = 1.1\text{,}\) \(y_1 = -0.1\text{.}\) Now \(x' = 1.1 + (-0.1) = 1.0\text{,}\) \(y' = -1.1 + (-0.1) = -1.2\text{.}\)
    üîó
    Update again:
    üîó
    \begin{equation*} x_2 = 1.1 + 0.1(1.0) = 1.2, \qquad y_2 = -0.1 + 0.1(-1.2) = -0.22. \end{equation*}
    We now have approximate values after two steps:
    üîó
    \begin{equation*} (x(0.2), y(0.2)) \approx (1.2, -0.22). \end{equation*}
    Each step shows how \(x\) and \(y\) ‚Äútalk‚Äù to each other as time progresses.
    üîó
    üîó
    üîó
    Euler‚Äôs Method may be simple, but it makes the flow of information between variables visible: each update for \(x\) affects \(y\) on the next step, and vice versa.
    üîó

    Checkpoint 286. ü§îüí≠ Numerical Solutions for Linear Systems Reading Questions.

    (a) ü§îüí≠ Numerical Methods for Systems.
    (i) Euler‚Äôs Method Thinking.
    What does Euler‚Äôs Method do when applied to a system of DEs?
    üîó
    • It updates each variable step by step using its derivative at the current point.
    • It shows how the variables influence each other from one step to the next.
    • It provides the exact solution in one step.
    • Euler‚Äôs Method is an approximation technique, not an exact one-step solution.
    • It can only be used for uncoupled systems.
    • Euler‚Äôs Method works for any system, coupled or not.
    üîó
    (ii) Interpreting Numerical Results.
    When we use Euler‚Äôs Method on a system, what do the computed points \((x_n, y_n)\) represent?
    üîó
    • They are approximate values of the solution at specific time steps.
    • They are exact values that replace the need for solving the system algebraically.
    • They‚Äôre approximate ‚Äî we still use exact methods when possible.
    • They are estimates of the system‚Äôs coefficients.
    • Euler‚Äôs Method updates \(x\) and \(y\text{,}\) not the coefficients.
    üîó
