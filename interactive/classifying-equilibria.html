    Section Classifying Equilibrium Solutions

    Equilibrium solutions mark the places where a system comes to rest. But not all equilibrium solutions are alike, some attract nearby solutions, others repel them, and some do a bit of both. In this section, we‚Äôll learn how to classify these points by examining the slope field, the sign of \(f(y)\text{,}\) and a tool called a phase line.
    üîó

    Types of Equilibria.

    Suppose \(y = c\) is an equilibrium solution of an autonomous equation \(y' = f(y)\text{.}\) If you nudge a solution slightly above or below \(c\text{,}\) it might drift back, move away, or react differently on each side. What it does determines the type of equilibrium it is.
    üîó
    There are three common types:
    üîó
    • Stable (sink): Solutions move toward the equilibrium from both sides.
      üîó
    • Unstable (source): Solutions move away from the equilibrium on both sides.
      üîó
    • Semi-stable (node): Solutions move toward the equilibrium on one side and away from it on the other.
      üîó
    In a slope field, a sink looks like arrows converging toward a horizontal line, a source shows arrows diverging away, and a node is a mix: converging on one side, diverging on the other. Next, we‚Äôll look at different ways to determine these behaviors.
    üîó
    üîó

    Classification via the Phase Line.

    Slope fields show a lot of information at once, but for autonomous equations we can simplify. Since the slope depends only on \(y\text{,}\) we can ‚Äúcompress‚Äù the slope field into a simple vertical diagram of just \(y\)-values. This is called a phase line.
    üîó
    On a phase line:
    üîó
    • Equilibria are marked with solid dots.
      üîó
    • Regions where \(f(y) > 0\) get upward arrows.
      üîó
    • Regions where \(f(y) < 0\) get downward arrows.
      üîó
    The arrows summarize how \(y(t)\) changes: whether solutions are rising or falling. Follow the arrows up or down and you‚Äôll see where solutions eventually settle‚Äîor whether they‚Äôre pushed away.
    üîó

    üåå Example 88. Growth and Decay.

    Find and classify the equilibrium solutions of the autonomous equation:
    \begin{equation*} \frac{dy}{dt} = y^2 - 4y \end{equation*}
    üîó
    Solution.
    First, set \(f(y)=0\text{:}\)
    \begin{equation*} y^2 - 4y = y(y - 4) = 0, \end{equation*}
    giving equilibrium solutions at \(y = 0\) and \(y = 4\text{.}\)
    üîó
    These values split the \(y\)-axis into three regions. We test the sign of \(f(y)\) in each:
    üîó
    Region Test Point \(f(y) = y(y - 4)\) Sign of \(f(y)\)
    \((-\infty, 0)\) \(y = -1\) \((-1)(-5) = 5\) Positive
    \((0,4)\) \(y = 1\) \((1)(-3) = -3\) Negative
    \((4,\infty)\) \(y = 5\) \((5)(1) = 5\) Positive
    üîó

      üîó

      üîó

    Classification via the Linearization Method.

    There‚Äôs another way to classify equilibria: use a quick calculus check on \(f(y)\text{.}\)
    üîó
    Notice in Figure¬†90 that the sign of \(f(y)\) flips as you pass through an equilibrium. The pattern of that flip tells you what kind of point it is:
    üîó
    • At the source, \(y=-1\text{,}\) \(f(y)\) goes from negative ‚Üí positive (increases).
      üîó
    • At the sink, \(y=1\text{,}\) \(f(y)\) goes from positive ‚Üí negative (decreases).
      üîó
    The important observation is that \(f(y)\) increases through sources and decreases through sinks. This is true in general, so we can classify equilibrium solutions by looking at the sign of \(f'(y)\) at the equilibrium points.
    üîó
    üîó
    If \(f(y_0) = 0\text{,}\) then \(y_0\) is an equilibrium. To classify it:
    üîó
    • If \(f'(y_0) < 0\text{,}\) it‚Äôs a sink (stable).
      üîó
    • If \(f'(y_0) > 0\text{,}\) it‚Äôs a source (unstable).
      üîó
    • If \(f'(y_0) = 0\text{,}\) the test is inconclusive‚Äîyou‚Äôll need more investigation.
      üîó
    This is called the linearization method, because it looks at the linear (slope) behavior of \(f(y)\) near the equilibrium.
    üîó

    üåå Example 92.

    Consider the autonomous equation:
    \begin{equation*} \frac{dy}{dt} = y^2 - 4y + 3. \end{equation*}
    Find and classify the equilibrium solutions using linearization.
    üîó
    Solution.
    First, solve \(f(y)=0\text{:}\)
    \begin{equation*} y^2 - 4y + 3 = (y - 1)(y - 3) = 0, \end{equation*}
    so \(y=1\) and \(y=3\) are equilibria.
    üîó
    Then compute the derivative:
    \begin{equation*} f'(y) = 2y - 4. \end{equation*}
    At each equilibrium:
    • \(f'(1) = -2\) ‚Üí sink (stable).
      üîó
    • \(f'(3) = 2\) ‚Üí source (unstable).
      üîó
    üîó
    üîó
    üîó

    üì§ Wrap-Up.

    üóùÔ∏è Key Takeaways...

    • Equilibrium solutions occur where \(y' = f(y) = 0\text{.}\)
      üîó
    • Sinks pull solutions in, sources push them out, and nodes do one of each.
      üîó
    • Phase lines show this behavior simply and clearly.
      üîó
    • The derivative \(f'(y)\) can classify stability quickly via the linearization method.
      üîó
    üîó

    \begin{equation*} \frac{dy}{dt} = y^2-3y \end{equation*}
